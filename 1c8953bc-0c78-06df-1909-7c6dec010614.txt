<br/><br/>　　    不得不说，巴特利特是一个看起来十分慈祥的人，而且他的演讲很有感染力，整个光明竞技场上的观众们早已经热血沸腾了。<br/>　　<br/>　　    这一次的东西大陆学院大赛在皓理曦城的光明竞技场举行，光明竞技场是皓理曦城，也是整个神佑联盟最大的比赛场地，最多可以同时容纳二十万观众。<br/>　　<br/>　　    今天尽管是大赛的第一天，但是整个光明竞技场座无虚席，早已经人满为患了。<br/>　　<br/>　　    就连最边角的座位都已经被黄牛从五个银币炒了到五个金币以上的价格，整整翻了十倍。可是即便如此，想要买到也不是一件容易的事情。<br/>　　<br/>　　    “现在有请我们最伟大、最无私、最强大的教皇陛下，为大家讲话！”巴特利特说了半个小时以后，忽然高声宣布道。<br/>　　<br/>　　    听到巴特利特的话以后，所有的观众一愣，然后就看到一名衣着华丽，头戴金冠的人从天而降，不是别人，正是教皇大人。<br/>　　<br/>　　    观众们紧全都从座位上站了起来，然后双膝跪地，山呼道：“教皇陛下，福寿无疆！”<br/>　　<br/>　　    那些并非神佑联盟的人并没有跪在地上，而是左手握拳，放在右肩膀处，弯腰表示自己的恭敬和祝福。<br/>　　<br/>　　    “光明神最忠诚的子民们啊！愿光明神与你们同在！”教皇张开双臂，柔和的光芒从他的身上散发出去。<br/>　　<br/>　　    光芒笼罩了全场，每个被光芒加身的人都觉得身心舒泰了好多。<br/>　　<br/>　　    德利莱卡目瞪口呆的看着这一幕，如此大范围的魔法，教皇并没有借助任何东西，而是很随意的就释放了出去。这让他心中震撼不已，对教皇的实力不得不重新评估一下了。<br/>　　<br/>　　    辅助技能相对来说控制要容易一些，但是一次笼罩如此大的范围，如此多的人，足以看出教皇的精神力以及对魔法元素的控制力是多么恐怖了。<br/>　　<br/>　　    “光明神赋予我们阳光、雨露、光明以及我们所拥有的一切，我们都是得到光明神赐福庇护的人，所以我们一定要怀着敬畏的心，去虔诚的信仰！这一次东西大陆学院大赛，将会得到光明神的祝福，愿所有参赛的学员们都能取得一个不错的成绩！大家都起来吧！”<br/>　　<br/>　　    大家陆陆续续的站起身来，看到教皇身上的光芒越来越明亮，却并不耀眼，只是让人看不清他的样貌。当光芒达到极致的时候，教皇竟然就那样凭空消失不见了。<br/>　　<br/>　　    德利莱卡隐约感受到了一丝空间的波动，不过他毕竟不是空间类魔法师，所以只是隐约感受到罢了。<br/>　　<br/>　　    巴特利特显然对教皇神龙见首不见尾的作风见怪不怪了，直接宣布了比赛的细节和流程。<br/>　　<br/>　　    这一次东西大陆学院大赛，禁止学员们在比赛过程中使用一切的魔法药剂，无论是攻击类还是恢复类，亦或是辅助类，严禁比赛前使用拥有强化类属性的药剂，比如狂化药剂、持续恢复类药剂等。<br/>　　<br/>　　    学员们使用武器的种类不受任何限制，不论是赤手空拳亦或是使用神器，那都是学员们自己的权利，比赛过程中，神圣教廷不会给任何参赛学员提供任何武器。<br/>　　<br/>　　    这一次大赛参赛学院一共有九所，卡罗帝国四所，神佑联盟三所，暮夜联邦两所。<br/>　　<br/>　　    卡罗帝国的学院分别是卡罗皇家学院、育才书院、晨风学院、龙威学院。<br/>　　<br/>　　    神佑联盟的学院分别是光明学院、天罡学府、王者学院。<br/>　　<br/>　　    暮夜联邦的学院分别是暮夜学院和暗夜学院。<br/>　　<br/>　　    预选赛比赛抽签选择对手，分为个人赛和团队赛。<br/>　　<br/>　　    个人赛为五场，赢一场得一分，七人团队赛一场，赢了得四分，输了不扣分。两场积分累计，总分高的队伍获得胜利。<br/>　　<br/>　　    也就是说，个人赛哪怕只赢得一场，但是团队赛获胜，依然可以赢得最后的胜利。如果个人赛五场全胜，那团队赛完全不用比，一样可以获得胜利。<br/>　　<br/>　　    得知比分规则以后，个人实力强的参赛队伍现在想要获得个人赛五连胜，这样不但可以省去团队赛，还可以隐藏队伍的实力。<br/>　　<br/>　　    个人实力稍微逊色一些的队伍，开始考虑如何赢得一场或者两场比赛，将比赛拖入到团队赛，然后通过强大的团队协作，来赢得比赛最后的胜利。<br/>　　<br/>　　    比赛先进行个人赛，然后是团队赛。<br/>　　<br/>　　    个人赛人员选定以后，不得更换，人员的出场顺序也是开赛前拟定，当场公布，不得变更。<br/>　　<br/>　　    团队赛人员名单等个人赛结束后拟定，团队赛人员不得多于七人，少于七人没有问题，哪怕只有一个人参赛，也是允许的。<br/>　　<br/>　　    由于今天是第一天，所以神圣教廷并没有让所有的学院都参加比赛，只是准备进行两场比赛。<br/>　　<br/>　　    一共有九支队伍，每一支队伍派一名学员代表上去抽签。代表卡罗皇家学院上去抽签的既不是李振邦，也不是慕容冰，而是张大友。<br/>　　<br/>　　    这段时间的磨合，大家已经基本确定了自己的定位，李振邦的特殊能力相当于整个团队的大脑，慕容冰是整个队伍中最锋利的刀尖，张大友是最强的护盾。<br/>　　<br/>　　    本来众人推举的是李振邦为队长，作为整个队伍的大脑，当队长也算是当之无愧的。<br/>　　<br/>　　    不过李振邦推辞了，成为队长以后，责任是更重一些，但是如果赢得胜利，最后得到的关注也会更多一些。不过李振邦还是喜欢闷声发大财，那些所谓的名利对他来说并没有什么意义，反而可能会成为一种桎梏，他更愿意做一个幕后推手。<br/>　　<br/>　　    李振邦不做队长，那最有资格的自然就是慕容冰了，不论是实力还是背景，绝对都够资格。不过慕容冰对这个让来的队长并没有兴趣，如果当上队长，她内心就觉得别扭，毕竟第一人选并不是她。<br/>　　<br/>　　    慕容冰放弃了队长，柳衣霜自然就成了实力最强者，不过她对这队长没有兴趣。<br/>　　<br/>　　    最后队长的人选就落在了浩南和张大友两人的头上，两个人相对来说，自然是张大友更有人气一些。要知道，这十人小队中，算上张大友的话，和他有关系的人可是整整六个，超过了半数。<br/>　　<br/>　　    关于队长人选，浩南最后直接放弃了。一个是因为他不想因为这个队长的事情和大家闹得不愉快，再一个他也知道，就算投票他也没有优势。<br/>　　<br/>　　    最主要的是这个队长说好听一点儿叫队长，实际上不过就是一个名头，没有任何实权。退一万步来说，就算他当上了队长，难道他能轻易指挥的动慕容冰或者李振邦？他还不如送个顺水人情来的实在。<br/>　　<br/>　　    神圣教廷为了表示对卡罗皇家学院的重视，让他们先抽取号码。抽奖箱中一共有九张卡片，其中一张是免战晋级，其他八张分别是一号到八号。<br/>　　<br/>　　    抽中一号卡片的队伍和抽中二号卡片的队伍比赛，三号和四号队伍比赛，依次类推。<br/>　　<br/>　　    张大友的运气也许在成为队长的时候就用完了，他并没有抽取到免战晋级卡。不但没有抽到晋级卡，他反而抽到的是二号签，意味着今天他们将会有一场比赛。<br/>　　<br/>　　    让张大友感到郁闷的是，抽中免战晋级卡的竟然是光明学院的人。更让他郁闷的是，抽中一号签的竟然是卡罗帝国的一家学院——育才学院。<br/>　　<br/>　　    也就是说，第一场比赛竟然就是卡罗帝国的学院之间的内斗，这样不但会淘汰掉一名卡罗帝国的学院，甚至有可能会过早的暴露出众人的实力，这可以说是非常不妙了。<br/>　　<br/>　　    卡罗皇家学院的学员们，甚至很多的老师们也感到有些无奈，不过也只能无可奈何。<br/>　　<br/>　　    抽到三号卡片的是之前有过接触的天罡学府，虽然他们当时被德利莱卡大师羞辱了，但是不可否认他们学员的强大实力。<br/>　　<br/>　　    众人心中有些安慰的是，抽到四号卡片的并不是卡罗帝国的学院了，而是暮夜联邦的暗夜学院。<br/>　　<br/>　　    暗夜学院是一个很奇怪的学院，并不像暮夜学院一样，什么样的学员都有。暗夜学院其实是暗夜精灵一族的专属学院，整个学院只有暗夜精灵一个种族的学员。<br/>　　<br/>　　    暗夜精灵一族和精灵一族完全不同，精灵一族的人各个肤白貌美，就连男子都有着让女人嫉妒的容颜。暗夜精灵一族的人相对精灵一族的人皮肤有些发黑，身上带着黑色的纹理，这是出生时就携带的。<br/>　　<br/>　　    精灵族的战士各个都是出色的弓箭手，魔法师擅长的是木系魔法，治疗和辅助，类似于翡翠巨龙欧米伽。<br/>　　<br/>　　    暗夜精灵一族的战士更擅长于暗杀、偷袭和下毒，魔法师是黑暗属性魔法师，黑暗魔法和亡灵魔法并不一样，黑暗魔法的腐蚀性很强，并没有操控尸体和灵魂的能力，这也是神圣教廷能允许暗夜精灵存在的一个重要原因。<br/>　　<br/>　　    暗夜精灵一族被精灵一族称之为背叛之族，精灵族信仰的是自然女神，据说暗夜精灵在上古时期就是精灵一族，不过由于他们背叛了自然女神，所以才被诅咒，变成了现在的暗夜精灵一族。<br/>　　<br/><br/><br/>
