上一次在他的生日上两个人不欢而散，他让花姐停掉了她所有的工作，还让她自己去想通了来找他，但是她没有来。
<br/>
<br/>        她不但没有来，还自己想办法搞了一场直播，叶翎这一场直播是很小众的，没什么影响力，他也就冷眼旁观着她离开他能搞出什么动静来。
<br/>
<br/>        她是在他的羽翼下长大的，被他无微不至的呵护着，这朵娇艳的玫瑰花从来没有受过风吹雨打。
<br/>
<br/>        她进入娱乐圈也是他一路开道，不但有金牌团队护着她，顶级资源也是主动送上门的。
<br/>
<br/>        顾夜瑾觉得自己是将她给宠坏了，她竟然都敢忤逆他了，如果不给她一点教训，她就不知道自己有几斤几两。
<br/>
<br/>        所以顾夜瑾并没有进直播间，没兴趣知道，他笃定了叶翎搞不出什么动静。
<br/>
<br/>        这时“叩叩”的敲门声响起了，叶翎的金牌经纪人花姐匆匆的走了进来，像是有要事汇报，“顾少。”
<br/>
<br/>        顾夜瑾淡淡的看了花姐一眼，“什么事？”
<br/>
<br/>        “顾少，翎翎直播的事情想必你也听说了吧，翎翎首场直播秀已经踢掉了夏妍妍那位天才少女的热搜登顶第一了，同时她的直播秀刷新了直播界整个记录了。”
<br/>
<br/>        “这场直播里翎翎从头到脚所有的穿戴都被抢购一空，有人问翎翎喜欢抹什么口红，这不是翎翎直播带货的范畴，但她随口说了几个，那些网友迅速爬墙那几个口红品牌，将人家的东西秒空了。”
<br/>
<br/>        “刚才我的电话都被打爆了，很多一线品牌联系我求跟翎翎合作，当然我也收到了直播界的投诉，他们都跪求翎翎别直播了，把饭碗还给他们。”
<br/>
<br/>        顾夜瑾直接抿了一下绯色的薄唇，他笃定搞不出什么动静来的人竟然给他闹出了这么大的动静。
<br/>
<br/>        顾夜瑾迅速拿出手机，直接忽略了陆寒霆投来的嘲笑目光，进了直播间。
<br/>
<br/>        叶翎已经准备变装秀了，直播里响着音乐，她身上是一件粉色公主裙，低扎的马尾有几缕茶色的卷发缠绕在了她的粉颈里，她随着音乐扭了两下屁股，那么明艳软媚的女孩现在清纯的能掐出水来。
<br/>
<br/>        这时音乐突然一变，叶翎扎着的马尾全部潋滟妖娆的散开，她身上的粉色公主裙也骤然变成了黑色西装。
<br/>
<br/>        刚才清纯软媚的女孩全然不见了，她眼梢看过来的时候高傲冷艳的挑了一下勾人的眼角，还又A又飒的扯了一下自己脖间的领带。
<br/>
<br/>        直播间的屏幕真的被刷爆了，简直在踩踏啊啊啊，翎翎的西装杀直接杀了我，我已死！
<br/>
<br/>        顾夜瑾的瞳仁被叶翎这一身的西装杀给刺激的狠狠一缩，这个女孩离开他的时候还是很乖巧柔软的一团，这两年他也想过她长大后的样子，就没有想到她已经是如此祸国殃民的妖姬模样。
<br/>
<br/>        花姐带了叶翎两年了，也再次被叶翎的盛世美颜给震慑住了，外面说的没错，叶翎这张脸天生就是为了娱乐圈而生的。
<br/>
<br/>        顾夜瑾看着直播，阴鹜的勾了一下薄唇，然后他随手往直播里打赏了一笔超级巨款，在烟花炸响欢迎他这位金主大人时，他发了一个说你爱我。
