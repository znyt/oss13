　　右手的沾有剧毒的利爪和左手锋利无比的鲨鱼刀交替着向鲨鱼发起攻击，一招快过一招，一招狠过一招，闷热的空气中，毒天使犀利无比的攻击留下一片残影。

<br/>　　巫师非常明确的拒绝了这个在他看来完全是异想天开的提议。
<br/>　　“既然如此，那你就去死吧！”
<br/>　　毒天使大怒，接连发起快攻，让巫师陷入了绝对的被动。

<br/>　　罗昊显然是对李然的“战术分配”感到不满。
<br/>　　“很简单，因为你是‘龙刺’，而我不是。这叫能者多劳。”
<br/>　　李然对着罗昊露出一个灿烂的笑容。
<br/>　　罗昊怔了怔，愣是没有做出任何反驳。因为李然说得就是事实，他根本无从反驳。

<br/>　　当啷！
<br/>　　手雷在落地上，在地上弹跳了一下，随后“骨碌碌”的往前滚去。
<br/>　　Bob在看到有东西朝着自己这边飞过来的时候，立刻意识到那是什么东西，第一时间让自己身边的“黑水”雇佣兵散开。
<br/>　　“轰——！”
<br/>　　手雷轰然炸开，一团火球拔地而起，冲击波席卷向周围几名“黑水”雇佣兵。
<br/>　　虽然这颗手雷并没有对那几名“黑水”雇佣兵造成有效的杀伤，但是却给罗昊、李毅峰两个人制造了进攻的机会。

<br/>　　哒哒哒！哒哒！哒哒哒！哒哒！
<br/>　　罗昊和李毅峰两个人用MPX冲锋枪对着周围的“黑水”雇佣兵进行点射，两个人交替着进行射击，保持着火力的延续性。
<br/>　　一条蓝色小蛇朝着罗昊和李毅峰两个人游来，可是还没完全靠近两个人，因为两个人身上所携带的避毒丸，又立刻游走了。
<br/>　　Bob靠在从地上快速的爬起来，按住耳麦对着他们的火力手Matt大声吩咐道：“Matt，压制住他们！压制住他们！”
<br/>　　火力手Matt把机枪架在一辆汽车的引擎盖上，对着罗昊和李毅峰的方向进行扫射，压制他们的行动。
<br/>　　哒哒哒哒哒哒哒哒哒哒……
<br/>　　M249SAW枪口吐出一道火舌，子弹倾斜而来，逼着罗昊和李毅峰两个人不得不散开，进行规避。
<br/>　　看到罗昊和李毅峰被自己这边的火力手压制，Bob立刻带着其余三个人朝着罗昊和李毅峰逼近过来。
<br/>　　而正在对罗昊和李毅峰进行火力压制的Matt根本没有注意到那条刚从罗昊和李毅峰脚边游开的蓝色小蛇，正以“S”型的运动轨迹，朝着他游来，死神已经露出了狞笑，正在向他靠近。
<br/>　　这些被毒天使放出去，极具攻击性的蓝色小蛇可不具备什么敌我识别，之前是因为“黑水”这边的人都在毒天使身边，加上距离上也比较远，所以这些蓝色小蛇就朝着龙隐部队这边攻击过去；但是现在情况发生了改变，巫师的介入，让龙隐部队这边避免了被蓝色小蛇攻击的可能，没有避毒丸保护的“黑水”雇佣兵们成了这些在停车场中四散开来的蓝色小蛇的攻击目标。

<br/>　　Matt手握M249SAW机枪对着罗昊和李毅峰所隐藏的方向疯狂扫射，此时他是兴奋的，无比兴奋，这简直比看到一个绝色女郎还要让他兴奋，而这种所导致的直接结果就是加速蛇毒在他体内的流动，加速他的死亡。
