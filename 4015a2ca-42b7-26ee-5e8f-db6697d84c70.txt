<br/>　　哗啦啦。

<br/>　　吞服丹药，剑祖荒身上的伤势，便是瞬间，全部恢复过来。
<br/>　　“好，我们联手，诛杀此人！”
<br/>　　剑祖荒冷喝一声，眼神中杀机更浓，看向楚风眠狂傲开口道。

<br/>　　在一旁的鹿战神，也是跟剑祖荒出手的一刻，瞬息之间，自己也是出手，突然打出一拳，无穷武意，笼罩着楚风眠，一拳降下。
<br/>　　剑祖荒，鹿战神，都是在这天羽神域之中赫赫有名的人物，如今而这两人联手之下，攻击凶猛到了极点。
<br/>　　“小子，我鹿战神以前从来不跟他人联手围攻，今日你就算是死，也是一种荣幸了！”
<br/>　　鹿战神向着楚风眠猛然开口道。
<br/>　　二人的攻势，已经是向着楚风眠轰击过来。
<br/>　　面对这剑祖荒，鹿战神二人的联手，几乎是任何武者，都要手忙脚乱起来，他们二人所代表的，都是生死境之中最为强大的力量了。
<br/>　　现在一起联手，力量都是要超越生死境范畴，直逼碎命之境。
<br/>　　任何武者碰到，都要胆战心惊。

<br/>　　这中心碰撞的力量，似乎不管是他们哪一方，都无法承受。
<br/>　　“什么？这都杀不死？”
<br/>　　不管是那些圣王宗弟子，还是天剑宗弟子，在看到剑祖荒，鹿战神联手之下，轰击楚风眠的时候，在心中都已经是觉得，楚风眠肯定是一个死人了。

<br/>　　看到楚风眠被逼退，在那圣王宗弟子中，突然一道身影，猛然冲了出来，冲着楚风眠，一拳轰击而出。
