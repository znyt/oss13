　　小马和吴嘉伟抓着叶少阳，又要运气抵抗水中的吸力，突然两人都感觉腋下一软，咯吱窝被人挠了一下，心头都是一惊，操，这个时候谁居然干这种事？
<br/>　　咯吱窝被挠，吸引了两人的注意，突然胸口又各自挨了一掌，定睛一看，是叶少阳干的！
<br/>　　“你们先上去，照顾好自己，我去水底看看！”叶少阳说完就游到了一边去，小马和吴嘉伟再想找他，水中乱流横生，根本无法靠近，在远处干着急。
<br/>　　叶少阳游开之后，并没有下水底去看——这句话是安慰他们的，他继续调息起来。
<br/>　　恢复实力，这是他唯一的希望。
<br/>　　随着身体逐渐下沉，叶少阳越发感到冰冷，流动的海水，不断带走他体内的修为——这还是小事，这种流失的速度，对他影响不大。可怕的是神识一直在被撕扯，似乎要从身体里出去。
<br/>　　这让他没法集中精神调息。
<br/>　　“少阳……”
<br/>　　身下突然传来一个声音，是碧清！
<br/>　　叶少阳这才想起她也跟着来了，好像自从事变之后，就再也见过她。
<br/>　　叶少阳往身下看去，一根花茎从下面升上来，将自己托住，稳定在水中，叶片舒展开来，中间长出一朵莲花，打开的叶瓣将他包裹起来，冰凉的感觉一下子消失了，叶少阳感觉很温暖，就像是一个软软的身体从后面贴住了他，双手环抱住他的脖子，在他身体上下抚摸……
<br/>　　“别分神！”碧清的声音传到他神识中去，“我在帮你调息。你享受这感觉就行了……”
<br/>　　享受……
<br/>　　叶少阳这才想起，碧清的本尊是一朵青莲，是水生植物，修为又深厚，不管什么样的水域，也困不住她，而且她是植物成精，天生木系，会治愈类的法术，当下心中大喜，抱元守一，心中默念着大周天吐纳心法，任凭碧清摆弄了。
<br/>　　叶少阳不知道的是，碧清是莲花成精，莲花在佛门中是一种特殊的植物，有涅槃的意思——因此菩萨和佛都是坐莲花台的。
<br/>　　这无色海对一般人来说是可怕的法阵，但她天生就是佛门圣灵，与无色海的属性正好相融，在这无色海中算是如鱼得水，比所有人都舒服。
<br/>　　“少阳，海底有四把剑，彼此灵力交融，便是组成这法阵的核心，我刚尝试了一下，但无法撼动……”碧清一边替他驱赶右臂中的法令虫，一边在神识中对叶少阳说道。
<br/>　　四把剑？
<br/>　　叶少阳心中疑惑，想到之前鲲鹏所说的剑阵，八成没有错了。
<br/>　　“你带到到海底，我看看！”
<br/>　　碧清用花蕊托着他，缓缓沉入水底——也不过二十来米深。那电弧一般的结界，一直连到水底，碧清托着他朝一个方向漂往结界的边界，离老远叶少阳就看到了一道强光，像一盏长杆的日光灯似的，竖立在水底。
<br/>　　近了之后，叶少阳看清，这是一把通体青色的剑，剑刃朝下，插在泥土中，锋芒极盛，但是都化作了电弧，不断朝周围扩散出去。
<br/>　　剑柄上有一颗绿色的珠子，忽明忽暗，灵力就是从这颗珠子上产生的。
<br/>　　叶少阳不认识这把见，但总觉得似乎在哪里见过，上下打量，突然在剑锋上发现了一行字，因为字迹是刻上去的，没有颜色，剑光又盛，不仔细看很难发现。
<br/>　　叶少阳仔细辨认起来，这四个篆字是：一剑绝仙。
<br/>　　绝仙？
<br/>　　心头猛地一颤，有三个字猛地从脑海中浮起来：绝仙剑！
<br/>　　没错，正是诛仙四剑中的绝仙剑！
<br/>　　叶少阳陡然想起了当初从张无生口中听到的那个传闻：诛仙四剑被偷走了。当时也曾怀疑是法术公会干的，但是没有证据和线索，后来发生了一系列的事情，让自己都忘记了这件事，没想到，诛仙四剑居然在这里！
<br/>　　诛仙四剑分为：诛仙剑、戮仙剑、陷仙剑、绝仙剑。
<br/>　　四把剑上都刻有文字，分别是：诛仙剑，诛定乾坤；戮仙剑，屠戮天下，陷仙剑：海阔地陷；绝仙剑：一剑绝仙！
<br/>　　这四把剑分开来，单个都是九段光的法器，虽然牛叉，但并不算逆天，不过若是四把剑在一起，组成诛仙剑阵，传说中能诛杀一切生灵。
<br/>　　当然这只是传说，叶少阳一开始也以为这四把剑很牛逼，那一次小九出世，法术界闻风而动，来了小半个法术界的人，一起布阵，当时几大宗门的大佬就祭出了诛仙剑阵，想要擒拿道风，威力大倒也大，但事后在叶少阳看来，也没有传说中那么厉害。
<br/>　　要知道，那时候的道风虽然已经是人间道神，但毕竟只斩了一尸，实力也不逆天，诛仙剑阵也没能真的杀了他，从这件事后，叶少阳就对这种传言不屑一顾了。
<br/>　　后来有一次回山，叶少阳跟青云子闲聊时候说起这件事，青云子告诉他，诛仙剑阵是真的很厉害，但能发挥出剑阵多大的威力，还是取决于控阵之人的实力，那一次围攻道风，人虽然不少，但最多也就是几大宗师，还有几个弟子在凑份子，自然无法完全发挥诛仙剑者的威力。
<br/>　　“少阳，你在想什么？”碧清突然开口，打断了叶少阳的思绪，与此同时，叶少阳感到右手一麻，好像被什么东西夹了一下，火辣辣的疼，低头看去，自己右手虎口位置多了两个牙印一样的伤口，一只黑色的形状像虾一样的东西被从牙印里硬生生挤出来，花瓣中射出一道绿波，击打在这东西身上，霎时间化作了一道烟波，被水流冲散。
<br/>　　“并没有想象中那么厉害。”碧清幽幽说道。
<br/>　　“那是因为有你帮忙，靠我自己就麻烦了。”叶少阳长吁了一口气，回想法术公会的手段，的确够厉害的，他们一定是率先就将诛仙四剑埋伏在仓库的周围，用仓库里那些法器吸引他们来到、聚集在一起，接着用法令虫伤害自己，激活诛仙剑阵……
<br/>　　这一系列计划，进行得行云流水，也异常凶狠。但他们少算了碧清。
