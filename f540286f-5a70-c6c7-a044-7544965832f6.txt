当年她救的那个男人？
<br/>
<br/>        夏夕绾回忆了一下，那一年她十二岁，在冰天雪地里救了一个已经昏迷的男人，她敢肯定如果她再晚去一会儿，那个男人就会死在雪地里了。
<br/>
<br/>        当时大雪封路，又濒临天黑，气温冷的整个人都在打颤，她吃力的将那个男人带到了附近的山洞里，又点燃了树枝来取暖，但是还是好冷，那个男人的四肢已经快冻僵了。
<br/>
<br/>        夏夕绾就脱了自己的衣服紧紧的抱住了那个男人，用彼此的体温来取暖。
<br/>
<br/>        就这样，那个男人熬了过来。
<br/>
<br/>        现在夏夕绾想一想，当年的自己才12岁，一心只想着救人，但是这在别人眼里却演变成了一幅极其香艳的画面，在夏小蝶的挑唆污蔑下，苏希竟然一直觉得她是一个脏了的女孩。
<br/>
<br/>        叶翎骂的很对，这些人真是龌龊不堪。
<br/>
<br/>        关于那个男人…
<br/>
<br/>        夏夕绾回道八年都快过去了，我已经记不清那个男人了，就算他站在我面前，我大概也认不出了，不过他清醒后曾经给过我一块玉佩，他说他会回来找我的。
<br/>
<br/>        叶翎玉佩呢？
<br/>
<br/>        夏夕绾丢了，找不到了。
<br/>
<br/>        叶翎直接发了一个用小石头将她给砸晕的表情包过来言情小说看过没，按照惯有套路你救的这个男人肯定是某个豪门太子爷，你救了他，他要以身相许的。
<br/>
<br/>        夏夕绾……
<br/>
<br/>        她是真的不知道那块玉佩到哪里去了，她明明记得自己将玉佩锁在了房间的抽屉里，但是等她再打开时，玉佩已经不见了。
<br/>
<br/>        现在想来当年大雪里一场救人，在场的不光是她和那个男人，还有苏希，夏小蝶。
<br/>
<br/>        这么多人都来了。
<br/>
<br/>        这时叶翎的信息来了绾绾，这是追踪苏希手机找到的，他打出去的那个电话在海城郊外的一个民舍里，林婶就在那里。
<br/>
<br/>        夏夕绾看着叶翎发来的地址，其实她去8206房间赴约已经有了计划，就是以自己为诱饵让苏希打出电话，叶翎再定位追踪到林婶的位置。
<br/>
<br/>        苏希将林婶藏在了郊外的民舍里。
<br/>
<br/>        太好了，她找到了！
<br/>
<br/>        夏夕绾发了一个大大的mua过去就结束了聊天，现在已经是晚上了，很方便行动，她要立刻去找林婶。
<br/>
<br/>        关于夏小蝶污蔑她的事情，她自然会找个好时机给夏小蝶一份大礼。
<br/>
<br/>        当务之急还是将林婶救出来。
<br/>
<br/>        夏夕绾准备出门的时候突然手机响了，来电话了。
<br/>
<br/>        她垂眸一看，手机屏幕上跳跃着“陆先生”三个字。
<br/>
<br/>        是陆寒霆打来的电话。
<br/>
<br/>        纤长的羽捷颤了几下，夏夕绾并没有接听，而是任由手机铃声响了一遍又一遍。
<br/>
<br/>        很快“叮”一声，有微信来了。
<br/>
<br/>        夏夕绾点开一看，是陆寒霆发来的睡了？
<br/>
<br/>        夏夕绾看着这两个字，是他一贯简洁强势的风格，再往上翻，两个人的聊天还停留在“敢骂我？回去再收拾你”这句话上。
<br/>
<br/>        那时的自己很甜蜜，像个情窦初开的小姑娘一样被他撩的不要不要的，现在的她已经恢复平静了。
<br/>
<br/>        她要在心里筑起高墙将他阻隔在外，不再被他诱惑，也不给他伤害自己的机会。
