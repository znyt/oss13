　　天蛇和玄龟看着前方打开运输机尾部舱门，脚下深踩油门，驾驶着防弹突击车加速向运输机冲去。
<br/>　　马路左侧的大楼，埋伏于大楼中的机动部队队员用自动步枪和轻机枪对着“支奴干”运输机进行扫射。
<br/>　　哒哒哒哒哒哒……
<br/>　　铿铿铿铿铿铿……
<br/>　　子弹扫在运输机机身上，碰撞出点点黄色的火星，机身上也是被子弹打出一个个弹孔。
<br/>　　“支奴干”运输机侧边舱门打开，周立手中抱着一挺M60通用机枪，左手上缠着弹链，对着周围大楼中的伏兵进行着火力反压制。
<br/>　　哒哒哒哒哒哒……
<br/>　　周立怒吼着，强健有力的右手牢牢地控制着机枪，手臂上如花岗岩一般坚硬的肌肉，凸显起来，彰显着爆炸性的力量。
<br/>　　外层刷着铜漆的弹壳和弹链子弹固定支架的塑料碎片，落在机舱金属地板上欢快的跳动着。
<br/>　　两辆防弹突击车转速表和时速表上的指针都是在向红区逼近，两台发动机飞快的工作着，为两辆突击车输送着马力。
<br/>　　400米！300米！200米！
<br/>　　眼看着防弹突击车距离运输机放下的舱门还有不到200米的距离，左侧大楼中，数名force-recon队员扛着AT4反坦克火箭筒出现在窗边。
<br/>　　“火箭筒！！”
<br/>　　周立看到了站在大楼窗口边，扛着火箭筒已经通过发射具上光学瞄准镜对运输机进行瞄准的force-recon队员，对着驾驶舱中的梅庆伟和陈应天大声提醒。
<br/>　　“妈的！”
<br/>　　梅庆伟咒骂着拉升运输机的高度，让运输机重新起飞，不然等到火箭弹招呼过来，就是机毁人亡的下场。
<br/>　　咻！咻！咻！
<br/>　　梅庆伟刚把运输机拉升起来，三枚火箭弹便是从马路左侧的楼房中飞出，飞向街道对面的餐馆、水果摊。
<br/>　　两枚火箭弹击中了摆设在马路对面的水果摊，两个水果摊被炸得粉碎，摆放在水果摊上的水果更是被炸成了果酱，木板碎片、水果碎片纷飞而起。
<br/>　　在天蛇和玄龟驾驶着防弹突击车转过街角冲上这条马路的时候，街道边的行人，水果摊边的顾客全都在第一时间就逃开了，寻找安全的地方跑去。
<br/>　　两个水果摊上没有造成人员伤亡，但是那枚飞入餐馆中的火箭弹就造成了大量人员的伤亡。
<br/>　　餐馆中不仅有原本在参观中就餐的顾客，还有从街道上逃跑进来避难的行人。再加上餐馆中还有燃气管道的关系，所以这一颗火箭弹造成的爆炸远远胜于火箭弹原先的爆炸威力。
<br/>　　轰！
<br/>　　天动地摇的爆炸中，炙热的火焰震碎了餐馆四周的玻璃窗，翻卷着从窗户中冲出，餐馆屋顶的一角也是被炸得坍塌下去，滚滚黑烟冲天而起，而在着黑烟中还有着火光在跳动，并且伴随着燃气管道被点燃，有着余爆发生。
<br/>　　躲藏在餐馆中避难的顾客和行人遭受到了灭顶之灾，他们中有好些人被气浪卷出了餐馆，他们全身焦黑，在刚才的爆炸中被炸得面目全非。
<br/>　　这座城市在这个国家属于经济中心一般的存在，在这个国家中的地位相当于中海市在国内的地位！就是在这样一座全国中心一般的城市中，多次发生大爆炸，并且伴有平民的死亡，事情的性质立刻就发生了改变！
<br/>　　“混蛋！这些混蛋！”
<br/>　　比洛多看着电视中从现场传递过来的画面，愤怒的用拳头敲击着自己的办公桌，怒声骂道：“维尔纳特这个混蛋到底想要干什么！？他是想要把这座城市都拆了吗！？”
<br/>　　“爸爸，我觉得你不能放任维尔纳特这么胡来了！”塔尼特走到比洛多面前，说道：“出现了平民伤亡，情况就会变得复杂起来，你现在正处于大选的关键时刻，任何风声舆论都有可能会对你的大选造成影响。”
<br/>　　比洛多手指在桌子上快速地在桌子上敲打着，心中做着权衡，最后他还是拿起了桌子上的电话，下令让机动部队开始撤退，这件事情已经超出了他的控制，就如塔尼特所言，他不能再让维尔纳特这么为所欲为了。
<br/>　　瓦斯汀办公室，本来瓦斯汀对这次的大选已经是不抱有任何的希望，打算等到大选结束后，辞职离开，安安稳稳的过完自己的下半辈子；可是现在瓦斯汀却又是看到了自己大选的希望。
<br/>　　瓦斯汀看着电视中的新闻直播，立刻打电话叫来了自己的秘书，吩咐道：“我需要立刻跟中方领导人联系！如果我能成功当选，我愿意跟他们完成一笔价值三亿美金的金属矿交易单！另外，比洛多这家伙这家伙一定已经开始安排危机公关，我们一定要把握好这次机会！只要能救下那些人，这就是我们对中方最大的诚意！”
<br/>　　对秘书仔细地吩咐了一番后，瓦斯汀靠坐在椅子中，看着电视中的现场报道，脸上挂着轻松的笑容，这对于瓦斯汀来说真的是峰回路转，柳暗花明，比洛多这个家伙在大势在握的情况下，居然会犯下这样的错误！
<br/>　　误伤平民，这件事情如果好好运作，绝对可以把比洛多踢出这次大选。
<br/>　　……
<br/>　　“去A2点汇合！”
<br/>　　齐凡看到梅庆伟驾驶的“支奴干”运输机因为火箭弹的干扰，而不得不拉升高度，甩脱开火箭弹攻击后，立刻通过步话机跟梅庆伟取得联系。
<br/>　　比洛多的撤退命令，很快就传到了现场对罗昊、齐凡等人围追堵截的机动部队手中，原本在天蛇和玄龟驾驶的防弹突击车后面紧追不舍的机动部队突然调转方向，脱离了战场。
<br/>　　失去了机动部队的支援，force-recon这边立刻就变得势单力薄起来。
<br/>　　Force-recon的行动指挥官发现情况的异样后，立刻就想要跟比洛多进行联系质问原因，可是这次他却是彻底失去了跟比洛多的联系。
<br/>　　“我们背后的追兵似乎少了。”天蛇驾驶着突击车看了眼后视镜，说道。
<br/>　　“不是似乎，是的确少了。”司徒空对他纠正道：“比洛多的机动部队撤出了行动，放弃了对我们的追击。”
