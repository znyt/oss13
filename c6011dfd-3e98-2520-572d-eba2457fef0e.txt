        &lt;h3&gt;第三卷第一百八十二章  郝景文到扬州&lt;/h3&gt;
<br/>
<br/>        郝景文随着陈思宇到了扬州，他们的物流和租赁已经遍地开花，他郝景文只要想去的地方都是会有人亲自接机的。
<br/>
<br/>        扬州的人早就将车开到杭州机场，等着他们年轻的老总。
<br/>
<br/>        “郝总。”扬州区域总经理蓝涛开着他的奥迪a8l来接郝景文的，他看了眼郝景文身边的陈思宇，迅速接过陈思宇手中的行李箱。
<br/>
<br/>        “不，不用，我自己提。”陈思宇有些难为情。
<br/>
<br/>        郝景文看了一眼陈思宇，陈思宇才将手中的行李箱交给蓝涛。
<br/>
<br/>        郝景文等着蓝涛将行李箱放好，打开车门，等着陈思宇坐好，然后才上车。
<br/>
<br/>        “郝总，我将酒店都定好了，我们直接去酒店？”蓝涛不知道陈思宇是什么情况，但是也能看出来这个小姑娘是郝景文的女朋友。
<br/>
<br/>        “思宇，我们应该去哪里？”郝景文问陈思宇。
<br/>
<br/>        “回家啊，住什么酒店呢。”陈思宇满心欢喜，到现在心里都和吃了蜂蜜一样。
<br/>
<br/>        “那就回家吧，不去酒店了。”郝景文对蓝涛说。
<br/>
<br/>        “回家是可以的，只是是要回我家吗？”蓝涛为难的说，我咋知道你的家在哪里啊。
<br/>
<br/>        “思宇，你得给蓝总说地址呢。”虽然给陈思宇安排了机票等一系列的活动并跟着陈思宇一起回家，但是郝景文不知道陈思宇家的具体地址啊。
<br/>
<br/>        “嘿嘿嘿，我竟然忘记了告诉你地址，不好意思啊，蓝总，我家在公园一号。”陈思宇开心的说。
<br/>
<br/>        “公园一号？”郝景文问陈思宇。
<br/>
<br/>        “嗯嗯。”陈思宇点点头。
<br/>
<br/>        郝景文看了一眼陈思宇，公园一号的房价他是知道的，陈思宇家里竟然在公园一号，出乎了他的意料。
<br/>
<br/>        郝景文的家里教育环境宽松，父亲从来不以家庭条件来要去郝景文娶妻，郝景文喜欢上了陈思宇，他独独就是喜欢她这个人，却没有过多的去关注她的家庭条件，所以陈思宇说出公园一号时他还是有点吃惊。
<br/>
<br/>        “我爸是个乡镇企业家，家里没有太多的钱，但是他想给我妈和我一个舒适的环境，所以就在公园一号给我们买了一套房子。”陈思宇轻声的解释。
<br/>
<br/>        “你爸爸对你们真好。”郝景文宠溺的看着陈思宇：“我会对你比他对你更好。”
<br/>
<br/>        陈思宇脸又红了：“我又不是你的女儿。”
<br/>
<br/>        郝景文：“你是我女儿的老妈！”
<br/>
<br/>        陈思宇：“文文，我发现你特会撩，八字还没一撇呢。”
<br/>
<br/>        郝景文：“思宇，我发现你情商总是会抛锚。”
<br/>
<br/>        开着车的蓝涛本来已经憋出内伤了，听见郝景文的话“咳咳咳”，他憋成咳嗽了好不好？
<br/>
<br/>        陈思宇挽着郝景文的胳膊，大大方方的走过小区，看你们谁还敢说我没有男朋友？
<br/>
<br/>        郝景文：“为什么不让蓝涛开车进来？小区不让进车吗？”
<br/>
<br/>        陈思宇：“像你这么帅气的人不拉出来溜溜就不是我陈思宇了。”
<br/>
<br/>        郝景文：“原来你是在溜我。”
<br/>
<br/>        陈思宇：“别乱说话，恩爱点。”陈思宇看见对面走过来了一对母女。
<br/>
<br/>        “哎哟，思宇回来了啊，这是带着男朋友回家了？”
<br/>
<br/>        陈思宇：“嗯，王姨好。”
<br/>
<br/>        王姨：“哎呀，这姑娘终于是开窍了，要嫁出去了。”
<br/>
<br/>        郝景文：“原来你是嫁不出去啊？”
<br/>
<br/>        陈思宇：“我要是早早就嫁出来了，现在还能便宜了你？”
<br/>
<br/>        ......
<br/>
<br/>        老丈人看女婿，越看越欢喜，陈思宇的父亲是爱女如命，他的老婆不着急女儿的婚事，他作父亲的操碎了心，怕自家的宝贝疙瘩受了委屈，他就用一个男人的心思去观察郝景文的一举一动，郝景文的一举一动都是不经意之间流露出来的，又偷偷的跑去查了一下郝景文的资料，心里对这个未来的女婿心生欢喜。“女儿的幸福指日可待啊！”
<br/>
<br/>        年关，虽说都要放假了，可是大家都很忙，走亲串友，辞年拜节，郝景文也不列外，到了陈家呆了一天，他就得回自己的家里去。
<br/>
<br/>        “叔叔，我明天要回家，年后再来看您。”临别之前，郝景文对陈思宇的父亲说。
<br/>
<br/>        “啊，明天就要回去啊？”
<br/>
<br/>        “是的，我也是很长时间没有回去了。”郝景文确实有几个月都没有回家了。
<br/>
<br/>        “那明天带着思宇吧，她也应该过去看看。”
<br/>
<br/>        “啊？”郝景文吃了一惊，他没有想到陈思宇的父亲会让陈思宇和他一起走：
<br/>
<br/>        “马上就要过年了，我带她回去就没有时间送回来，您和阿姨两人在家？”
<br/>
<br/>        “我们准备去她姥姥家陪她姥姥，你带着她吧，年后给我完完整整的送回来就是。”陈思宇的爸爸说。
<br/>
<br/>        “爸，你这个重色轻友的人，你说你是不是看上文文那张脸了？你就这样将你女儿送出去了？”陈思宇从外面进来听见她老爸和郝景文的谈话又气又笑。
<br/>
<br/>        “我重色轻友也是为了你好，你竟然不领我的情，太伤人心了。”陈思宇的爸爸故作难过的模样。
<br/>
<br/>        “我说你们这一对活宝什么时候能长大啊，一个个的幼稚得不行了。”陈妈妈走进来笑着说
<br/>
<br/>        “景文，我们家的这俩个就是长不大的孩子，你担待着点。”
<br/>
<br/>        “不，阿姨，我和他们一样幼稚，我也是长不大的孩子。”郝景文很喜欢陈家的这种甜蜜蜜温馨的感觉，像陈思宇所说，她爸爸是一个乡镇企业家，母亲是一个普通的工作人员，不是很有钱，却很有爱。
<br/>
<br/>        第二天，郝景文带着他女儿的妈妈回到了福建。
<br/>
<br/>        郝爸爸和郝妈妈见到儿子带回来一个清秀灵动的小姑娘，心里大喜，儿子终于脱单了，最重要的是，他们不再受提亲之苦，来他们家提亲的人都快将他们的门槛踏破了。
<br/>
<br/>        别人家的是生女儿提亲的多，他们家的是生儿子提亲的多，搞得他们俩个老人家不胜其烦。
<br/>
<br/>        “文仔，这是确定了？要是确定了我们过几天就请亲戚朋友来坐坐，让大家见见思宇。”郝爸爸问郝景文。
<br/>
<br/>        “她都跟着来了，我不确定能行吗？这一辈子只能交代给她了。”郝景文笑着说。
<br/>
<br/>        “好像谁没有过女朋友一样，你嘚瑟啥呢？”郝爸爸看着儿子那满眼的笑意“哼”了一身转身去安排年后请亲戚朋友的事情去了。
<br/>
<br/>        看着爸爸离去，郝景文也陷入沉思，公开恋情得想陈思宇求婚吧？
