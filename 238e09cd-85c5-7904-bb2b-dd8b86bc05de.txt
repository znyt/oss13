　　徐倩倩顿时脸色煞白，但是又不敢反抗，吓得娇躯直颤。
<br/>　　就在这时，一旁的李虎霍然站了起来，直接挡在了徐倩倩的面前，怒声斥道：
<br/>　　“你是什么人？凭什么和倩倩这么说话？立即向她道歉！”
<br/>　　那黄发青年听到这话，先是一愣，继而仿佛听到了世上最好笑的笑话，忍不住哈哈大笑起来，
<br/>　　“死胖子，你可知道我是谁吗？竟然敢和我这么说话？”
<br/>　　李虎两眼一瞪，
<br/>　　“我管你是谁！倩倩是我朋友，你想欺负她，要先问问我答不答应！”
<br/>　　一旁的徐倩倩听到李虎这话，非但没有任何感激之色，反而脸色更加的惨白，连忙扯了扯李虎的衣袖，满脸急切之色道：
<br/>　　“李虎，你快少说两句！这位是豹哥，暗魅酒吧就是他的产业……”
<br/>　　“什么！”
<br/>　　听到这话，李虎以及唐威等人，顿时纷纷脸色大变。
<br/>　　他们之前不止一次听人说过，这家酒吧的幕后老板，混迹黑白两道，背景可是相当的深。
<br/>　　没想到居然就是眼前这个年轻人！
<br/>　　‘豹哥’看到众人的反应，脸上顿时露出一抹得意之色，一脸邪笑道：
<br/>　　“死胖子，你倒是挺有种嘛！看在你还算有几分胆识的份上，本少也不为难你，只要你乖乖跪下给本少磕三个响头，本少便饶你一次，如何？”
<br/>　　还没等李虎开口，一旁的徐倩倩当即又拉了拉他的胳膊，
<br/>　　“李虎，难得豹哥不和你计较，你还不快跪下道歉！”
<br/>　　李虎听到这话，简直无法相信自己的耳朵，
<br/>　　“倩倩，你怎么能说出这种话？我李虎堂堂男儿，怎么能……”
<br/>　　还没等李虎说完，徐倩倩直接冷笑打断道：
<br/>　　“得了吧！识时务者为俊杰，好汉还不吃眼前亏呢！豹哥可不是我们能得罪的起的，你自己不要命，难道想要把我们也一起连累了吗？”
<br/>　　蹬蹬蹬！
<br/>　　李虎听到这话，顿时如遭雷劈，万万没有想到，自己心仪的女孩，竟然会说出这番话！
<br/>　　“倩倩，我可是为了帮你，你怎么能说出这种话？”
<br/>　　徐倩倩在李虎的质问之下，脸色也顿时有些不自然，道：
<br/>　　“我这也是为了你好！得罪了豹哥，我们这些人，都不会有好果子吃！”
<br/>　　说完这话，徐倩倩扭着小蛮腰，凑到豹哥的身前，赔笑道：
<br/>　　“豹哥，我朋友不懂事，要不您看在我的面子上，就不要和他一般见识了，好不好？”
<br/>　　然而，
<br/>　　“啪！”
<br/>　　面对徐倩倩的求情，豹哥抬手就是一巴掌，险些将徐倩倩扇飞，倒在了一旁的沙发上面。
<br/>　　“臭表子！你还真把自己当根葱了？本少的事情你也敢管？”
<br/>　　徐倩倩羞得满脸通红，捂着脸不敢再吭声。
<br/>　　李虎见状，顿时彻底怒了，气得头上青筋直冒，
<br/>　　“我特么和你拼了！”
<br/>　　李虎大吼了一声，直接抄起一只红酒瓶，就朝豹哥扑了过去。
<br/>　　然而那豹哥却不慌不忙，脚下微微一错，便从容躲开了李虎的攻击，然后直接一拳斜出，直奔李虎的小腹砸落。
<br/>　　一旁坐在角落的叶尘，看到这一幕，顿时眉头一皱。
<br/>　　以他如今的实力，想要杀掉豹哥这种蝼蚁，只须一道念头即可。
<br/>　　不过，他有意要让李虎看清楚徐倩倩的真面目，所以最终选择暂时冷眼旁观。
<br/>　　砰！
<br/>　　豹哥一拳砸在李虎的小腹上，后者顿时身体蜷缩，倒在了一旁的沙发上，痛苦的呻吟起来。
<br/>　　“老四！”
<br/>　　唐威见状，当即大喝了一声，同时飞起一脚，朝豹哥踹了过来。
<br/>　　“嗯？”
<br/>　　豹哥显然也没有想到，这群学生里面，居然还有练家子，当即双臂交错，险而又险的挡下了唐威这一脚。
<br/>　　蹬蹬蹬！
<br/>　　唐威的武功虽然不高，但好歹也是内劲武者，豹哥同样也不是什么高手，顿时一连倒退了七八步，方才站稳住身形，连同酒吧里的桌椅也撞坏了不少，顿时把周围众人的目光，也全都被吸引了过来。
<br/>　　“我去！我没有看错吧？有人竟然对豹哥出手了！”
<br/>　　“什么人这么大胆？不要命了吗？”
<br/>　　“之前有人得罪了豹哥，第二天就被人丢进了天江喂鱼！”
<br/>　　……
<br/>　　众人议论纷纷，而豹哥被唐威一脚打退，也顿时觉得有些脸上有些挂不住，眼眸中露出一抹浓浓的煞气，
<br/>　　“好小子！竟然敢在老子的地盘动手，今天你们这些人，一个也别想走！”
<br/>　　随着豹哥声音落下，
<br/>　　哗啦啦！
<br/>　　周围顿时窜出十多名身材魁梧的彪形大汉，迅速将整个卡座周围都包围了起来。
<br/>　　众人顿时纷纷脸色大变，连唐威也脸色凝重起来。
<br/>　　眼前的这些大汉，显然都是练家子，以他武功，自保或许还可以勉强做到，可是想要护住其他几人的周全，那就绝对不可能了。
<br/>　　而就在这时，
<br/>　　徐倩倩忽然一把甩开李虎的胳膊，快步冲到豹哥的跟前，一脸讨好道：
<br/>　　“豹哥，我和他们几个真的不熟，求您饶了我和我的室友吧！”
<br/>　　“倩倩，你……”
<br/>　　李虎看到这一幕，顿时脸色煞白，
<br/>　　他怎么也没有想到，自己为这个女人出头，这个女人却在关键时刻，为了保全自己，竟然将他一脚踢开！
<br/>　　豹哥听到这话，顿时桀桀一笑，两眼在徐倩倩和她的几名室友身上瞄了一眼，一脸婬色道：
<br/>　　“好你个小骚货，你倒是挺识时务的嘛！也罢，只要你们乖乖去把我的那位贵客伺候舒服了，此事我可以既往不咎！”
<br/>　　徐倩倩听到这话，顿时松了一口气，一脸风骚道：
<br/>　　“多谢豹哥~~”
<br/>　　说完这话，又冲着身后另外三名女生试了一眼色，
<br/>　　“你们几个，还不快过来谢谢豹哥！”
<br/>　　那三名女生何曾见过这架势，早就快吓坏了，当即纷纷包围圈中逃了出来。
<br/>　　于是整个卡座里面，便只剩下李虎、唐威、陈东和叶尘四人了。
<br/>　　李虎三人的脸色，顿时变得越发难看，尤其是李虎，万万没有想到，自己心仪的女神，竟然有如此风骚淫荡的一面。
<br/>　　“唉！”
<br/>　　眼看到了这个地步，叶尘知道自己不能再坐视不理了，微微叹了一口，缓缓从角落里站了起来，走到众人跟前，抬头瞥了一眼豹哥，缓缓开口道：
<br/>　　“豹哥是吧？带上你的人，立即消失！否则，我保证，你看不到明天的太阳！”
