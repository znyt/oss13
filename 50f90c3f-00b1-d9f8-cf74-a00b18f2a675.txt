　　公孙林显然没有想到，叶尘居然这么快就追了上来，吓得他慌忙还要转身逃窜。
<br/>　　“还想跑？”
<br/>　　叶尘虚手一抓，一道硕大的金色巨掌，已经将公孙林，死死握在手中。
<br/>　　“不要！你不能杀我！我还有一个大秘密要告诉你！”
<br/>　　叶尘顿时乐了，
<br/>　　“可惜，我已经没有兴趣知道了！”
<br/>　　说着，叶尘手上微微发力，公孙林顿时一声惨叫，
<br/>　　“叶狂仙！你杀了我，药王宗一定不会放过你的！你以为宗门只派我一人前来吗？你太天真了！啊！！”
<br/>　　“彭！”
<br/>　　公孙林的话还没有说完，整个人瞬间爆裂，化成了一团血雾，彻底命丧于此。
<br/>　　不过，
<br/>　　他的最后那一句话，却让叶尘眉头一皱，旋即想到了什么，
<br/>　　“不好！他们这是调虎离山！”
<br/>　　一念及此，叶尘当即掏出手机，拨通了神龙卫暗部的电话，
<br/>　　“立即给我确认，萧若曦和苏蔓的行踪！”
<br/>　　如果公孙林所言不错，药王宗派了不止他一个人来，那么其他人的目标，必定是叶尘的其他朋友或亲人。
<br/>　　云州是叶尘的大本营，有狂仙门坐镇，周围还有军队和神龙卫保护，药王宗未必能讨得了好，所以他们的目标，极有可能会放在，身在天海市的萧若曦和苏蔓两人身上！
<br/>　　“但愿我的猜测是错的！”
<br/>　　叶尘自言自语了一句，同时迅速赶回学校，回到了唐清雅身边。
<br/>　　“清雅，京都不安全，你随我一起回天南吧，等我处理完一些事情之后，你再继续回来读书！”
<br/>　　唐清雅也清楚，自己一旦被人掳走，肯定会被人拿来要挟叶尘，所以没有犹豫，当即点了头，答应了下来。
<br/>　　于是，
<br/>　　两人又直接离开了学校，找到了唐清山。
<br/>　　而就在这时，
<br/>　　神龙卫暗部的电话也打了回来，
<br/>　　“叶帅，不好了！若曦小姐在不久前被人袭击！”
<br/>　　“什么！”
<br/>　　叶尘顿时两眼一瞪，
<br/>　　电话那头继续道：
<br/>　　“不过，据负责保护若曦小姐的兄弟说，若曦小姐在最后关头，忽然爆发出来一股强大的力量，将偷袭者就地轰杀！但之后若曦小姐也晕倒了……”
<br/>　　叶尘听到这话，顿时松了一口长气，旋即脸上有浮现出一抹浓浓的杀意！
<br/>　　曦月和他一样，同样是转世而来，而且若论底蕴和修炼的时间，曦月还在叶尘之上，自然也有一些保命的手段。
<br/>　　不过，她如今毕竟只是残魂状态，虽然轰杀了偷袭者，自己恐怕也要付出不小的代价！
<br/>　　一想到这里，叶尘就气得牙根直痒痒，同时心急如焚，恨不得赶紧赶回天海。
<br/>　　“对了，蔓姐呢？她怎么样了？”
<br/>　　叶尘忽然想起苏蔓，当即继续追问道。
<br/>　　电话那头，沉默了片刻，
<br/>　　“叶帅，抱歉，苏蔓小姐好像……好像失踪了！”
<br/>　　“你说什么！”
<br/>　　叶尘才刚刚松了一口气，一颗心一下子就提了起来，继而冲着电话那头大吼，
<br/>　　“查！立即给我查！在我回到天海之前，务必给我调查清楚怎么回事！”
<br/>　　叶尘忽然大吼了一声，一旁的众人，顿时全都惊呆了。
<br/>　　这还是他们，第一次看到，叶尘发这么大的火！
<br/>　　“叶尘，出什么事了？”
<br/>　　唐清雅忍不住开口问道。
<br/>　　叶尘并没有离开回答，而是先冲唐清山道：
<br/>　　“立刻返回天海！”
<br/>　　等到众人上了车，车子朝飞奔而出，叶尘深吸了一口气，平复了一下自己的情绪，方才对一旁的唐清雅，缓缓道：
<br/>　　“蔓姐可能出事了！”
<br/>　　唐清雅顿时一惊，不由得捂着小嘴，
<br/>　　“怎么会这样？”
<br/>　　叶尘摆了摆手，
<br/>　　“现在情况还不是特别清楚，蔓姐极有可能是遇到了和你相同的状况！”
<br/>　　唐清雅顿时一阵自责，
<br/>　　“都怪我，如果不是因为我，说不定苏蔓姐就不会出事了……”
<br/>　　叶尘摇了摇头，
<br/>　　“清雅，这事和你无关，你千万不要这么想！”
<br/>　　“而且从那个公孙林对付你的情况来看，他们应该只是想对我的亲人朋友下手，来以此要挟我！”
<br/>　　“所以，蔓姐即便是真的被他的同伙掳走了，应该暂时也不至于会有生命危险！”
<br/>　　叶尘虽然嘴上这么说，可是心里却担心的不行。
<br/>　　他这段时间，整日和曦月待在一起，倒是忽略了自己的这个姐姐。
<br/>　　苏蔓虽然不是他的亲姐姐，可是对于叶尘来说，却比亲姐姐还有亲，是他最为重要的家人！
<br/>　　“药王宗！你们这是在找死！！”
<br/>　　叶尘越想越不安，身上不由自主的就升起了一股强大的杀意。
<br/>　　……
<br/>　　众人赶到机场之后，叶尘却并没有上飞机，
<br/>　　“清山兄，你先带清雅直接返回云州，我先行一步！”
<br/>　　说完这话，叶尘直接纵身一跃，犹如火箭般冲而起，继而随着一道紫色剑芒，又如同流星般，朝南面爆射而去！
<br/>　　这情景，看的唐清山等人一阵咋舌，不由得纷纷感慨，
<br/>　　“飞天遁地！御空飞行！果然是仙人才具备的能力啊！”
<br/>　　“这速度，恐怕比超音速飞机还要快！”
<br/>　　“事不宜迟，我们也要按照叶先生所说，速速赶回云州！”
<br/>　　……
<br/>　　只用了一个多小时，叶尘便已经从京都，回到了天海市的上空。
<br/>　　苏蔓在天海市的一家外贸公司实习，所以在公司附近的一个中档小区租了一套单身公寓，叶尘之前来过一次。
<br/>　　来到那处小区前，
<br/>　　叶尘神念一扫，已经迅速锁定了，神龙卫暗部的行踪，于是身体一晃，便直接来到众人跟前。
<br/>　　“叶帅！”
<br/>　　神龙卫众人看到叶尘，犹如天神下凡一般，突然出现在他们的面前，顿时一个个又惊又喜。
<br/>　　叶尘则一脸的阴沉，
<br/>　　“到底是什么情况？”
<br/>　　为首的一名暗部首领，当即上前一步道：
<br/>　　“回叶帅！我们奉命，负责保护苏蔓小姐的安全，今天苏蔓小姐从公司下班回来之后，一切都像往常一样，并没有发现任何异样，可是之前接到叶帅的提醒之后，我们冲进去之后方才发现，苏蔓小姐不见了，不过……”
<br/>　　说到这里，那人又冲怀里掏出了一张古老的帛纸，
<br/>　　“叶帅，我们在屋里发现了这个！”
<br/>　　叶尘眉头一皱，接过帛纸，大开一看，脸上顿时浮现出一抹浓浓的杀意。
