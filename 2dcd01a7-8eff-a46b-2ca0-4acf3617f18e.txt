　　猎妖殿，任务处。
<br/>　　工作人员看了眼萧逸的令牌，点了点头。
<br/>　　“嗯，四级猎妖师，易霄。”
<br/>　　“任务大多是在百武城猎妖殿接的，怎么横跨十数城，到千乐城来了。”
<br/>　　工作人员疑惑地问了句。
<br/>　　千乐城，便是这十数城范围内的大城。
<br/>　　千乐城，十数城范围；上方，毗邻坐落着裂天剑派的百武城十数城范围；右方，则接连妖兽纵横的东荒十八城。
<br/>　　如此得天独厚的环境，让得这里商业家族无数，商业极为繁华。
<br/>　　也因此，这里极尽奢华，乃是北山郡极有名的游乐、享受之处。
<br/>　　这里，仿佛是一个金钱横生的大城。
<br/>　　只要有钱，可以得到一切意想不到的美妙。
<br/>　　“一路猎妖，途径此地罢了。”萧逸淡淡地回了一句。
<br/>　　紫炎易霄，在百武城和破玄城那边都有名。
<br/>　　而在千乐城，并非什么名人。
<br/>　　毕竟，易霄只是个天才，而并非真的是名震一方的强者。
<br/>　　“哦。”任务处的人员点点头，而后仔细地对照了一下任务手册。
<br/>　　“总共100个四级任务，悉数完成。”
<br/>　　“这是你的任务悬赏金，任务点也记录好了。”
<br/>　　“还有什么需要我帮忙的吗？”
<br/>　　“没了。”萧逸摇摇头，打量了一下周围。
<br/>　　果然，如他所料，每个猎妖殿，里面的布局几乎都一样。
<br/>　　他轻车熟路地去了任务栏，再接了大量的任务后，快步离开了猎妖殿。
<br/>　　……
<br/>　　时间飞逝，不知不觉间，他竟在千乐城待了两个月。
<br/>　　完成了近百个五级任务。
<br/>　　全是千乐城内，更大商业家族到猎妖殿发布的，寻求炼药师的任务。
<br/>　　所炼丹药，最差都是四品，大多是四品巅峰，甚至有五品丹药。
<br/>　　萧逸现在才明白，炼药师是一个何等有钱的职业。
<br/>　　那些五级的炼药任务，悬赏金最低都在万两以上，随便都是数万两。
<br/>　　两个月来，足足攒了一万多任务点，算上前两个月的，足有一万三千多点。
<br/>　　五级任务，任务点足有100点。
<br/>　　而金钱，则攒了数百万两，身家一下就丰厚了起来。
<br/>　　其实，整个北山郡，五品炼药师，不过寥寥几人，全是各大霸主势力中的座上宾。
<br/>　　一般人别说请不动他们，就算请动了，其代价也极大。
<br/>　　故，两月来，易霄二字，在千乐城十数城范围内，几乎成了最炙手可热的存在。
<br/>　　而，正当易霄声名鹊起之时，却忽然神秘地销声匿迹，谁都找不到他。
<br/>　　…….
<br/>　　野外，一道人影快速疾驰着。
<br/>　　人影自然是萧逸。
<br/>　　他正从千乐城十数城范围，赶回百武城。
<br/>　　离开剑派，已经四个月了，他仍旧没能破丹化脉。
<br/>　　而他现在，还有更重要的事要处理。
<br/>　　因为，算起时间来，他离开萧家，接近一年了。
<br/>　　猎妖令，一年的保护期，也没剩几个月了。
<br/>　　猎妖令，是他离开萧家三月后，在猎妖殿兑换的。
<br/>　　他要赶回百武城猎妖殿，问清楚那总执事要如何延续猎妖令的时间。
<br/>　　是数月后，猎妖令失效后，才能再兑换；抑或，可以提前兑换，并将新的猎妖令送回家族。
<br/>　　一直以来，萧家的安危，是萧逸最上心之事。
<br/>　　十天后，萧逸回到百武城。
<br/>　　……
<br/>　　百武城，猎妖殿内。
<br/>　　仍旧是上次的那个女子，将萧逸带到后堂的工作人员办公处。
<br/>　　仍旧是居于最中间的那间房子。
<br/>　　但，在萧逸面前的，却不再是那个‘猥琐狡猾’的老者，而是一个中年人。
<br/>　　“总执事不在吗？”萧逸皱眉问道。
<br/>　　中年人，其实才是这座猎妖殿的负责人，也是真正的总执事。
<br/>　　只不过，他却笑着说道，“我便是总执事。”
<br/>　　“上次的那位，已经调到别的猎妖殿了，我则是从别的猎妖殿调过来的。”
<br/>　　中年人并没有将上次那位老者的真实身份道出，而是转换了话题。
<br/>　　“紫炎易霄，我们百武城风头最盛的天才，亦是我们猎妖殿的执事。”
<br/>　　“今日前来，所谓何事阿？”
<br/>　　萧逸皱了皱眉头，虽说见不到上次的老者，有些意外。
<br/>　　但他还是选择相信面前的中年人。
<br/>　　猎妖殿，作为大陆最古老的势力之一。不与任何势力打交道，也不与任何人有利益关系。
<br/>　　它存在的意义，只是帮助天下猎妖师，以及猎杀天下妖兽。
<br/>　　所以，萧逸并不需要顾忌，将心头的疑惑问了出来。
<br/>　　那中年人闻言，点了点头，道，“哦，原来是此事。”
<br/>　　“猎妖令，确实只有1年的保护时间。不过，往后你不用为此事而特地过来了。”
<br/>　　“延续猎妖令时间所需的任务点，我们会直接在你的令牌上扣除。”
<br/>　　“嗯？”萧逸面露疑惑之色，问道，“不是说，猎妖令过期后，必须自己亲自来兑换，并亲自交回家族吗？”
<br/>　　中年人忽然站了起来，走到萧逸面前，重重地拍了拍萧逸的肩膀。
<br/>　　而后笑道，“你可是我们北山郡猎妖殿历史上最年轻的执事。”
<br/>　　“自然是有一定特权的。”
<br/>　　“额。”萧逸一愣，面前这总执事，挺好说话的。
<br/>　　不过，萧逸也不疑有他。毕竟，猎妖殿九大总执事，可是响当当的大人物，没人敢冒认，也不会欺骗他这样的小人物。
<br/>　　这时，那中年人见萧逸面露迟疑，当即道，“猎妖令，分子母令。母令有特殊的力量，会随时间而自动消散。”
<br/>　　“待得里面的力量完全消散，便代表这枚猎妖令已经失效。”
<br/>　　“你可随时查看自己的母令，里面是否有力量存在。”
<br/>　　“若有，便代表猎妖令仍处于有效时间。”
<br/>　　萧逸点点头，这个他知道。
<br/>　　猎妖殿的讲解书籍里，记载着这一点，这也是兑换者判断猎妖令失效的最直接办法。
<br/>　　中年人笑笑，道，“往后，你可以不必为了猎妖令的事特地来这里了。”
<br/>　　“谢总执事。”萧逸礼貌地行了一礼。
<br/>　　中年人摆摆手，道，“还有其它事吗？”
<br/>　　“有。”萧逸点点头，道，“难得来猎妖殿一趟，想看看有什么需要的东西兑换。”
<br/>　　“哦。”中年人笑道，“我看过你的记录，这几个月，你的任务点可是蹭蹭地上涨。”
<br/>　　“替猎妖殿完成了如此多任务，自然是要得到应有的奖励的。”
<br/>　　“妙妙，把兑换图册拿来。”中年人对身旁的女子吩咐道。
<br/>　　不多时，女子将兑换图册拿来。
<br/>　　萧逸其实早有想要兑换的东西。
<br/>　　上一次，兑换猎妖令时，他已经大概讲兑换图册上的宝物记了下来。
<br/>　　而他想要兑换之物，便是…..
<br/>　　“我要兑换此物，幻羽之翼。”
<br/>　　幻羽之翼，稀有的地阶武技。并非攻击型，而是辅助型。
<br/>　　练成者，可幻化出羽翼，翱翔天际。
<br/>　　“幻羽之翼？”中年人一愣，疑惑道，“这东西可是个鸡肋，换来何用？”
<br/>　　“以你洞玄六重的修为，随时可御空飞行。”
<br/>　　数月前，萧逸打败慕容杀之事，百武城内人所皆知。
<br/>　　自然的，总执事以为他是洞玄六重修为。
<br/>　　“哦，对了。这武技虽然攻击性很低，但对飞行还是有很大帮助的。”
<br/>　　“你可是猎妖师，用来追杀或者逃命都是大有用途。”
<br/>　　“确定要换吗？”中年人问道。
<br/>　　“嗯。”萧逸点点头。
<br/>　　“好，待会儿幻羽之翼的武技会拿来。相应的，会在你令牌上扣除一万任务点。”
<br/>　　萧逸自己有分寸，一万三千多任务点。
<br/>　　两千是用来延续猎妖令时间，一万则是用来兑换幻羽之翼。
<br/>　　待得兑换手续完成，萧逸对那中年人拱了拱手，礼貌地道了个别，便离开了猎妖殿。
<br/>　　…..
<br/>　　房间内，中年人看着萧逸已然离开，淡淡地笑了笑。
<br/>　　身旁的女子，轻笑道，“总执事，我还是第一次见你说谎呢。算起来，易霄先生，已经被分殿主和你各骗了一次呢。”
<br/>　　中年人苦笑一声，道，“我这算说谎吗？”
<br/>　　“虽然不知师傅为什么要帮这少年，不过，师傅既然吩咐我不能说出实情，我只能随便编个慌了。”
<br/>　　事实上，猎妖令的规矩，谁都无法改变。
<br/>　　保护时间失效后，兑换者必须亲自来兑换，亲自送回家族。
<br/>　　然而，当初分殿主给萧逸兑换的，乃是殿主级别猎妖令，有效时间为五年。
<br/>　　自然的，这五年内，萧逸都无需来重新兑换。
<br/>　　中年人，也就是总执事，没办法，只能编了个谎。
<br/>(本章完)
