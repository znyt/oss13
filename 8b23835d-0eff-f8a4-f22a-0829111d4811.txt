<br/>　　不死魔尊大声咆哮一声，魔意席卷，他的身形也是猛然追了上去，一下子离开了密室之中。
<br/>　　楚风眠离开了密室，直接便是按照血脉的指引，向着颜古龙帝洞府中最深处飞去。
<br/>　　这越是接近到颜古龙帝洞府的深处，其中的禁制也就越是强大，楚风眠想要从不死魔尊的手中逃脱。
<br/>　　他必须要借助这颜古龙帝洞府中禁制的力量，这些禁制楚风眠可以轻易穿过，却可以拖延一下不死魔尊。

<br/>　　这颜古龙帝的洞府中，沉睡的真龙祖灵，肯定远远不止那七尊，楚风眠只要是唤醒更多的真龙祖灵，也可以有更多的力量，相助与他。
<br/>　　而且这越是接近到这颜古龙帝洞府的深处，楚风眠心中也就越是感觉到，这颜古龙帝洞府中真正的宝藏，绝对是非同小可。
<br/>　　一旦是可以得到，楚风眠的实力将会有着一种质的飞跃。
<br/>　　楚风眠感觉到，他若是可以得到这其中的宝藏，就算是再对付不死魔尊，都未必是没有一战之力。
<br/>　　楚风眠连连不知道是穿越了多少禁制，冲入这颜古龙帝洞府的深处。
<br/>　　一路上畅通无阻，以楚风眠的速度，一瞬间便是数十公里的距离。
<br/>　　但是楚风眠的背后，那庞大的魔意，却是丝毫没有被甩开，反而是越来越近。
<br/>　　不死魔尊，完全没有放过楚风眠的打算。
<br/>　　就在楚风眠不断的向着颜古龙帝洞府深处飞遁的同时，他也是在后面，紧追不舍。
<br/>　　这一路上的禁止，对许多武者都是致命的，可是对不死魔尊这样的大人物来说，却是根本远远无法阻止他的步伐。
<br/>　　甚至是现在，楚风眠都要被追上。
<br/>　　三个呼吸的时间，这不死魔尊，已经是追了上来。
<br/>　　“可恶！这不死魔尊是拼了全力，今日也要杀了我！”
<br/>　　楚风眠心中明白，这不死魔尊已经是动了必杀之心。
<br/>　　这样一尊生死境高阶的大人物，动了必杀之心，到底是有着多恐怖。
<br/>　　不死魔尊，根本便是无视无数禁制，横冲直撞，直接冲入到了颜古龙帝的洞府中去。

<br/>　　不死魔尊的手中，突然出现了一座七层宝塔，猛然对着楚风眠落了下去。

<br/>　　其中蕴含这几分真正万魔覆天塔的力量，比起丰饶碑这样的天级灵器，都只差一步。
