        两个人面对面坐在大床上，傅夜沉见童筱颖一脸认真地给他的两个脚指头抹药，还把他当小孩子替他“吹吹”的样子，就觉得好笑，同时又觉得自己很幸福。
<br/>
<br/>        “我给你办了法国的绿卡，护照、签证那些资料和身份证一起，明天就会被快递员送上门。”傅夜沉的手肘抵在膝盖上，手背撑着太阳穴，凝视着身着白色吊带睡裙，披散着长发，拿着棉签和药油正给他另一只脚的大拇指抹药的童筱颖说道。
<br/>
<br/>        白色丝绸质地的裙摆下，两条雪白的大腿很是诱人，她还盘腿而坐，微微俯身，虽然只是一个不经意间的动作，胸前和两腿间的春光无限，全部尽收他傅夜沉的眼底。
<br/>
<br/>        这小女人又惹得他一身热火朝天。
<br/>
<br/>        童筱颖丝毫未察觉到自己的走光，认认真真地在给他擦药，头也不抬地回答道：“那你给我订明天晚上回国的机票吧！”
<br/>
<br/>        “这么快就要走？”傅夜沉皱了下眉头，抿了抿薄唇。
<br/>
<br/>        童筱颖接着回答道：“市体育馆的那个项目已经进入工程预算造价阶段了，孙部长跟我说，预算造价师对我的设计图有疑问，希望我能回去，参加图纸答疑。”
<br/>
<br/>        “这种事情，找人代劳就行了。”傅夜沉云淡风轻地说。
<br/>
<br/>        童筱颖抬眸看了傅夜沉一眼，莞尔一笑：“我跟你们公司是签了工作合同的！这是我的本职工作，不能不去做。”
<br/>
<br/>        “那参加完图纸答疑后呢？”傅夜沉意味深长地对上童筱颖的眼睛。
<br/>
<br/>        童筱颖嘴角微扬：“我会安排好我的工作时间，以及怀宝宝的时间。”
<br/>
<br/>        “还有。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “回去后，不要背着我去见不该见的人。”
<br/>
<br/>        “什么不该见的人？”童筱颖一脸迷茫。
<br/>
<br/>        傅夜沉也不打算藏着掖着，开门见山地坦白道：“比如说，阿景。还有，夜惑里的那个十爷。”
<br/>
<br/>        听到傅夜沉提及“十爷”，童筱颖身心一怔。
<br/>
<br/>        “我和你洞房花烛夜的第二天，你确实没有跟阿景私奔，但是你去见了夜惑里的人。”傅夜沉忽然严肃地说道。
<br/>
<br/>        这件事情，他是事后才知道的。而且，还是他上头那边的人调查出来的。
<br/>
<br/>        他们之所以监视她童筱颖，是觉得她童筱颖跟他们正在追查的人物可能有着什么不可告人的关系。
<br/>
<br/>        “大哥，你别多心。我……”
<br/>
<br/>        “你在乎的那个朋友，马智杰的死，其真正的死因，可能跟夜惑的人有关系。我父亲以及阿景的生母的真正死因，也有可能跟夜惑的人有关。而夜惑里暗中所制的那些不被法律认可的药，跟你外婆的那本‘蛊札’有关。筱颖，我知道你一定知道一些我不知道的事情，但是我不会逼你说出来。我上头的人，想把你抓回去审讯，但我不允许。因为，我会派人去调查清楚。只希望，在这期间，筱颖不要做任何背叛我的事情。”傅夜沉一字一句，说得严肃又认真。
<br/>
<br/>        童筱颖这才恍然大悟，也真正意识到，那位十爷，以及八爷，真的和大哥是死对头。
<br/>
<br/>        “夜惑的费尔南跟我说，十爷很护你，但他不肯跟我坦白其中十爷为什么护你的真正原因。我想你一定知道，可你不愿意告诉我，你和那个十爷之间到底是什么关系。你可以有你的秘密，我唯一能做的就是相信你不会伤害我。”傅夜沉接着说道。
<br/>
<br/>        童筱颖也明白了傅夜沉的意思，脸上的笑容渐敛，忽然很真挚地恳求道：“大哥，如果你们真的查出了什么来，到时候，可不可以放过十爷？”
<br/>
<br/>        “这个我说了不算，得交由法律去制裁。”傅夜沉淡定道。
<br/>
<br/>        童筱颖放下手中的棉签和药油，轻轻地拉了拉傅夜沉睡衣的衣角，再次哀求：“大哥，求你，好不好？”
<br/>
<br/>        雅珊那么爱十爷，万一十爷真有个什么三长两短，她一定会很伤心。
<br/>
<br/>        “筱颖，我问你。如果，我和那个十爷之间必须得死一个。你会最先护着谁？”傅夜沉的黑眸里顿时布上了少许血丝。
<br/>
<br/>        童筱颖身心一怔，却斩钉截铁地回答：“当然是你！”
<br/>
<br/>        她不是圣人，不可能为了朋友牺牲自己的爱人，她没有那么伟大。
<br/>
<br/>        “好！如果，我能帮他，为了你，我会尽量去做。”傅夜沉抬起手来，轻轻地刮了刮童筱颖的鼻梁，“我只能尽量，不一定能保证做到。所以，不管到时候结果如何，你都不能怪我。”
<br/>
<br/>        “嗯，好。”童筱颖欣慰地笑了笑。
<br/>
<br/>        她要求得不多，只希望他能为她尽力而为即可。
