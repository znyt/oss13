<br/>    对于吴云卿来说，这又是一个难眠的夜晚。因为她的丈夫许仕达在晚下班的时候，很难得的没有去应酬什么，早早地回家了。作为一个副市长来说，这样的时候真的是很少的。大多时候的晚，吴云卿都是很难得能够见到许仕达按时回家的，往往是一个人独守空房。这让她都有了几分悔教夫婿觅封侯的感触了。<br/>　　<br/>　　    但这也仅仅是是一种感触而已，对于许仕达能够爬到今天副市长的位置，吴云卿家族是起了很重要的作用的。他的叔叔曾经是岭南省的高官，级别达到副省级，虽然现在已经退居二线，但是在岭南还是有些影响力的。<br/>　　<br/>　　    而许仕达一方，则是很普通的工薪阶层，他的父母都是工人。许仕达能够跟吴云卿走到一起，完全是应该感激吴云卿的慧眼识人。当时吴云卿刚刚大学毕业，分配到了学校教书，一眼看了同校同是教书人的许仕达，她当时觉得许仕达身有一种不甘于平凡的精神，将来一定会出人头地的。<br/>　　<br/>　　    两人结婚之后，吴云卿的叔叔帮许仕达脱离了教职，进入了公务员系统。许仕达本身也确实有能力，再加吴云卿叔叔从旁协助，许仕达也一路青云直，直到现在成为了分管城建的副市长。吴云卿也被身边的人赞说慧眼识人，能够在芸芸众生看许仕达这个有潜力的千里马。<br/>　　<br/>　　    不过随着许仕达职务的升迁，吴云卿也越来越觉得许仕达对她的疏离，夫妻两人几乎很少再有当初刚结婚那种亲密劲了。虽然偶尔许仕达没有什么应酬也没喝酒的时候，会临幸她一次半次的，基本也是应付差事一样的草草了事。<br/>　　<br/>　　    这让吴云卿的日子过得很是苦闷了。表面看，她是副市长夫人，走到哪里别人都是笑脸相迎，尽量的奉承她的样子。但她自己心里却很清楚她过的究竟是什么样的日子。对于一个还不满五十岁的女人来说，光有锦衣玉食是无法让她得到真正的快乐的，她的身体现在很空旷，需要男人的身体来填饱她。<br/>　　<br/>　　    不过吴云卿的家教很严，打小受到的教育让她从心底是无法接受丈夫之外的男人的，因此她虽然很寂寞，却从没把主意打到许仕达之外的男人身。对于丈夫的忙碌她也是能够理解的，她家族那些追逐仕途的男人大多都是这么忙的。<br/>　　<br/>　　    理解是理解，不代表吴云卿不希望得到丈夫的慰藉，情感的空虚可以通过对丈夫为了仕途打拼的理解来解决，身体的空虚却是很难通过理解可以获得解决的。既然今天许仕达罕见的没喝酒很早回家了，吴云卿心里燃起了小小的火苗，也许今天晚丈夫回来这么早是为了跟她好好亲热一下的呢。<br/>　　<br/>　　    于是吴云卿早早的做了晚饭，跟丈夫早早的吃完，然后早早的洗漱完毕进了卧室。她觉得自己已经向丈夫发出了很明确很强烈的信号，许仕达应该清楚的知道她这么做是为了什么，相信过不了一会儿，许仕达应该会到卧室里跟她亲热一番的。<br/>　　<br/>　　    但是很快吴云卿失望了，许仕达吃完饭之后，去了书房，也不知道在书房捣鼓什么一直不肯出来。吴云卿在床等的心焦，翻来覆去的睡不着，但是许仕达依旧呆在书房不肯出来。直到天快亮了，吴云卿终于有了困意，这才在不知不觉睡了过去。<br/>　　<br/>　　    等再醒过来的时候，天光已经大亮。吴云卿看了看床应该是许仕达睡的地方，枕头和被子都还是那么整齐，可见昨晚许仕达根本没有过来睡觉。这家伙居然在书房那里呆了一晚。<br/>　　<br/>　　    起床之后，吴云卿简单的梳洗了一下，去书房那里看了看，许仕达已经不在书房，应该是班去了。她过去书桌那里收拾了一下桌被许仕达翻乱了的东西。<br/>　　<br/>　　    她只是把东西规整一下的，书房这里向来是许仕达的领地，基本她是很少动许仕达的东西的。她的家族很多人都是官场人，知道丈夫的一些东西不能乱动，因为很可能涉及到一些机密什么的。<br/>　　<br/>　　    当她收拾好桌面的东西的时候，一低头看到了放在桌子下左手边的那个保险柜的钥匙没拔下来，想来许仕达早走得很急，并没有把保险柜锁好走了。虽然在家，除了她不会再有别人进来这个房间了，但是吴云卿还是伸手将保险柜锁好，把钥匙拔了下来。<br/>　　<br/>　　    做完这一切，吴云卿又简单地收拾了一下屋子的杂物，这才准备离开这间书房出去给自己弄点早饭吃。正当吴云卿走出书房，要关书房门的时候，她脑海里忽然闪过了一个念头，为什么她不看看许仕达保险柜里究竟锁了些什么呢？反正是看了许仕达也是不会知道的。<br/>　　<br/>　　    这些年来，书房这里一直都是许仕达的领地，保险柜更是锁的严严实实。每次许仕达回家，都会躲在书房里磨蹭很长一段时间的，她也不知道许仕达究竟躲在书房磨蹭什么。尤其是昨天晚，这家伙居然在书房待了一夜。<br/>　　<br/>　　    这几年来，随着许仕达官越做越大，夫妻俩的交流越来越少，她也不清楚许仕达心里想的是什么。现在一个了解许仕达想法的机会摆在了她的面前，为什么她不利用这个机会窥视一下丈夫的内心世界呢？<br/>　　<br/>　　    好害死猫，好心这种苗头一产生，吴云卿难以再把它摁下去了，她在书房的门口徘徊了几秒钟，最终毅然决然返回到保险柜前，用钥匙将保险柜门重新打开了。在打开门的一刹那，吴云卿心还是犹豫了一下子的，毕竟门里面锁着的，可能是许仕达最隐秘的内心世界。<br/>　　<br/>　　    门打开之后，首先映入吴云卿眼帘的，并不是什么黄金美钞宝石首饰之类的贵重物品，而是八大本放得整整齐齐日记本。这应该是许仕达写的日记。许仕达是有把日常生活的趣事写日记的习惯。、<br/>　　<br/>　　    最初吴云卿见到的许仕达的日记，还是他们做同事的时候，偶然的机会她看到了许仕达写的一篇日记，日记记载了许仕达那一天教学生活的点点滴滴，并总结了他所作所为的得失。吴云卿也是从这一篇日记，看出来许仕达是个可造之材，因为善于总结自己行为得失的人，一定是一个能进步的人。<br/>　　<br/>　　    不过那之后，她再也没见过许仕达日记都记了些什么了。丈夫将他的日记都锁了起来，不在让她看到了。吴云卿到也可以接受，毕竟一个人的日记，往往都记载着内心最隐秘想法，很多对人难以启齿的事情都是记载其，即使他们是夫妻了，这些也是不能共享的。<br/>　　<br/>　　    现在这些日记静静的摆在那里，这等于是把许仕达的内心世界都摆在了她的面前，吴云卿真是难以抗拒这些对她的诱惑，她真的很想知道丈夫这几年内心究竟在想什么，特别是为什么跟她越来越疏离。<br/>　　<br/>　　    吴云卿控制这心情的激动去拿起了放在最面的那本日记，翻开了封面，便看到扉页写了三个龙凤凤舞的大字：千人斩。这似乎是许仕达给这本日记起的名字。吴云卿是一个行事风格很板正的人民教师，并不明白这三个字真正的含义是什么，只是觉得这三个字有点凶神恶煞的意味。<br/>　　<br/>　　    她想不明白丈夫为什么要把记载内心世界的这本日记以这么凶的名字命名，只是猜测丈夫可能这些年的官场生活过得很是压抑，才起了这么凶的名字来发泄心的怨气。<br/>　　<br/>　　    如果是那些对日本爱情动作片很熟悉的人，看到这个名字肯定马会明白这三个字代表的是什么。（别装了，说的是你，你脸那会心的微笑已经泄露了你纯净的内心，你应该很清楚这三个字代表着什么，甚至你还看过与这三个字有关系的爱情动作片，或者日本漫画什么的。）<br/>　　<br/>　　    其实千人斩字面的意思是杀一千个人，但在日漫或者爱情动作片里，却是被引申为跟一千个异性发生过那种关系。在日本，有些对那种亲密关系存着一些幻想的人，会发愿说这一生要跟一千个异性发生亲密关系，甚至还有人为此建立起一些联盟协会之类的组织，定期交流自己在这方面最新的斩获。<br/>　　<br/>　　    我们的许副市长以千人斩命名这本日记，当然不是想说要杀一千个人，而是在发愿想要跟一千个异性发生亲密关系。不过此时我们板正的吴老师还不明白许副市长真正想要表达的意思。<br/>　　<br/>　　    吴云卿把扉页翻了过去，看到了吴云卿三个字，吴云卿当然知道这代表着她的名字，不用猜了，许仕达肯定在这一部分写下了对她的真实感受。对此她还有点终于窥视到许仕达内心的小激动，通过这本日记她可以清楚的知道，枕边人究竟是怎么看待她的了。<br/><br/><br/>
