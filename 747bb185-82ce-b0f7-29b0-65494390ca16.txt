　　“圣玄草！北海净水！好，这一次设下计谋，灭杀了这三位老祖，果然是有着无数好处！”
<br/>　　楚风眠在轰杀了这三位老祖之后，便是将那三位监察使，也一并轰杀，随后来到了虚空之中。
<br/>　　在虚空的一处宫殿之中，楚风眠坐在其中，打开了那三位老祖的空戒，他一眼望了过去，便是足以看见这三位老祖空戒之中的无数灵石。
<br/>　　这灵石，大多数都是王品灵石，堆积如山，如同一座座的山峰一般，这种财富，让许多武者看到，只怕都会疯狂，不由的想要夺取。
<br/>　　这财富，简直是比起一部分小宗门的全部财富，都要庞大了，只怕是一些皇者，都没有如此之多的财富。
<br/>　　如今这三位老祖，毕竟是七大宗门之人，极为崇高，所积攒下来的财富，也是要远远超越寻常的老祖。
<br/>　　不过这些财富，却是不被楚风眠放在眼中，现在还要是楚风眠愿意，他足以是凝聚出无穷无尽的灵石出来，灵石对楚风眠来说，只是一个数字罢了。
<br/>　　令他最为惊喜的一点，还是从这三位老祖的空戒之中，楚风眠居然是发现了一部分的圣药。
<br/>　　这些圣药，都隐藏在这空戒之中最为深处的角落之中，看起来就算是这三位老祖，也对这些圣药极为珍贵。
<br/>　　从这一批圣药之中，楚风眠发现了圣玄草，北海净水，这两种圣药，可都是圣药之中极为少见的存在，正是炼制那玉露破命丹的必要之物。
<br/>　　就算是从古家家主哪里，都没有买的到的圣药，如今在这三位老祖的空戒之中发现，自然是让楚风眠的心中，极为欣喜。
<br/>　　这也意味着，距离楚风眠真正收集齐关于这玉露破命丹的材料，又是更近了一步。
<br/>　　“这可真是走了大奇遇，这种级别的圣药，在外面几乎是重金难求，都被一些老祖把持在手中，不管是多少财富都买不到，看来没有这个机会，想要将雨露破命丹的材料收集齐，根本是天方夜谭。”
<br/>　　楚风眠不由的感叹道，他如今也是意识到了这，想要收集这雨露破命丹的材料，到底是一件多么困难的事情。
<br/>　　也怪不得那些老祖，甚至是皇者，想要收集齐，都需要数十年，甚至是数百年的时间才可以做到。
<br/>　　这还是在楚风眠财富无限的情况下，只要是有价之物，他都可以买来，他收集起来，都是如此困难。
<br/>　　那些老祖，皇者所拥有的财富，甚至都还远远无法跟楚风眠媲美，那些那些老祖圣者，为了收集一份圣品灵丹的材料，甚至都会倾家荡产。
<br/>　　相比之下，楚风眠收集的步伐已经是快得多了。
<br/>　　“天泉老祖的记忆之中，天泉商会中，的确是有着一批圣药的存在。”
<br/>　　楚风眠突然脑海一动，他轰杀了天泉老祖，却是将这天泉老祖的记忆夺取了过来，从天泉老祖的记忆之中，楚风眠便是发现，这天泉商会中，的确是有着一批圣药。
<br/>　　不过却是没有像是这天泉老祖所说的那样，楚风眠所需要的所有圣药，天泉商会之中都有。
<br/>　　毕竟那个时候，天泉老祖已经是被三位监察使所收买，准备是来对付，他所说的话，也只是为了让楚风眠跟他合作才故意去说的。
<br/>　　只是还的确是有着几种楚风眠所需要的圣药，是存在这天泉商会之中。
<br/>　　“天泉商会，已经没有存在的必要了！”
<br/>　　楚风眠目光闪烁，天泉老祖死了，这天泉商会，的确也没有存在的必要的。
<br/>　　既然是敢来对付楚风眠，那楚风眠也就要世人知道，敢来对付楚风眠的代价。
<br/>　　楚风眠的身形一动，猛然之间凝聚出了一道空间隧道，他的身躯一跃之间便是进入到了空间隧道之中。
<br/>　　天泉商会所在的区域，不在宁州古城，而是就在一处名为天泉城的城池之中。
<br/>　　整个天泉城，都是属于天泉商会的管辖范围之内，虽然不比宁州古城繁华，却也是一座小有名气的商业城池了。
<br/>　　随着如今天泉商会的越来越庞大，这天泉城的名声，也是越来越响。
<br/>　　半个时辰不到，天泉城之外，天空便是裂开，从上面一尊人影，缓缓的走了出来。
<br/>　　就在楚风眠的脚下，便是一座庞大无比的城池，城池之中，人声鼎沸，无数武者都在其中，做着生意，这里，便是天泉城。
<br/>　　楚风眠凌空而立，看向这天泉城，突然之间，他便是心神一动，一掌直接向着天泉城，抓了过去。
<br/>　　这一掌凌空打出，顿时化为了一尊巨大无比的战龙之爪，向着天泉城上空，猛然轰击过去。
<br/>　　这天泉城上，一座巨大无比的阵法，感受到这战龙之爪的气息，突然出现，这便是天泉城的护山大阵。
<br/>　　可是在下一刻，这护山大阵，却是在战龙之爪的面前，被硬生生的撕裂。
<br/>　　“轰隆！”
<br/>　　随着护山大阵的破碎，整个天泉城都是突然之间，出现了一阵震动，眨眼之间，一道身影，已经是降临在了这天泉城的上空。
<br/>　　“怎么回事？”
<br/>　　“发生了什么事？”
<br/>　　“有人来攻打天泉城？”
<br/>　　天泉城中，几位隐藏极深的老祖，都是突然之间感受到了天泉城的震荡，纷纷现身出来。
<br/>　　猛然之间，便是出现在了这天泉城之上，面对着楚风眠，隔空相望。
<br/>　　“阁下是谁！为何来攻击我天泉城的护山大阵！”
<br/>　　其中一尊老祖，看到楚风眠的身形，他的目光之中都是露出了几分震惊。
<br/>　　天泉城虽然不算数中域大城，但是这上面布置的护山大阵，也是寻常老祖都无法撕裂的。
<br/>　　本来在他们的预料之中，这出手之人，至少也要是一位极为强的老祖，至少是在中域之中声名显赫之人。
<br/>　　可没想到，居然只是一个看起来不足二十岁的少年，而且楚风眠的境界，却仅仅只有生死境的境界。
<br/>　　一名生死境的武者，突然撕裂了他们天泉城的大阵。
<br/>　　如若不是亲眼所看见，这老祖都不会相信，居然还会出现这样的事情。
