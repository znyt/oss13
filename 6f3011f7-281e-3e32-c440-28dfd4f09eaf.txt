        身子随着惯性向后倾，一只大手从她手臂旁绕过，用力托在了她后背蝴蝶骨的中央，稳住她的身子揽入怀中时，不小心来了个“胸咚”。
<br/>
<br/>        她的酥胸撞上他的胸膛时，童筱颖尴尬地瞪圆了那双黑白分明的美眸。
<br/>
<br/>        眼前端着一张俊容，浑身散发出高贵幽冷气息的男人，正是她的二婚老公──傅夜沉。
<br/>
<br/>        傅夜沉微微俯身，那双似笑非笑的黑眸，正用一股精锐的目光审视着她。
<br/>
<br/>        “大、大哥……”他的突然靠近让她莫名其妙地心慌了一下，童筱颖缩了缩肩头，小手抵住他的胸膛，防止他贴得太紧，“我、我只是路过！”
<br/>
<br/>        饥不择食、寒不择衣，她现在是慌不择言！
<br/>
<br/>        “嗯？路过？！”傅夜沉剑眉微挑。
<br/>
<br/>        她今天穿了他给她挑的淡粉色连衣裙，未施脂粉的白嫩脸蛋上挂着一双心不在焉的漂亮水眸，缕缕飘逸的长发如他想看的那样轻泻及腰。
<br/>
<br/>        这些看在他傅夜沉的眼底，成了悄无声息的诱惑。
<br/>
<br/>        看得出来，她是在投他所好。
<br/>
<br/>        嗯？莫非，一日不见，她是打算一边讨好他，一边又带着糖衣炮弹来跟他谈离婚？
<br/>
<br/>        傅夜沉的黑眸里露出了一抹狐疑的目光。
<br/>
<br/>        童筱颖愧疚地将目光挪向一旁，略微丰腴的唇瓣微启，声音软糯犹如出水芙蓉般清甜地关问道：“大哥，你……身体好些了吗？”
<br/>
<br/>        “不大好，医生说我有心脏病，活不了多久了。”傅夜沉不痛不痒地撒谎，其实他只不过是这三年来对她相思成疾落下的心痛病罢了。
<br/>
<br/>        心脏病？！不会吧！
<br/>
<br/>        童筱颖顿时回眸，眨巴着漂亮的大眼睛，难以置信地看着傅夜沉。
<br/>
<br/>        “你与其想着跟我离婚，还不如盼着我早死好来得更有效。”他接着冷冷地说道。
<br/>
<br/>        童筱颖差点被他这句话给笑到岔气。
<br/>
<br/>        “我不是来跟你谈离婚的，我只是来给你送衣服和晚餐的。大哥，你没必要这样诅咒自己啊！你会长命百岁的！”她一脸淡定地说。
<br/>
<br/>        “你刚还说你只是路过。”傅夜沉阴沉着一张俊脸。
<br/>
<br/>        童筱颖咧嘴干笑：“我只是路过这里，准备去你的病房来着。”
<br/>
<br/>        他能不能不要这么跟她较真啊！
<br/>
<br/>        “衣服给我。”他随之放开了她，向她伸出手来。
<br/>
<br/>        童筱颖连忙将臂弯上的衣服纸袋挂在了他的掌心上，顿了一下，询问：“这晚餐……”
<br/>
<br/>        “我已经吃过饭了，没其他事的话，你可以走了。”傅夜沉不假思索地说完这句话后，心里就后悔了。
<br/>
<br/>        其实，他不想她走……
<br/>
<br/>        怎么就对她说狠话了？
<br/>
<br/>        童筱颖愣了一下，完全没料到他会毫不客气地对自己下逐客令。
<br/>
<br/>        更何况，她应该不是“客人”吧！
<br/>
<br/>        “那我能在你病房里自己先吃了饭再走吗？我还没吃晚饭。”她微微低垂着脑袋小声嘟囔。
<br/>
<br/>        听到她的话，傅夜沉皱了皱眉，眸色微微一黯，脸色不免沉下几分，口是心非地傲娇：“随你便。”
<br/>
<br/>        “打扰了……”童筱颖讷讷地垂下眼帘，心想着他……他该不会还在生她的气吧？
<br/>
<br/>        她跟着他去了他的VIP病房，一个年轻的女护士随后走进来，直接跟傅夜沉发起了牢骚。
<br/>
<br/>        女护士说：“冷先生，请您不要再私自拔针头了。”
<br/>
<br/>        “很抱歉。”傅夜沉淡淡地道歉。
<br/>
<br/>        女护士重新给傅夜沉上好了点滴后，无意间看到一旁沙发上坐着的童筱颖，身心一愣。
<br/>
<br/>        童筱颖不知道这女护士心里是作何感想，这个时候，傅夜沉坐在病床上，云淡风轻地介绍：“她是我太太。”
<br/>
<br/>        “……”女护士尴尬地笑了笑，向童筱颖礼貌性地点了点头，便端着打针盘离开了病房。
<br/>
<br/>        童筱颖算是长见识了。
<br/>
<br/>        这长得帅气又多金的男人，到哪儿都能令单身又年轻的女人对之动歪念头。
<br/>
<br/>        总裁与女明星、总裁与女秘书、总裁与女护士……
<br/>
<br/>        满满的激情梗。
<br/>
<br/>        算了，她还是淡定自若地把自己做的晚饭全吃了吧！
<br/>
<br/>        民以食为天，她不能让自己饿肚子。
<br/>
<br/>        童筱颖打开了饭盒，自顾自地吃起来。
<br/>
<br/>        本来，她还想着陪他一起吃晚饭的，看样子，他确实已经跟那女秘书落以柔一起吃过了。
<br/>
<br/>        想必，那个女秘书落以柔做的饭菜一定比她做的要好吃。
<br/>
<br/>        两人明明在同一个屋檐下，却各有所思。
<br/>
<br/>        童筱颖超级淡定地把饭吃完，殊不知，她在吃饭的这过程中，傅夜沉的目光已经无数次总在有意无意地落向她身上。
<br/>
<br/>        她倒是一直在专心致志地吃饭，完全没有去在意傅夜沉那边的动静。
