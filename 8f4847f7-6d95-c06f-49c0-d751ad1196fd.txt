　　悟道古茶树下，元始静静思悟，他的周身，大道道则翻腾，神通道则飞舞，将一种种神通演化到极致。
<br/>　　“开天印！”
<br/>　　元始不悲不喜，徐徐一道印法打出，顿时出现鸿蒙开辟，玄黄初分，阳清升为天，阴浊降为地，日月星辰，江河湖海，大山森林、天地万物，尽皆演化。
<br/>　　在他的面前，出现一幕开天辟地的壮阔景象。
<br/>　　以他如今的法力实力，开天印的威能已然大得不可思议，不过开天印并没有将他全部的大道道则囊括，开天印强分混沌，开天劈地，其中蕴藏着后天的种种大道，玄黄阴阳、四象五行、七星八卦、九宫十方、天干地支、天罡地煞，亿万种大道，但其中并没有元始大道。
<br/>　　“灭世印！”
<br/>　　元始不停，又一印打出，他身前那刚刚开辟的广阔世界陡然间崩溃坍塌，之后变成一个巨大的灭世磨盘，将天地悉数笼罩。
<br/>　　磨盘搅动，将天地吞没，一切尽皆绞碎，磨尽天地，磨碎一切。顷刻之间，天地重归混沌，万道破碎，即便是天道，也被绞碎，复归鸿蒙！
<br/>　　“诛神印！”
<br/>　　元始身前，混沌涌动，各种大道道则汇聚，不断凝聚成一道巨大的紫金长枪，枪杆闪闪放光，枪尖杀气冰冷，雷光电鸣，气势恢宏，一股无法冒犯的威严，自在其中。
<br/>　　“五行印！”
<br/>　　“四象印！”
<br/>　　“不周印！”
<br/>　　“元心印！”
<br/>　　元始不停，又连续打出四印，顿时，长枪破灭，分化出各种流光，道光闪烁，神光飞舞，仙光缥缈，一道化为赤、青、黄、黑、白五色的五行大印，大印之上，金木水火土五行流转，相生相克，生生不息。
<br/>　　一道化为朱雀、白虎、青龙、玄武四象神兽，朱雀喷火，白虎跃杀、青龙探爪、玄武擎天，地水火风涌动，异象纷繁，道则不断浮现。
<br/>　　一道化为一座巨大的不周神山，上顶苍天，下踏大地，威压宇宙众生，神威莫测。
<br/>　　最后一道化为一片虚无缥缈的彩色迷烟，迷烟幻化为各种形状，形状不定，变幻莫测，最是让人不可捉摸。
<br/>　　“阴阳印！”
<br/>　　又是一道印法打出，各种异象晃动，全都化为黑白两色，白色在上，宛如一道巨大天幕，黑色在下，仿佛一片大地。白色下降，黑色上升，黑白两色，相互交织，相互缠绕。形成一道巨大无比的阴阳太极图，太极旋转，阴阳相生，玄光迸发，像是在演化天地大道，玄之又玄，妙不可言。
<br/>　　“道一印！”
<br/>　　黑白相融，阴阳相合，化而为一，万般大道，尽在此刻万化三，三化二，二化一。诸般大道，尽皆归一，万道相融，万法相合。天地寂静，时空停滞，形成一个圆点。
<br/>　　一切的一切，不管是时间，还是空间，因果，造化，都汇聚道这个点上。
<br/>　　这一点，就是一切的起始，也是一切的终结。
<br/>　　“无极印！”
<br/>　　元始依旧不停，又是一印打出。他仿佛越过了那一点，跨过无尽时空，来到了一片虚无之中。在这里，没有天，没有地，没有时间，没有空间，甚至没有了大道的存在。
<br/>　　万物寂静！
<br/>　　突然一声轻微的心跳声传出。
<br/>　　这一声心跳就仿佛宇宙尚未开辟之时，一切死寂，无光无色、无声无神、无生无死，万籁寂静，没有任何生灵之时，传来的第一道声音。这是先天之道音，是大寂灭大荒芜中的第一道生机。
<br/>　　咚——
<br/>　　元始的气息渐渐增长，越来越强，越来越大，他如同是大道未出，混沌未现之时的鸿蒙中诞生的第一尊魔神，无比古老的魔神。
<br/>　　他的身体在复苏，在轰鸣，在他的手上，出现了一柄神斧！
<br/>　　咔擦——
<br/>　　他一斧劈出，开天辟地，切开鸿蒙，紫气分裂，化为玄黄二气，玄黄分开，衍化天地万物。
<br/>　　轰——
<br/>　　天地蓦然剧烈震动，化为巨大磨盘，将天地尽皆笼罩，磨盘搅动，将一切磨碎，天地复归混沌鸿蒙！
<br/>　　五行相生，四象镇世，阴阳流转，不周神威，元心莫测，各种异象，全在此刻尽皆归一，化为一个最初的圆点。
<br/>　　圆点动，混沌无极生，大道源头出现，本源出，大道河流开始复苏，万道交织，奔流不息。
<br/>　　九道印法，此刻在他手中一气呵成。
<br/>　　元始盘坐于悟道古茶树下，不悲不喜，双手只捏着一道印法，他的身躯和道则与他，交织在一起，共同组成一个玄之又玄，妙之又妙的印法。
<br/>　　这印法之中蕴藏着一种微妙的律动，毫无气息，没有惊天动地的威能，但它却蕴藏着惊天动地的威能，拥有包含宇宙的玄机！
<br/>　　它就仿佛是一切之元，一切之始。
<br/>　　这印法曰：元
<br/>　　曰：始。
<br/>　　终极印法，元始印！
<br/>　　九印聚，元始现！
<br/>　　这一刻，万道合流，同归元始，在这一印之下，所有大道不分仙道，不分神道，不分魔道，不分佛道，亦不分天道，不分先天、后天种种大道，通通复归元始。
<br/>　　在他的体内，所有大道不再分彼此，再也没有什么天道道则，没有大道道则，没有先天道则，没有后天道则，没有混沌大道，没有鸿蒙大道，没有太初大道。
<br/>　　有的，只是一条大道。
<br/>　　它叫，元始大道！
<br/>　　元始缓缓睁开双眼，眼中不见任何异象，有的只是平淡如水，清净自然。他的声音响彻元始本源世界：
<br/>　　“从今天往后，诸天万界，吾名元始！”
<br/>　　话音刚落，元始身上的气息便在不断上涨，仿佛身上以前锁住他的枷锁被打开，在这一刻，没有什么能够束缚住他。
<br/>　　神威如狱，神恩如海！
<br/>　　混元大罗金仙一重天！
<br/>　　混元大罗金仙二重天！
<br/>　　混元大罗金仙三重天！
<br/>　　轰隆隆！
<br/>　　最后，元始身上的气息不断上涨，直至涨到混元大罗金仙三重天圆满，才缓缓停了下来。只差一步，便可突破混元中期境界。
<br/>　　同时，他体内的三百六十五个神窍世界在快速扩张。
<br/>　　七万里。
<br/>　　七八万里。
<br/>　　……
<br/>　　十二万九千六百里。
<br/>　　轰隆隆！
<br/>　　世界一直扩张到小千世界的极限，再进一步，便是中千世界了。三百六十五个极限小千世界之中，星辰在凝聚，高山在拔起，江海在扩张，灵气变得更加充裕了，世界规则变得更加强大了。
<br/>　　同时，他身体之中的每一个细胞，都由之前的空间洞天扩张到了如今的小世界。也就是说，他的身体，如今就是一个个世界组成的。
<br/>　　强大！
<br/>　　真正的强大！
<br/>　　如今的他，完全可以吊打十个之前的自己。
<br/>　　之后，他身上的气息全部消失，内敛于身。在他的身上，再也看不出丝毫的气机，他此时就仿佛是一个最为平凡的凡人。
<br/>　　没有天花乱坠，没有地涌金莲，没有紫气东来三万里，没有圣威凌天下，没有星辰耀青天，没有道音袅袅、仙音悦耳，也没有神光纵横，仙光缥缈。没有成圣时的各种异象。
<br/>　　有的，只是最初的平凡。
<br/>　　元始大道！
<br/>　　历经种种磨难，感悟万般大道，自今日起，他的元始大道终于踏出最为关键的一步，一步踏天，成就混元。
<br/>　　从此，万般大道，尽在吾手。
