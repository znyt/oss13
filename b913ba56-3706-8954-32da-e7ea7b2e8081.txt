　　陈三是个不入流的道士，叶少阳和毛小方聊的这些事情，他完全没有听说过，之前看他们聊在兴头上，不敢打扰，这会儿实在忍不住，先劝他们吃菜，抓住机会问道：“二位说的无极天师，是何等样人？”
<br/>　　毛小方露出十分仰慕的神情，说道：“无极天师，号称人间道神，如果光论人间法术界，没有一个法力在他之上，不敢说前无古人，绝对是后无来者了。”
<br/>　　叶少阳听了这话，突然想到了师父青云子，还有道风。其实比硬实力的话，道风不在无极天师之下，就算是师父青云子，临终那一战，也是与无极天师同归于尽……但是这事没法说，只是笑了笑，没有反驳。
<br/>　　说完了法术界之殇，叶少阳借口自己之前都在山上修行，还是头一回下山，跟毛小方打听起当今法术界的一些事情，说起来，毛小方是他在这个世界遇到的真正入流的道士，知道的也比之前那些人要多，他起初说的跟张道长之前说的差不多，一些人名和门派。
<br/>　　不过说到深处，他告诉叶少阳一个惊人的消息：如今的法术界，真正最高层的并不是道佛二宗，而是法术公会。
<br/>　　法术公会的成员，叶少阳听青云子简单说过，是轩辕氏后人，老家不在人间，而是在不周山。至于不周山在什么地方，叶少阳不知道，本来想问毛小方，估计他也不知道，而且这些事情，跟自己一毛钱关系都没有，想想还是不问了。
<br/>　　毛小方所说的，跟青云子曾经说过的差不多一样：在这个乱世，南北不通，很多宗派之间无法无相联系，法术公会前来人间坐镇，是要凝聚整个法术界的力量，保护人间这方面的太平，以免妖孽趁乱世作祟。
<br/>　　叶少阳恍然想起，青云子在讲都有关法术公会的时候，也讲过这一段，不过作为当代人，毛小方知道的更多一点，在言谈之中，叶少阳听出他对法术公会似乎有点不满，忍不住询问。
<br/>　　“我们人间，自有人间的规矩，法术公会强行插手人间法术界，指手画脚，号令天下，说起来实在有点让人反感。”
<br/>　　叶少阳沉吟起来，心里想着也是，人间法术界，本来就该由人类自己来管，法术公会的人虽然长的像人，但毕竟不在这个世界，严格来说算不上是人，凭什么插手人间的事情？
<br/>　　在自己那个时代，法术公会基本上就是个虚拟的组织，掌管者也是几大宗门的掌教，共同执掌，如果自己那个时代也有来自别的世界活空间的人来对人家法术界指手画脚，自己第一个不答应。不过，这毕竟不是自己的世界，叶少阳只是好奇，明明两个世界是一脉相承的顺延关系，那么，法术公会是什么时候基本离开这个世界的，为什么到自己那个时代，已经看不到什么法术公会留下的痕迹？
<br/>　　叶少阳虽然好奇，但也没有想要一探究竟的念头，甚至相反：他不想在这个世界留下太多的痕迹，他有点担心自己会扰乱历史的走向，会造成什么样的结果，他也不知道。
<br/>　　“对了叶天师，你这次下山，有什么打算？”聊完了这些大事件之后，毛小方端着酒杯问道。
<br/>　　“我……我要去一趟龙虎山，找道渊真人有点私事。”
<br/>　　“哦。”毛小方显然不知道道渊真人是谁，表情略有点失望，被叶少阳捕捉到，问道：“道兄有什么就直说。”
<br/>　　毛小方抿了一口酒，说道：“是这样的，我在追赶我那师弟的路上，经过一个山坳里，当时是晚上，那个山坳里竟然出现了百鬼夜行的异象，我差一点就着了道，我等到天亮，尝试一探究竟，结果在山谷里并没有发现什么异常，我因为要追逐那个师弟，也没有太多时间停留，但是我心里一直惦记这件事，现在任务完成了，我想要回去一探究竟，但是叶天师你也知道的，百鬼夜行，必有异象，我生怕自己一个人对付不了，所以想找个人陪着我一起。”
<br/>　　叶少阳一听就明白了，皱眉说道：“最近不是清明，也不是鬼节，怎么会有百鬼夜行？除非那是一片极阴之地，或者那一代有什么封印邪物的东西，百鬼夜行，乃是怨气郁结，但也不会有百鬼那么多。”
<br/>　　毛小方叹道：“何止百鬼，当时整个山坳里都是鬼象，因此我才差异万分，觉得自己一个人未必能处理得了。但我辈法师，遇到这种邪事，自然不能不管，山高路远，现在回山门找人也来不及了，我听了叶天师除人形煞的事情，得知也是浩然正气的法师，本想跟叶天师一起去走这一趟，既然叶天师有事，那就不强求了。”
<br/>　　叶少阳听他说完，内心犹豫起来，从理智上来说，自己真的不想管这档子事，但毛小方说的没错，自己骨子里的性格……遇到这种事，还真不能不管，可是鹰潭龙虎山就在前面不远了，自己再折回头跟毛小方走，又得耽误不少时间……
<br/>　　想了半天，叶少阳灵机一动，提议道：“毛道长，你看这样行吗，前面就是龙虎山不远了，不然你跟我一起去龙虎山，找到道渊之后，我跟他说一件事情，如果没什么意外，我再陪你去处理这宗公案，你看怎么样？”
<br/>　　毛小方一听，眼前一亮，说道：“这个好，正好我也没去过龙虎山，一直想去探访一番，借这个机会，你我结伴同行！”
<br/>　　做了决定之后，大家都比较轻松，于是继续吃火锅，喝酒。
<br/>　　叶少阳不想在这里逗留，跟毛小方商量之后，决定当天就动身，于是毛小方回自己住的客栈打点行李，约好了待会来找他。
<br/>　　叶少阳三人回到自己住的那个名字土得掉渣的客栈，叶少阳仔细想了想，跟翠云提出，让她留在这里等着自己，反正自己从龙虎山回来，还是要经过这里，到时候在来这里接她比较好。
