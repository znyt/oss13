　　第119章
<br/>　　一个好人：正义的少年喔，在乱世中保持自己的善良是如此珍贵，请铭记，善良的人运气不会太差，你的正义会吸引更多志同道合的少年。
<br/>　　这是程大雷的隐藏属性，他恍然意识到，自己的隐藏属性好像不止和魅力值有关，似乎在某种情况下，还可以提升自己的运气值。
<br/>　　比如现在。
<br/>　　不然难以解释，为何孙德龙如此死心塌地的帮自己。
<br/>　　孙德龙果然没有夸口，在第二天日落前，两大船的粮食已备好，随时可以离岸。
<br/>　　孙德龙行到程大雷面前，微弯着身子：“六公子，幸不辱命，您要的粮食已经备齐，公子可否多留几日，让老奴尽一份心。”
<br/>　　“不啦，家里的孩子老婆都快饿死了，实在耽搁不起。”
<br/>　　“老婆孩子……”孙德龙皱起眉头：“六公子已经娶亲？”
<br/>　　“啊……哈哈。”
<br/>　　孙德龙看了柳芷和苏樱一眼，点点头道：“明白，明白。本来我还准备了十位歌姬，想着一路寂寞，可以让他们陪侍公子。”
<br/>　　程大雷眼前一亮，恨不得抱住孙德龙，给他来个男人间的拥抱。
<br/>　　“但既然公子随身已有侍女，老奴便不多此一举。公子身份尊崇，怎样的女人得不到，莫要被美色迷眼，还是要多读诗书，修身勤政。”
<br/>　　“……”程大雷。
<br/>　　众人一一上船，升起船帆就要起航。
<br/>　　“欧公子，欧公子……”
<br/>　　此刻，码头忽然闯进来一帮人。连孙德龙都有些奇怪，扭过头去，只见都是文士儒衫打扮。
<br/>　　“欧公子留步，留步……”
<br/>　　这些人以唐如才为首，一窝蜂的冲到船头。见程大雷已经上船，纷纷将手中的柳枝丢到船上。
<br/>　　南方的冬季，柳芷也只是冒着新绿，程大雷立时面临一阵柳枝雨，他顿时感觉一阵气闷：搞什么，这是给我送终么。
<br/>　　程大雷却是有所不知，在此时柳枝是送别的信物，柳有『留』的意思，柳树随插随活，又有祝愿在远方一切顺利的意思在。
<br/>　　程大雷那半篇词已由唐如才之口，在仕子之间传开，自然造成不小的反响。这帮人还想寻机会去拜访程大雷，谁知道程大雷今日已经要离开。于是纷纷追到码头相送。
<br/>　　这些人还有带着歌姬的，此时也招着手中绣帕。
<br/>　　“欧公子，欧公子……”
<br/>　　“欧巴，欧巴，留下吧，留下吧。”
<br/>　　“哪儿干不是干呢，欧巴留下吧。”
<br/>　　程大雷过了一会儿才明白这帮人的意思。
<br/>　　此时只见唐如才挥舞着双手道：“欧公子，此情此景，可否赋诗一首，我辈留不下公子的人，留下公子一首诗也是好的。”
<br/>　　赋诗一首，咦，又到装X的时候了么。
<br/>　　程大雷负手立在船头，做出高人模样，此刻正搜肠刮肚，琢磨抄那首。
<br/>　　抄也不好抄，要应时应景，还得没有典故。
<br/>　　『桃花潭水深千尺……』这个不行，『孤帆远影碧空尽……』这是送别的诗，『莫愁前路无知己』这也是送别的诗，而且前面是什么自己记不得了。
<br/>　　这个平日没有储备还真不好办。
<br/>　　看着岸上众人殷殷的目光，程大雷怎忍心辜负，关键是他怎么能辜负这个转X的机会。但此刻船已立岸，再不抄就来不及了。
<br/>　　“文章薄伎，固弃於高贤；刀笔小能，不容於先达。”
<br/>　　程大雷令柳芷取出酒来，对着岸边众人饮尽。
<br/>　　“都在酒里。”
<br/>　　岸上人头噪噪，有人冲程大雷挥手，口中喊着什么，离得太远，却已听不太真切。
<br/>　　“……留下吧，留下吧……”
<br/>　　扬州是个好地方，既有舞文弄墨的书生才子，也有浅吟低唱的青楼女子，若是有时间，程大雷还真想在这繁华地多流连几日。但这次委实没有时间，下次吧。这次委实没有时间了。
<br/>　　虽然粮食都已装上船，但因为是逆风而行，船只能走之字形路线，回去的路要比来时的路更加困难。况大海瞬息万变，一路上回遇到什么意外，谁也没法预料。在没有回到蛤蟆寨之前，程大雷这颗悬着的心就没办法放下去。
<br/>　　黄昏离岸，行不到半日，天已经彻底黑下来。今夜没有月亮，大海之上伸手不见五指。
<br/>　　在平静的海面上，停着几艘小船，没有点灯，随着海波的欺负而轻轻摇曳。
<br/>　　一男人趴在甲板上：“到了么？”
<br/>　　“大当家从扬州传来的消息，看时辰约莫快到了，已经让嫂子先去了。”
<br/>　　“等他们进了包围圈就动手，里应外合，人杀了，船留下。”
<br/>　　“二当家放心，这事情咱们不是做过一次两次，船上的消息都打探清楚了，五十多号人，领船的是个公子哥，估计见到刀就怂了。”说着，这人发出两声寒笑。
<br/>　　男人也笑了：“咱骷髅岛做事，却也没出过岔子，好了，做事吧。”
<br/>　　……
<br/>　　船行水上，除了哗哗的水响，没有任何声息。
<br/>　　“大当家，水里有人？”徐神机过来道。
<br/>　　“什么人？”
<br/>　　“说是遭了海贼落难，里面有两个女人，想上咱们的船。”
<br/>　　“走，过去看看。”
<br/>　　程大雷来到船头，看到水里一块船板搭着五个人，三男两女，脸都快被水泡白了。
<br/>　　“恩人，求求求你救救我们……”其中一个鹅蛋脸的女人撩起额前长发，看模样竟也有几分姿色。
<br/>　　“你们是怎么会事，怎么在海上？”徐神机问道。
<br/>　　“我们随老爷回乡，路上遇到海贼，老爷夫人都被海贼杀了，只有我们几个逃出来。求求您开恩，救救我们。”女人哭哭啼啼道。
<br/>　　柳芷听得可怜，道：“大当家，咱们救他们上来吧，太可怜了。”
<br/>　　程大雷仔细打量着女人一眼，女人生就一双桃花眼，不笑有三分媚意。
<br/>　　“救当然是要救的，你们也不容易。”
<br/>　　程大雷把他们五人都搭上船，那女人抹着眼泪：“恩人，太谢谢你了，奴家来生做牛做马也要报答恩人。”
<br/>　　“什么！”
<br/>　　程大雷忽然脸色一变，挥手拔剑，一剑捅进女人怀里。
<br/>　　女人睁大眼睛，眼底闪过一丝惊愕。
<br/>　　“为什么要当牛做马，为什么不是以身相许，呀，杀呀！”
<br/>　　
<br/>(本章完)
