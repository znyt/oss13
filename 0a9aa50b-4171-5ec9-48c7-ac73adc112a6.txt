　　第318章
<br/>　　你好……
<br/>　　看着眼前浮现的两个字，程大雷心情久久不能平静，这两个字在他眼中看来，如同天启。
<br/>　　是谁在回应自己？系统精灵，游戏GM，或者是……客服？
<br/>　　程大雷深吸一口气，稳住砰砰跳动的心脏，又说出两个字。
<br/>　　“你是？”
<br/>　　『……』
<br/>　　回应程大雷的是一串省略号，对方没有给出回答。
<br/>　　“我需要你的帮助。”
<br/>　　『什么？』
<br/>　　“请让林少羽和林冲恢复健康？”
<br/>　　『权限不足，无法完成』
<br/>　　程大雷怔了怔，重新输入命令。
<br/>　　“有没有能治好二人的药？”
<br/>　　『搜索中……搜索完成，查到170种符合条件的药品』
<br/>　　程大雷沉默下来。
<br/>　　片刻，程大雷眼前又浮现一行字。
<br/>　　『想要？』
<br/>　　程大雷点点头。
<br/>　　『那么交换吧』
<br/>　　这句话让程大雷感觉自己是在和魔鬼交易，他屏住呼吸，问：“交换什么？”
<br/>　　『给予你想得到的，拿走你不想失去的』
<br/>　　“什么？”
<br/>　　『交换么？』
<br/>　　又是古板而机械的询问，可程大雷隐隐感觉，其背后当是一个智慧生物。不能答应，不能答应，答应后就会失去很重要的东西。
<br/>　　『交换么？』
<br/>　　不停重复的询问，仿佛程大雷不回应，询问就不会停止。
<br/>　　“交换。”程大雷咬紧了牙关。
<br/>　　嘟，获得顶级疗伤药血红丸。
<br/>　　嘟，失去隐藏属性：一个好人
<br/>　　程大雷手中多了两颗药丸，他查看自己的信息，发现隐藏属性已经消失。
<br/>　　姓名：程大雷（名扬四方的顶级山贼）
<br/>　　年龄：20
<br/>　　技能：雄辩，大魔王三板斧，阿飞快剑，身无彩凤双飞翼，心有灵犀一点通。
<br/>　　隐藏属性：无
<br/>　　失去一项隐藏属性，却能救活两个人。值得么？程大雷觉得还是蛮值得的。日后未必没有机会重新获得隐藏属性，可林少羽要是死了，命可就救不回了。
<br/>　　程大雷急忙拿着药去给林少羽服下，即便服用系统奖励的药，二人也没有立刻好转。程大雷一颗心一直牵挂着伤势，便是与苏樱久别重逢，也没有与她多说话。
<br/>　　一直这样三天三夜，二人才悠悠醒转，呼吸平稳下来，真正将伤养好还需要很长一段时间，但性命显然是保住了。
<br/>　　程大雷一口气松下来，自己也累倒了。他在床上睡了一天，醒来后，见天色昏暗，已是黄昏时分。
<br/>　　朦胧中感觉有湿润的毛巾从自己脸上拂过，程大雷拿住了对方的手腕，见苏樱坐在自己身边。
<br/>　　苏樱扯了扯，没有扯动，半是气半是怨的哼了一声，目光才稍稍变得柔和。
<br/>　　“你既然醒了，就去看看吧，林少侠已经醒了，刚喝了一些细粥。”
<br/>　　程大雷确实也想看看林少羽，其二也是不知该如何面对苏樱，正好趁这个机会躲一躲。
<br/>　　屋子里，一左一右各摆着一张木床，屋子里的空气有些昏暗。林冲和林少羽见程大雷进来，忙挣扎着坐起，身子靠在墙上。
<br/>　　程大雷看到秦蛮和徐神机也在，据说徐灵儿得到消息，也正从落玉寨赶回来。
<br/>　　虽然都是一个山寨的兄弟，但最初的蛤蟆寨只有五人，关系自然和其他人不同。
<br/>　　程大雷摆摆手，见二人虽然现在还不能下床，但脸上已经有了血色。
<br/>　　于是程大雷也放下心来，没好气的看了林少羽一眼，呵斥道：“嚯，长本事啦，不是告诉我，要归隐林泉么？”
<br/>　　“大当家……”林少羽面带愧色。
<br/>　　“好啦，好啦。”终究是自家兄弟，程大雷也谈不上有多兄弟：“能活着就是好事，以后都不要折腾了，就留在山寨，大家仍旧做兄弟。”
<br/>　　“我们这次来，就是投奔大当家的。”林少羽。
<br/>　　程大雷心中还是有几分欣慰的，林少羽出了事后要来找自己，证明自己这大当家在他心里还是有些位置的。
<br/>　　据林少羽所说，正义教起事兵败后，他和林冲从重围中突出来。之后，就是要来西北投奔程大雷，只是路上恰好碰到秦蛮。
<br/>　　“你早就该来找我！”程大雷冷哼一声：“我这里家大业大，怎么不能管你一碗饭吃，咱们这里不服王法管，逍遥自在，你偏偏要去入什么正义教。”
<br/>　　林少羽叹息一声：“这次的确失算了，本不该这么早起事，若再有一两年积蓄实力，绝不会像这次一样败得这么惨。”
<br/>　　程大雷愣了愣，听着感觉有些不对了，于是问：“你们是怎么加入的正义教？”
<br/>　　林少羽和林冲都保持着沉默。
<br/>　　程大雷再次追问：“你在正义教是什么身份？”
<br/>　　林少羽欲言又止，没有回答。
<br/>　　“莫非是一地的首领？”
<br/>　　林少羽摇摇头。
<br/>　　“难道还是一地的信长？”程大雷有些惊讶，一城之信长，这身份可是不低，林少羽可以呀。
<br/>　　林少羽又摇了摇头。
<br/>　　“怎么……该不会还是中枢人物吧？”程大雷难以置信，不过，如果林少羽这样的人都能够出谋划策，也不怪正义教拜得这么惨。
<br/>　　林少羽再次摇头。
<br/>　　“那我就放心了。”程大雷松了口气：“那你直接说吧，你们在正义教管多少事，是个什么身份，神主，神将，信长，首领……我对正义教也不是一点不了解。”
<br/>　　林少羽和林冲对视一眼，两人目光闪烁，林冲道：“少羽，当着大当家的面，你还有什么不能说的，我们这次为何来，你难道忘了？”
<br/>　　林少羽深吸一口气，终于舍得开口：“不瞒大当家，我既不是信长，也不是首领，这正义教……本就是我一手创立。”
<br/>　　程大雷怔了怔，随即哈哈大笑：“快别扯淡了，你那脑袋我还不了解。”
<br/>　　林少羽目光灼灼的盯着他，没有摇头也没有点头。
<br/>　　程大雷一激灵，难以置信的问：“你在正义教是……”
<br/>　　林少羽缓缓吐出两个字：“神将。”
<br/>　　啊！
<br/>　　程大雷发出一声惊呼，感觉天地都倒悬了过来，神将是正义教首脑，地位仅次于那闻名不露面的神主。
<br/>　　猛地，程大雷有种不好的预感，他开口，声音已经颤抖起来。
<br/>　　“那么，正义教的神主是？”
<br/>　　林少羽和林冲的视线都看着他，几乎是异口同声道：
<br/>　　“就是大当家您呐！”
