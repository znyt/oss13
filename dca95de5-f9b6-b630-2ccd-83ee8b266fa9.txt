　　西市，繁华依旧，熙熙攘攘的人群摩肩接踵，路两侧的摊贩高声叫卖。
<br/>　　李承乾换了一身普通的衣服肩膀上扛着一直刚刚从樱桃树上折下来的树枝，树枝上挂满了红通通的樱桃。
<br/>　　冬季大棚的大面积应用使得一些本不应该在秋季的水果违反常理的出现在长安街头。
<br/>　　程琳跟在李承乾的身边，时不时从树枝上摘下一颗樱桃放进嘴里，慢慢的品尝着。
<br/>　　这是数年来难得的一次逛街，苏玫和孔雯、程琳几女份外珍惜，拉着李承乾东游西逛，显得十分活泼，就像回到了少女时代一样。
<br/>　　而李承乾也十分享受这样的生活，扛着樱桃树枝竟也不嫌累，跟在他身边的苏猛要了几次都没要下来，最后只能任由他自己扛。
<br/>　　不过在街上逛了一会儿之后，李承乾敏锐的发现身后多了一个小尾巴，回头一看却是一个年龄在四、五岁左右的小男孩儿，正仰着头垂涎欲滴的看着他肩上的樱桃枝。
<br/>　　这是谁家的娃娃，莫不是走丢了？这是李承乾第一想到的问题。
<br/>　　但很快他便从这小家伙身上的配饰上看出了一些端倪，于是嘴角微微一挑，心中坏水便冒了出来。
<br/>　　……
<br/>　　“唔，真香，真好吃。”
<br/>　　“唔，太好吃了，吧唧吧唧……”
<br/>　　无良的太子殿下时不时从樱桃枝上摘下一颗放进嘴里，然后便一边吧唧嘴，一边咕哝。
<br/>　　这下，跟在后面的小家伙再馋了，口水自都流到了外面。
<br/>　　而程琳等人此时也发现了李承乾的异常，同时也看到了身后的小男孩儿。
<br/>　　“咦？这是谁家的孩子？”孔雯挺着大肚子，看着那孩子的眼神带着担忧。
<br/>　　“翼国公家的呗。”李承乾一边随口回答着，一边又摘了一颗樱桃放进嘴里：“唔，好吃！”
<br/>　　“噗嗤……”程琳在一边看着李承乾孩子气的样子，再看看小男孩那张跟着李承乾嘴巴蠕动而不断开合的小嘴，终于忍不住笑出声来。
<br/>　　“殿下……”苏玫在一边明显也看不下去了，轻轻戳了李承乾一下。
<br/>　　作为一个孩子的母亲，苏玫看到那小男孩便想到了自己的孩子。
<br/>　　李承乾此时也觉得差不多了，再逗下去那小家伙非得哭了不可，于是蹲下身子对那小男孩问道：“你叫秦怀玉，对不对？”
<br/>　　“嗯。”小男孩点点头，眼睛依旧没有离开樱桃枝。
<br/>　　按说秦家的孩子绝对不会没有吃过樱桃，可是奈何李承乾的吃法太过变态，丫竟然是扛着树枝边走边吃，这样的行为对一个小孩子来说简直就是天大的诱惑。
<br/>　　“那你知不知道我是谁？”李承乾见那小家伙眼巴巴的样子，顺便将肩膀上的树枝杵到了地上，摘了一颗大一些的樱桃递给他。
<br/>　　小家伙很有礼貌，先是对李承乾行了个礼然后才接过樱桃，不过却没有吃，小眼睛依旧盯着树枝，口中答道：“不知道！”
<br/>　　“那你是跟谁出来的？”
<br/>　　“跟阿娘！”
<br/>　　“那你娘呢？”
<br/>　　“我娘……”直到此时，小家伙才发现，带着他出来的娘亲竟然不在身边，眼中顿时露出一丝恐惧。
<br/>　　“我带你去找你娘好不好？”李承乾一见小家伙要哭，顿时不敢再逗他，樱桃树枝往小家伙手里一递：“这个也送给你！”
<br/>　　“我不要，我要找娘亲！”找不到娘的小家伙哪里还有心思要什么樱桃树，游目四顾的四下搜索，试图在人群中找到母亲的影子。
<br/>　　“好好好，我带你找娘亲好不好？”
<br/>　　“我不要，我要找娘亲！”
<br/>　　李承乾这下尴尬了，虽然他不是故意把这小子引走的，但这件事多少和他也有些关系，若是老秦将来知道了这件事情，非得找自己麻烦不可。
<br/>　　想到这里，李承乾连忙把树枝交给苏猛，然后对那小家伙说道：“你还记不记得你的玉佩是谁给你的？”
<br/>　　“玉佩？”小家伙愣了一下，看着李承乾说道：“是，是太子殿下在我出生的时候送给我的。”
<br/>　　“那么如果我说我就是送你玉佩的人，你相信么？”李承乾继续哄孩子。
<br/>　　“不信，你怎么可能是太子殿下，太子殿下是很厉害的，走在路上会有上千人给他开路！”小家伙被李承乾带歪了心思，顺着他的话说道。
<br/>　　不过这小子并不知道，他无心的一句话让李承乾尴尬不已。
<br/>　　苏玫眼见李承乾哄孩子的计划失败，不由有些好笑，但作为太子妃她此时也不能站在一边看着，所以便也在那小家伙面前蹲了下来，轻声说道：“怀玉乖，姐姐带你去找娘亲好不好？”
<br/>　　“……”小家伙没有说话，看了一眼李承乾，又看看苏玫，最后点点头向苏玫走了过去，留下李承乾蹲在地上无奈的叹了口气：“这特么辈份论的……。”
<br/>　　……
<br/>　　西市市署，也就是传说中的市场管理所。
<br/>　　秦氏在一个小丫鬟的搀扶下正一脸担心的向外面张望着。
<br/>　　而在她身边，西市署令赔着小心说道：“夫人，您放心，下官已经命人封锁了西市所有出口，一旦发现小公爷，马上就会将人带回来。”
<br/>　　“你的保证有什么用……”秦氏看都没看那署令一眼，略有些失望的转回身。
<br/>　　本来她这次并不想带着儿子出来，可是耐不住儿子多番央求，最后还是带着他出来了。
<br/>　　结果没想到，就在跟丫鬟商量哪根簪子好看些的时候，那臭小子竟然自己就跑没了影子。
<br/>　　这下可吓坏了她和那个丫鬟，两人找寻秦怀玉不见之后，便来到市署报了案。
<br/>　　可报案之后时间过去了那么久，孩子却依旧没有找到，这上她怎么会有心情搭理那署令。
<br/>　　而就在秦氏六神无主，不知如何是好的时候，一个清脆的童音突然传进了她的耳朵：“娘亲，娘亲！”
<br/>　　儿子！被熟悉的声音呼唤，秦氏的身体僵硬了那么一个瞬间，但很快便迅速的转回身。
<br/>　　却见已经走失了好久的儿子正张着两只小手飞快的向自己跑来，小脸上正闪动着喜悦的光芒。
