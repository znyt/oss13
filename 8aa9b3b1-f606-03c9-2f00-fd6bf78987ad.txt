<br/><br/>　　    圣殿骑士离众人还有一段距离的时候缓缓减速，手里高举着一块令牌表明着自己的身份。<br/>　　<br/>　　    所有人神圣联盟的人都单膝跪地，低下了头。甚至一些地位较低的人直接双膝跪地，身体前伸，双臂极力的伸展到了前方，以头拄地。<br/>　　<br/>　　    “教皇陛下福寿无疆！”所有人都在高声唱喏着。<br/>　　<br/>　　    “奉教皇陛下口谕，请卡罗皇家学院的老师和学员们入城！”圣殿骑士声音浑厚，语气中充满了恭敬。<br/>　　<br/>　　    “领法旨！”圣殿骑士说完以后，所有神圣联盟的人恭敬的说道。<br/>　　<br/>　　    紧接着，神圣联盟的人一个个的站了起来，轻轻拍打了一下衣服上的尘土，然后开始忙碌了起来。<br/>　　<br/>　　    “德利莱卡大师，在下还要回去向教皇陛下复命，无法陪伴一同入城，还请见谅！”报信的圣殿骑士对着德利莱卡拱拱手。语气虽然恭敬，但是听起来多少还有一丝傲慢。<br/>　　<br/>　　    德利莱卡不知道是处于伤心之中没有听出来，还是听出来以后并没有在意，只是平静的拱拱手回道：“客气，有劳了！”<br/>　　<br/>　　    卡罗皇家学院的人看到德利莱卡没有追究，他们就更不好追究了，不过心里都觉得有些无奈和憋屈。<br/>　　<br/>　　    德利莱卡似乎并没有感受到大家的不满，转身坐上了马车。很快一行人也都收拾好了，其实也没有什么，就是简单的整理一下着装。前队开始启动之后，车夫一挥长鞭，马车缓缓跟随着前队行去……<br/>　　<br/>　　    “嘭嘭嘭！”无数魔法礼炮在皓理曦城的上空炸响，皓理曦城城门口红毯铺地，两侧铺满了鲜花。<br/>　　<br/>　　    无数身着盛装的百姓捧着鲜花夹道欢迎，很多银盔银甲的士兵在维持着现场的秩序。<br/>　　<br/>　　    十几名穿着白色长袍的中年人簇拥着几名穿着白色长袍的老者，正在红毯的另一头等待着，为首的一人正是之前见过的那名长老巴特利特。<br/>　　<br/>　　    卡罗皇家学院的一行人看着面前这盛大的欢迎仪式有些发懵，他们实在难以置信，这欢迎仪式是为了他们而举办的。<br/>　　<br/>　　    德利莱卡反而从容了很多，他心里很清楚，这场欢迎仪式其实就是一个官方的作秀，对外显示对自己一行人的重视，其实就是对卡罗帝国的示好。再一个也是给自己一个面子，毕竟自己是一名圣魔导师，面子工程还是要做一下的。<br/>　　<br/>　　    “德利莱卡大师，对这个欢迎仪式您是否还满意？”巴特利特微笑着看向德利莱卡。<br/>　　<br/>　　    这些人里面显然职位最高的就是这个巴特利特了，难怪他要急着回来，一个是通报，再一个是迎接。<br/>　　<br/>　　    “还好！没想到你们竟然会将迎接仪式搞得如此隆重，让我们反而有种受宠若惊的感觉了。”德利莱卡礼貌的回应着，不过心中却担忧了起来。<br/>　　<br/>　　    德利莱卡早就料到教皇肯定不会亲自来迎接的，毕竟那是相当于国王赵天龙的存在。尽管自己一行人肩负着的是卡罗帝国的荣誉，但毕竟明面上代表的还是卡罗皇家学院。一国的首脑屈尊去迎接另一个国家的学院代表，显然是不合时宜的。<br/>　　<br/>　　    其实这一次来到神佑联盟的并不是只有卡罗皇家学院这一所学院，卡罗帝国还有三支学院队伍早已经来到了这里，不过他们并没有受到如此隆重的欢迎。尽管他们没有全部来迎接，但还是派了一些代表前来。<br/>　　<br/>　　    三个学院的代表各个都是人精，他们三所学院虽然不能和卡罗皇家学院相比，但是也算是卡罗帝国排的上名号的学院了，学院之间的明争暗斗也不是一次两次了，大家对彼此之间还是比较了解的。<br/>　　<br/>　　    尽管他们心中都有些不满，但并没有人表现出来，反而迎接卡罗皇家学院的人的时候一个比一个热情，仿佛和卡罗皇家学院的人是多年不见的老朋友一般。<br/>　　<br/>　　    卡罗帝国学院的老师们没有表现出来什么异样，不代表其他学院的人没有异样。<br/>　　<br/>　　    暮夜联邦这一次派来了三支学院队伍，可是一直当他们到达皓理曦城的时候，也没有受到神佑联盟任何人的接待，也没有享受到任何优惠的待遇，甚至他们吃饭和住宿还要像驿馆交钱。<br/>　　<br/>　　    本来卡罗帝国其他三个学院来到以后，白吃白喝白住已经让他们心中很是不满了，现在看到卡罗皇家学院竟然被如此对待，他们心中又怎么可能咽的下这口气。<br/>　　<br/>　　    不过他们也不是傻子，现在这种情况自然不能做什么，不过暗地里做些手脚还是可以的。暮夜联邦一直在国内大肆宣扬卡罗帝国贵族们对暮夜联邦人民的残害，加深着人民的种族，仇恨，所以他们这些学员原本对卡罗帝国的人就没有好感，此刻卡罗帝国的人在他们眼中变得愈发讨厌了。<br/>　　<br/>　　    神佑联盟如此大张旗鼓的迎接卡罗皇家学院的人，难保不是也有这么一层想要激化矛盾的意思。<br/>　　<br/>　　    要知道表面上卡罗皇家学院是卡罗帝国实力最强的学院，让他们内部学院之间不和，让暮夜联邦的学院和他们矛盾进一步激化，最后受益的只能是他们神佑联盟。<br/>　　<br/>　　    尽管那些迎接的老师们极力奉承，但他还是从他们眼中的最深处看到了羡慕嫉妒恨等复杂的情绪。<br/>　　<br/>　　    德利莱卡自然不会惧怕他们，也不会害怕他们捣鬼，但是这么一来，卡罗皇家学院就会成为众矢之的，李振邦他们这十名学员的压力就会大增，万一出现折损可不是他想看到的。<br/>　　<br/>　　    “一会儿我带你们参观一下我们圣都，你们不辞劳苦远道而来，我们也好尽一下地主之谊，让你们在圣都好好游玩两天，一切的花费都有我们教廷来负责！离东西大陆的学院大赛还有一段时间，你们也可以好好休息一下。”巴特利特热情的说道。<br/>　　<br/>　　    “老师和学员们都已经很累了，我想他们暂时更需要休息而不是游玩。你的好意我们心领了，不过还是先带着他们去休息吧！我现在最想知道的是，你什么时候会带我去见你们的教皇。”德利莱卡现在的心中已经有些焦躁起来，语气也变得强硬了一些。<br/>　　<br/>　　    听到德利莱卡语气有些不善，巴特利特十分不解，但还是点了点头，放弃了带着他们游玩的念头。安排人带领着卡罗皇家学院的师生们去住宿，自己亲自带着德利莱卡去面见教皇……<br/>　　<br/>　　    卡罗皇家学院的师生们并没有被安排在驿馆之中，而是全都被安排进了皓理曦城最奢华的酒店——神恩酒店之中。<br/>　　<br/>　　    东西大陆来参赛的所有学院，除了位于皓理曦城的光明学院的师生住在学院以外，其他的学院人员一律住在驿馆。<br/>　　<br/>　　    待遇最差的自然是那些暮夜联邦的师生们了，他们吃住行全都要自费。驿馆的东西相对外面的饭店自然要便宜很多，但是暮夜联邦本身经济就很拮据，即便驿馆的饭菜便宜，他们每个人一天也只能是勉强能填饱肚子。<br/>　　<br/>　　    最后没有办法，为了改善参赛学员的伙食，暮夜联邦学院的老师们纷纷出城为学员们抓来野味改善生活。<br/>　　<br/>　　    神佑联盟的师生们在吃住上的待遇和卡罗帝国来的另外三支队伍差不多，吃住行完全免费，饭菜种类繁多，也比较丰盛，晚餐每个人都会有一块儿魔兽肉提供。<br/>　　<br/>　　    不过依然还是有区别的，最大的区别就是，那些服务人员面对神佑联盟的师生们比较热情，面对卡罗帝国其他学院的师生的时候都板着一张冷脸。<br/>　　<br/>　　    和这些学院的师生们相比，卡罗皇家学院的师生们简直就是生活在天堂一般。一天二十四小时饭菜随叫随到，吃住不但完全免费，而且每天都有专门的服务员为他们打扫卫生。这还不算，驿馆的人员见到卡罗皇家学院的师生们都热情的打着招呼。<br/>　　<br/>　　    一次两次也许大家也就得过且过了，问题是离真正比赛还有一段时间，而众人又不会窝在房间中一天不出门，所以大家都是抬头不见低头见。也正是因为经常相见，所以这份积怨也就越来越深。<br/>　　<br/>　　    所有能来参加比赛的学员们，哪一个在学院里面不都是天之骄子，学院里面最好的资源都会倾向于他们。<br/>　　<br/>　　    这些天之骄子哪里受到过这种憋屈，现在每个人心中都憋着一口气，就连同是卡罗帝国那些学院的师生们对卡罗皇家学院的师生们也是颇有微词了。<br/>　　<br/>　　    德利莱卡去拜见教皇的时候，等了整整一个下午，这才见到教皇一面，结果教皇还以政务繁忙为由，只是和德利莱卡简单的寒暄了几句话，就让他先回去了。<br/>　　<br/>　　    回来以后德利莱卡才发现自己去见教皇就是一个最大的错误，卡罗皇家学院的众人已经被安排进了神恩酒店，现在生米已经煮成熟饭，德利莱卡也没有什么好办法了。<br/>　　<br/>　　    看到众人享受的样子，德利莱卡实在不忍心让众人放弃优越的条件去选择住宿驿馆。先不说大家会不会愿意，就算愿意，恐怕驿馆也没有那么多的地方来安置这些人了。<br/>　　<br/>　　    德利莱卡现在只能眼睁睁的看着大家沦为众矢之的，却无可奈何……<br/>　　<br/>　　    （第五卷了，并没有像其他人一样，按剧情分卷，我都是一百章一卷。也许等到完结的时候会详细的分卷吧！希望大家继续支持，谢谢！）<br/>　　<br/>　　    最快更新m.
