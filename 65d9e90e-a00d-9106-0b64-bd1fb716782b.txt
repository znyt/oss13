<br/>				虽然以叶尘如今的实力，已经不需要法器来防身，但他要为亲人朋友打造护身法器，还要为自己的青龙大阵，搜寻一些极品玉石之类的原材料。<br/>　　<br/>　　在大壮和阿虎的带领下，一行四人很快便来到那个地下拍卖场。<br/>　　<br/>　　“传言那大山深处，珠峰附近，有仙人墓穴，因而经常有人进入其中寻宝探险，而一些人得到宝贝之后，便会拿到这个地下拍卖场来拍卖……”<br/>　　<br/>　　“据说这个地下拍卖场，背后有鬼灵宗支持，所以没有人敢在这里闹事！”<br/>　　<br/>　　大壮和阿虎两人如同献宝般，轮流向一旁的楚飞烟介绍着情况，楚飞烟则随口敷衍着。<br/>　　<br/>　　四人来到地下拍卖会的门前，每人交了1ooo块钱的入场费，门口的保安这才放行。<br/>　　<br/>　　进入其中，只见里面十分简陋，中间是一个圆形的拍卖台，周围则是一些桌椅，不过里面的人居然不少，足足有近百人！<br/>　　<br/>　　四人进去的时候，刚好在拍卖一颗佛珠，拍卖师激情四射的推销着，台下人声鼎沸，十分热闹。<br/>　　<br/>　　很快，四人寻了一空着的位置，坐了下来。<br/>　　<br/>　　“楚小姐，待会你若是看上什么法器宝贝，尽管告诉我，百八十万对我来说，也就是一个月的零花钱而已！”<br/>　　<br/>　　大壮拍了拍胸脯，冲着楚飞烟财大气粗道。<br/>　　<br/>　　正所谓穷文富武，两人能拜在萨市鼎鼎有名的郭师傅门下习武，家境情况自然都不差。<br/>　　<br/>　　阿虎也当即跟着道：<br/>　　<br/>　　“是呀，楚小姐若是有什么需要，尽管跟我们兄弟讲！”<br/>　　<br/>　　可惜，楚飞烟对两人这一套，根本不感冒，淡淡道：<br/>　　<br/>　　“不用！我自己有钱！”<br/>　　<br/>　　说完这话，楚飞烟微微一顿，又望向一旁的叶尘，轻声道：<br/>　　<br/>　　“叶师，你如果有看上的物件，就告诉我，我可以帮你拍！”<br/>　　<br/>　　大壮和阿虎听到楚飞烟这话，顿时纷纷朝叶尘投来嫉恨的目光。<br/>　　<br/>　　原本，他们见叶尘不过就是一个小屁孩，所以并没有将他放在眼里，可是这么长时间接触下来，他们这才猛然现，这位楚大美女，似乎对这个少年，热心的有些乎寻常，而对他们两个却总是爱理不理。<br/>　　<br/>　　但，这还不是最可气的，最可气的是，面对楚飞烟乎寻常的关怀，叶尘却仿佛石头人一般，始终不咸不淡，不冷不热。<br/>　　<br/>　　有句歌词怎么唱的来着，得不到的永远在骚动，被偏爱的有恃无恐……<br/>　　<br/>　　这让他们两个，如何不气不怒不嫉妒？<br/>　　<br/>　　望着楚飞烟那一脸认真的俏丽脸庞，叶尘也不由得暗暗苦笑，心想说，<br/>　　<br/>　　大姐呀，你这是在故意给我拉仇恨吗？难道你就没有看到，这两个家伙已经恨不得要吃了我吗？<br/>　　<br/>　　不过，一阵苦笑过后，对于两人的浓浓敌意，叶尘也没有太放在心上，随口说了一句“不用！”之后，目光便落在拍卖台上，开始观察起来。<br/>　　<br/>　　大约过去了二十来分钟，叶尘并没有现什么值得拍卖的东西，而且此时拍卖会似乎也已经进入尾声，叶尘不由得摇了摇头，正打算提前离开。<br/>　　<br/>　　就在这时，一件新的拍卖品，被送到了台上，瞬间便吸引了他的注意！<br/>　　<br/>　　这是一只黑不溜秋的小钟，看上去似乎平平无奇，但是叶尘一看之下，顿时两眼微微一瞪，脸色露出一抹惊色，<br/>　　<br/>　　“竟然是灵铁打造的法器……不，确切的说，这应该是一件半灵器才对！”<br/>　　<br/>　　在修真界中，灵器是比普通法器更高等级的存在，是由灵铁打造而成，而且和普通法器最大的区别在于，灵器已经诞生了灵智！<br/>　　<br/>　　叶尘之所以说眼前这个小钟是半灵器，就是因为它虽然是由灵铁打造，已经具备了成为灵器的资质，但是却没有最终完成，所以只能算是一个半成品。<br/>　　<br/>　　“倘若经过我手，将其炼成真正的防御灵器，然后再让它以岚姨为主，今后即便是化劲宗师、神通境的修道者，也休想伤岚姨分毫！”<br/>　　<br/>　　一念及此，叶尘顿时大为振奋。<br/>　　<br/>　　如此，便能彻底解决这个让他最为担心的问题。<br/>　　<br/>　　就在叶尘思绪纷飞之时，台上的那个拍卖师，已经将那小钟向众人展示了一圈，朗声道：<br/>　　<br/>　　“诸位莫要看此物不起眼，据此物的主人讲，此物乃是从那个传说中的仙人之墓中，取出来的物件，虽然迄今为止，还没有人研究出它的使用之法，但也绝对是一件值得收藏的宝物……”<br/>　　<br/>　　经过那位拍卖师
