
<br/>　　<br/>　　    没到年关的时候，去陇西之前，陈三岁又下了一趟湖，当时抚仙湖四周天气不太好，云层压的有点低，一直都在淅淅沥沥的下着小雨，陈三岁划船到离岸三十多米远的水域，就一个猛子扎到了湖里，准备趁着下大雨起风前赶紧回来。<br/>　　<br/>　　    当他跃入水下后，游了片刻就看见了诗远的尸体立在下面，等当他正要在继续下潜的时候，忽然间发现有一条三四十公分长的鱼正朝着尸体游过去，开始时他没太在意，自己也下潜，等到那鱼游到诗远前方的时候，陈三岁就木然发觉，尸体突然张开了嘴，紧接着那条游到她嘴边的鱼就被她一口给咬住了。<br/>　　<br/>　　    当时这一幕陈三岁看的极其清楚和惊恐，因为眼睁睁的瞧见，那条鱼的鱼头在诗远嘴中时后半截身子和尾巴还在剧烈的抖动着，但没几下这鱼居然被硬生生的就给吞了进去。<br/>　　<br/>　　    “我当时就懵了，这要不是我亲眼所见，我是绝对不会信的”陈三岁现在说起的时候，还挺心有余悸的。<br/>　　<br/>　　    王惊蛰皱眉问道：“没走眼？毕竟当时天气不好，还下了雨”<br/>　　<br/>　　    “绝对不会走远，因为……”<br/>　　<br/>　　    “怎么了？”<br/>　　<br/>　　    陈三岁咽了口唾沫，眼神略带惊悚的说道：“因为，那条鱼被她吃完了后，我正好已经下潜到了附近，就看见她忽然转过脑袋，本来紧闭的眼睛突然就睁开了，看向了我”<br/>　　<br/>　　    尸体的那个眼神，陈三岁到现在都记忆犹新，因为她的眼睛里还带着点感情色彩。<br/>　　<br/>　　    王惊蛰听完，也有点愣了，掏出烟点上后，沉思着说道：“这也不太可能啊，魂是被封在尸体里了，可顶多对肢体有支配的能力，连僵尸都算不上，完全没理由出现这种状况啊，往后呢，你又去了么？”<br/>　　<br/>　　    “第二天又去了一次，但这次一切如常，我观察了半天，在水下来回了几趟，确定尸体在没有什么反常之处后我才又开始下手把她封上了”<br/>　　<br/>　　    “然后你紧接着就去了陇西……”<br/>　　<br/>　　    陈三岁摇了摇头说道：“这时本来我还没想去呢，但是又过了一天，晚上我做梦的时候就梦到了诗远，在梦里她翻来覆去的和我说自己很痛苦，让我再把她送回阴间，我随即就醒了，然后就看见窗外的云层压得很低，似乎要下雨了，我走到院子里，看向抚仙湖那边，发觉水浪有些翻滚……湖上阴气非常重”<br/>　　<br/>　　    那天过后，陈三岁觉察到应该是诗远给他拖了梦，这种事在一般人来看不会多想，但在陈三岁来说，这可能就是个预兆了，所以他马上就离开了澄江县，直接飞往陇西找到了茅小草。<br/>　　<br/>　　    这天晚上，王惊蛰陈三岁聊过后，他们暂时也没捋出什么线索和原因，华夏上下五千多年的历史，修道，参佛的人数不胜数，阳间，阴曹地府两个世界，发生什么离奇古怪的事都有可能，就拿那些常人所说的未解之谜来讲吧，几千年都不知道发生多少件了，依旧有很多无法解释出原因来。<br/>　　<br/>　　    于是当天晚上，他俩睡了一觉，准备明天再去抚仙湖上看看。<br/>　　<br/>　　    一夜无话，第二天清晨，吃过早饭以后，两人就出门了，陈三岁说他在湖边有一条小船，到时可以划过去，沉尸的地方离岸边不远，也就三十来米左右，但是再往里湖水就深了。<br/>　　<br/>　　    抚仙湖是国内差不多得算最大的深淡水湖之一了，在滇西也就比滇池，洱海小一点，但水深却是一般湖都比不上的，最深处有一百六十米深左右，这个深度都要比一般沿海的海域还要深了。<br/>　　<br/>　　    两人从家里出来，走在村路上，王惊蛰就看见村子当中一棵树下，坐着几个老人抽着水烟袋在闲聊。<br/>　　<br/>　　    “你在这里三个月了，有没有跟村里的这帮老人们聊过？”王惊蛰问道。<br/>　　<br/>　　    “没有，我平时不太和他们接触，没怎么说过话”陈三岁摇头说道。<br/>　　<br/>　　    王惊蛰叹了口气：“你脑子里装的都是尸油么？抚仙湖养尸地，这都是多少年的传说了，你就不会和当地人打听下，有什么典故或者说法？”<br/>　　<br/>　　    陈三岁的脸“唰”的一下就红了：“我，我不善言辞”<br/>　　<br/>　　    “一把年纪的人了，愣是给自己活成了个二狗子状态，我是真服了！”王惊蛰呵斥了两句，然后走到那颗老树下，提了提裤腿子蹲了下来，掏出一包烟就笑呵呵的散了出去。<br/>　　<br/>　　    “来，来，几位大爷抽一根……”<br/>　　<br/>　　    王惊蛰把烟散了出去，先是扯了几句家常，然后话锋一转说自己是来抚仙湖旅游的，就想跟他们打听下这抚仙湖有啥故事。<br/>　　<br/>　　    “大爷，我听说这湖下面好像有很多尸体啊？人都说，抚仙湖是养尸的地方，这是从哪里来的说法啊？”<br/>　　<br/>　　    一老头敲了敲手里的烟袋，从耳朵上拿下王惊蛰递给他的烟，说道：“这事吧以前没有人说，是二十几年前才传出来的，是有人下到湖底看见的，不过呢，在这之前我们也见过有尸体飘上来”<br/>　　<br/>　　    王惊蛰顿时一愣，好奇的问道：“这是怎么说的呢？”<br/>　　<br/>　　    另外一老头，接着说道：“我就碰到过一次啊，年轻的时候我们都在湖上打鱼，有一年的夏天吧，我和几个人开船到了差不多湖中心一带，这天气忽然就变了，嚯，没用上几分钟大雨就下了，然后还起了大风，湖面上的浪太大了，我们都没来得及回岸，那渔船来来回回的晃荡着，差一点就要翻了，不过还好这雨下的时间也不是很长，半个多小时就过去了，风浪小了以后湖面就平静了下来，然后我就看见远处的水面上飘着具尸体，越飘越近，没过多久就到了渔船这边，于是穿上的几个人就把那尸体捞了上来，开始还以为是有人溺水了，或者是以前淹死的沉到了水下，被水浪给卷起来的，等后来我们一看呢，顿时就吓了一大跳，这才觉得不是那么回事”<br/>　　<br/>　　    王惊蛰问道：“怎么就不是了呢？”<br/>　　<br/>　　    那老人说道：“呵呵，因为这尸体穿的衣服就不对啊，现在的人谁还会穿古装留长发啊……”
