　　第210章
<br/>　　“你也到该嫁人的年纪，毕竟哪儿都大了。”程大雷打量李婉儿一眼：“嫁给谁啊？”
<br/>　　“我还不知道呢。”
<br/>　　“呃……你让我觉得很尴尬。”
<br/>　　“父皇属意我嫁给今次的武状元。”
<br/>　　“那还不是好事。从来美女配英雄么，你虽然算不上美女，但娶个英雄还不偷着乐去。”
<br/>　　“你根本不知道这次武科举背后的事情。”李婉儿怒目道：“你什么都不知道。”
<br/>　　“那请董事长传达一下精神，我们也好学习学习。”程大雷道。
<br/>　　李婉儿冷哼一声，稳了稳神道：“这次武科举，本是为帝国选拔带兵打仗的将领，不仅只考武艺，还包括谋略，阵法，统兵等等。”
<br/>　　“原也应该。”程大雷点点头。
<br/>　　“朝廷定下三千六百五十二个名帖，传到各州各城，帝国十三州一百单八城，每城分些名额，被选拔出来的人，拿着写着他们名字的名帖，可以到长安参加武举。”
<br/>　　“以帝国之大，这三千六百五十二人中，一定是高手如云。”
<br/>　　“不是高手如云，而是贵人如云。”李婉儿冷哼一声：“你知道他们怎么确定人选么？”
<br/>　　“当然是选拔武艺出众之人，在参加长安大武举之前，各城先来一次小武举，选出来的都是顶尖人物……”程大雷说着，忽然顿了顿：“看你的意思，好像不是这样。”
<br/>　　“当然不是，这次朝廷出了大力气，即便最后没有夺魁，只要过程中表现得好，被朝廷看到，也会委以重任。”李婉儿道：“这样的机会，绝对给不了平民，各大世家已经抢破了头。有些城池是公然出售，谁出的价高名帖就是谁的。有些是城里的大世家瓜分。”
<br/>　　“若是这样，好好一次武科举，岂不就成了各大势力的分赃大会。”
<br/>　　这就像前生的选秀一样，如果安插一两个关系户其实可以理解，但要都是关系户，能选出怎样货色。程大雷正想着，忽然一顿，觉得事情可能比李婉儿想的还要复杂。李婉儿知道庙堂之高，这是人家的血统天赋，但李婉儿不懂得江湖之远，这是程大雷的职业天赋。
<br/>　　武科举并不是三年一届，五年一届，而是从来没有。以前帝国只有文举，没有武举，一直帝国的武人都没有向上的通道。江湖人可不管你这个那个，也不问你武举除了靠武艺，还要考军事。好不容易有个机会，谁肯放过。
<br/>　　没有名帖怎么办，抢呗，守着穷山恶岭，看到有进京赶考的，直接一刀捅了，拿着他们的名帖上京。反正这时代也没有照片，我说我叫名帖上的名字，谁知道我在家叫狗蛋。
<br/>　　虽然名帖上的三个字他们都可能认不全。
<br/>　　程大雷这种想法并不夸张，很可能这种事情已经出现了。如今长安聚集的武者，又不知道里面有多少山贼，多少凶徒。
<br/>　　程大雷想着，忽然失声笑出来：事情热闹了喔。
<br/>　　“你笑什么，军国大事，很可笑么？”
<br/>　　“没有，没有，李董请继续。”
<br/>　　李婉儿白了程大雷一眼，又道：“这只是一个名帖，便争成这般模样，何况最后的状元。谁若这次夺魁，是要直接统领精兵，镇守西北边关的，日后帝国朝廷，必有他的一席之地。那些世家怎么可能让这位置落到旁人手中，如今长安的各大势力已经争得不可开交，势要让这状元出在自己派系。而主持这次武举的也正是他们这些人。”
<br/>　　“又当裁判又当选手啊。”程大雷一惊。
<br/>　　“若是单纯比武艺，却还好说，输就输赢就是赢。可这兵法怎么比，又不是真刀实枪打仗，无非是纸上文章而已。还不是他们说谁胜就是谁胜。”
<br/>　　“评委都不是好东西。”程大雷重重哼一声，忽然道：“那这种事，你父皇不知道吗？就看着下面人把武举搞得这样乌烟瘴气。”
<br/>　　“知道啊，所以我父皇要把我嫁给状元，无论是那个派系的人夺了状元，最后都变成父皇的女婿。”
<br/>　　“嘶！”程大雷倒抽一口凉气：“明帝陛下竟然恐怖如斯！”
<br/>　　“可是我并不想成为谁的政治手段，我要嫁的人必须是个顶天立地的大英雄，胸怀精兵百万。”李婉儿抬起头，眼中隐隐发光：“就像那帝国战神杨龙停一般。”
<br/>　　噗！
<br/>　　徐神机三人同时把茶吐了出来。
<br/>　　“他们怎么了？”李婉儿。
<br/>　　“大概是觉得你应该多读一些书，提升下审美观吧。”程大雷吐着嘴里的茶叶。
<br/>　　“程大雷，你可是我的人，这次一定要帮我。”
<br/>　　“怎么帮，是不是要我毁了这次武科举，我现在就去办，召集兄弟，十日之内，必定让长安城大乱，武科举也搞不成。”
<br/>　　“呃……你还真是山贼出身啊，能不能不要这么暴力。”
<br/>　　“那是什么……”程大雷一楞，道：“你该不会，该不会是想让我参加这次武举吧，这可不行，我只是一个山贼，可没想给帝国镇守边关。”
<br/>　　“你倒是想，先得看你有没有这个本事，你一个山贼还想夺武状元。”
<br/>　　“那是干嘛。”程大雷顿了顿：“请公主殿下明示，小生当真是糊涂了。”
<br/>　　“不是让你去考科举，而是让你……”李婉儿压低声音：“是让你去当考官。”
<br/>　　“呃……这个是不是有些太扯了？”程大雷道。
<br/>　　“不扯，合情合理。”李婉儿道：“你去做考官，把那些面目丑陋的，歪瓜裂枣的，我看不上的淘汰掉。”
<br/>　　“那么，你已经能把我安排进考场了。”程大雷道：“你如今这么大势力？是个小阴谋家啊。”
<br/>　　“不能。”李婉儿道：“不过我已经查清楚了，这次考官有些是相府派的，有些是将军府派的，有些其他势力派的，但有一个人，不属于任何一家势力，长安也从没人见过他，我要你乔装成他的模样，混进考场。”
<br/>　　“谁？”
<br/>　　李婉儿凑到程大雷耳边，轻启贝齿，口中吐出一股兰香，缓缓道出三个字。
<br/>　　“牛三斤。”
<br/>　　
<br/>(本章完)
