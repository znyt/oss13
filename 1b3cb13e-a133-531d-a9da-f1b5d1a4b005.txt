        &lt;h3&gt;第三卷第一百一十九章  都是戏精&lt;/h3&gt;
<br/>
<br/>        陈子昂看着坐下的人们和剩下的一个座位，心里是百感交集，三十年河东，三十年河西，世上之事就是这么奇妙。
<br/>
<br/>        十年，弹指一挥间，苍茫两云烟，俯首花轻谢，留影成昨天。
<br/>
<br/>        十年，不怕春风老，不怕寒风斜，唯有心如旧，浓情亦入血。
<br/>
<br/>        细想一想，她和张函已经分别快十年多了。
<br/>
<br/>        时间真是磨人的小妖精，总是在不经意间给你花样的小动作，总是让你措不及防。
<br/>
<br/>        服务生已经替她拉开了座椅，她道了一声“谢谢”，轻柔的坐下。看着她是风淡云轻，其实她心早已经像沸腾的开水，咕嘟咕嘟的冒着泡，灼得她心隐隐作痛。
<br/>
<br/>        他们终究会相遇，终究和普通的朋友一样，会坐下来聊天、说话，这样也好。
<br/>
<br/>        酒场上的规矩，都是主坐上的人先发言，今天原本是他们几个人聚会，应该是没有那么多的规矩，只是加入了张函，张函不是他们这个圈子上的人，第一：年龄比他们都大，是江俞轩的妻哥，第二：张函身份特殊。所以就显得有些正式了。
<br/>
<br/>        陈子昂坐下之后，气氛多少有点滞，如果说是在平时的应酬上陈子昂会发挥自如，可是今天，她控制得再好，心里还是乱的。
<br/>
<br/>        “张总，说实话，我都没有想到我能遇见你。”寒伯安打破微微有点尴尬的气氛接着说“平时只是闻名，今天一见你，真的可以用人中翘楚来形容了。”
<br/>
<br/>        “寒总说这话，我这老脸该红了，在座的那个不是人中翘楚啊。”张函笑着看了一眼大家。
<br/>
<br/>        “当然，当然，我和哥哥也是人中翘楚。”落妍适时的补上一句话，她不能让爸爸孤军作战。
<br/>
<br/>        张函这几年在工作上的磨练，对情感的处理，让他已经变了很多，早已经不是陈子昂当初认识的那个张函了。
<br/>
<br/>        他低调、沉稳、内敛，虽然穿着休闲，可是遮不住那散发出来的浑然天成的成熟男人的魅力以及一股由内而外的威严和正义感。
<br/>
<br/>        大家被落妍的一句话逗笑了。
<br/>
<br/>        “是啊，尤其俩个小朋友更是人中龙凤。”张函不忘夸一下自己的一双宝贝。
<br/>
<br/>        寒伯安及江俞轩他们立马被张函的厚颜折服了，哈。有这样夸自己的孩子吗？不过这演戏的功夫真是炉火纯青啊，不去当演员太可惜了。
<br/>
<br/>        “谢谢张总夸奖，我们俩个也就是妈妈调皮的野小孩。”陈虞漫不经心的感谢着张函的夸奖。
<br/>
<br/>        陈子昂听了陈虞的话，心里一惊，这孩子，怎么这样说话呢？
<br/>
<br/>        “怎么会是野小孩呢？这么可爱的孩子，对了孩子的爸爸没来？”张函转过身问陈子昂。
<br/>
<br/>        “这脸皮比城墙还厚了，怪不得陈子昂乖乖的被他骗。”三个男人唏嘘着，厉害，佩服啊。
<br/>
<br/>        “我们长这么大都没有见过爸爸，不知道爸爸是个什么东西。”落妍口无遮拦，反正要逼着妈妈承认爸爸啊。
<br/>
<br/>        “嘟嘟！”陈子昂有点急。
<br/>
<br/>        江俞轩正喝着一口茶，“咳咳咳”，江俞轩被呛住了。
<br/>
<br/>        寒伯安实在是忍不住，捂住了嘴。
<br/>
<br/>        郝景文低下头，他憋不住想笑怎么办？
<br/>
<br/>        “妍妍的爸爸是去哪里了吗？”安男终于是忍不住心疼的问，孩子都长这么大了还没见过爸爸，真的是好可怜。
<br/>
<br/>        “妈妈说，爸爸去干大事情去了，过一段时间就回来。”陈虞看着安男解释。
<br/>
<br/>        大家看陈子昂的眼神有一点同情的，陈子昂被这三个戏精包围着，心里一定很“快乐”吧？
<br/>
<br/>        “哎呀，再大的事情也不能对孩子孩子不管不问，孩子都长这么大了，这算是什么男人呢。”安男又气愤又心疼的说，原来陈子昂这么辛苦。
<br/>
<br/>        “就是，这要是在我们哪里铁定就会被骂“渣男”的，我们那里的人才不管你是干什么大事情呢，不管老婆孩子的男人就是“渣男”。”陈思宇伸手搂着落妍，她不但心疼孩子，她还心疼他们陈总，原来陈总这么辛苦，嘤嘤嘤。
<br/>
<br/>        张函感觉风向变了，他看了一眼江俞轩，不是帮着他认孩子的，怎么就帮成“渣男”了？
<br/>
<br/>        “陈阿姨，我爸爸比你们这里的人都好！”落妍不高兴了，敢说她爸爸是“渣男”？她的眼里蓄满了泪水，直接从姐姐将陈思宇升级成了阿姨，并且推开了陈思宇的手。
<br/>
<br/>        看得张函心里一阵心疼，他就要忍不住了。
<br/>
<br/>        “嘟嘟，来坐妈妈这里。”陈子昂看着马上要哭出声的落妍心像被针扎了一样，张函除了对他们母子不闻不问外，在工作上，那可以说是应了当初他自己说的话：“要让这个世界一片清明”。
<br/>
<br/>        他对得起他身上穿的制服，头上戴的帽子和帽子上的徽章。
<br/>
<br/>        她和张函只是缘浅而已。
<br/>
<br/>        落妍起身走到妈妈的身边，站在张函和陈子昂的中间，一副难过委屈的模样。
<br/>
<br/>        陈虞神助攻的换到了妹妹坐的位置上，轻轻的难过的低声叫了一声“妈妈。”
<br/>
<br/>        落妍如愿以偿的坐在了张函和陈子昂的中间。
<br/>
<br/>        “唉，我说子昂，渣男就是渣男，你干嘛还要那么维护他？”江俞轩也看着陈子昂说。
<br/>
<br/>        “对，换我早将他公之于众，接受大家的审判！十年都不管孩子，你还维护他呢，告诉孩子真相呗。他就是渣！”郝景文也义愤填膺的说。
<br/>
<br/>        “咳咳咳，孩子在这里呢，看看你们说的什么话？再怎么渣，也是孩子的爸爸不是？”寒伯安敲了敲桌子。
<br/>
<br/>        “哇”的一声，落妍哭出了声，九岁的孩子怎么能禁得住几个大人的心思。
<br/>
<br/>        “爸爸，你告诉他们你不渣，你告诉他们啊。”落妍摇着张函的胳膊，他们说好了要让爸爸和妈妈偶遇，但是不能让别人说爸爸是渣男吧。
<br/>
<br/>        一声霹雷炸在陈子昂的头顶，陈子昂蒙了。
<br/>
<br/>        “妈妈，你告诉他们，爸爸不渣，你告诉他们啊。”落妍哭着又摇着陈子昂的胳膊，陈子昂任由落妍摇晃着。
<br/>
<br/>        “你说他是你爸爸？”陈子昂艰难的开口。
<br/>
<br/>        落妍看着妈妈的样子吓了一跳，她发现她好像闯祸了，她求助的看着哥哥，陈子昂也看着陈虞。
<br/>
<br/>        “妈妈，难道他不是我们的爸爸吗？”陈虞一脸无畏的看着妈妈，被妈妈骂总比没有爸爸好，虽然他知道爸爸已经有老婆了，但是不妨碍做他们的爸爸啊。
<br/>
<br/>        “你们怎么知道？”陈子昂完全忘记了还有旁人，她的心很慌，脑袋很蒙。
<br/>
<br/>        “我们在你的书房里看过你和爸爸的照片。”陈虞心里也很心疼妈妈，他还是想换一种委婉的方式告诉妈妈，虽然照片是在认识爸爸之后看到的。
<br/>
<br/>        “子昂，子昂，我对不起你们母子，我.......”张函眼含热泪。
<br/>
<br/>        “你闭嘴，有你什么事情？”陈子昂冲张函喊。
<br/>
<br/>        “对，你闭嘴，有你什么事情？”大家同时冲着张函喊。
<br/>
<br/>        陈子昂心里又难过，又想笑。大家这一会都静静的看着陈子昂。
<br/>
<br/>        她轻轻的叹了口气，对俩个孩子也是对大家说；“没错，他是孩子的爸爸。只不过我们分手的时候他并不知道我怀孕了。”
<br/>
<br/>        “对不起啊，张总，我不知道情况，我错了。”陈思宇急忙道歉。
<br/>
<br/>        “对不起啊，张总，我不知道情况，我错了。”大家就像是复读机一样。
<br/>
<br/>        “今天意义重大，恭喜俩个孩子喜得爸爸。来，开香槟！”寒伯安大声说。
<br/>
<br/>        “恭喜陈虞落妍喜得爸爸！”大家举起酒杯。
<br/>
<br/>        罢了，这样总好过以后找借口给孩子解释。
<br/>
<br/>        “我知道大家都是为了我们好，谢谢大家。”陈子昂说。
<br/>
<br/>        ”子昂，以后要是函哥对你和孩子不好了，你给我和俞轩说。“张倩楠看得热泪盈眶，哥哥终于可以和他们见面。
<br/>
<br/>        “对，子昂，你给我说，我替你收拾他！”江俞轩心里的石头终于落地了。
<br/>
<br/>        陈子昂就这样被一群戏精逼着无奈的接受了现实。
