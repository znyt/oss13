文和丁武顿时懵逼，两人面面相觑。<br/>　　<br/>　　    “这，就是你说的什么大道，归一？”<br/>　　<br/>　　    菜刀文捂着脸说道：“哎呀，别说了，我他么是丢不起这个人了”<br/>　　<br/>　　    陈臣身后，那七窍流血的鬼脸，缓缓的探出脑袋，狰狞的冲着王惊蛰呲起了獠牙，很有种挑衅的意味。<br/>　　<br/>　　    “不知好歹！”王惊蛰突然一把推在陈臣的脑门上，直接将她按在了沙发上，右手拇指突兀的就顶在了她的眉心上。<br/>　　<br/>　　    “嘶······”王惊蛰深吸了口气，胸腹顿时膨胀起来，他对着陈臣的小嘴，探过脑袋离着能有三四寸左右的距离，然后猛然一吐气。<br/>　　<br/>　　    一股阳气，瞬间渡进了陈臣的体内，她肩头的两盏阳火“噗”的一下就迅速旺了起来。<br/>　　<br/>　　    女子体虚，本易招鬼，就是因为阳火比男子要小了那么一些，此时她的阳火旺盛之后，身后那鬼脸直接就承受不了，从陈臣附身的三寸处，缓缓的脱离开来。<br/>　　<br/>　　    菜刀文戏虐的神情连忙收敛，他讶异的说道：“不对啊，他的阳气怎么会这么重的，有点超乎寻常了”<br/>　　<br/>　　    基本上咱们都听过一个词，对男人来说叫壮阳，这个壮阳你们可千万别想歪了，不是那个通俗易懂的意思，真正的壮阳，指的是一个男人的阳气方面大壮，一旦阳气壮大到了一定程度，几乎就是妖邪不侵，鬼怪不入的地步了。<br/>　　<br/>　　    其实说来也是，男人阳气重了，身体方面自然也就杠杠有力了。<br/>　　<br/>　　    王惊蛰用一股阳气轻飘飘的把陈臣身后的鬼面给逼出来后，那东西呲着獠牙张牙舞爪的就奔着他扑了过去，王惊蛰不闪不避，任由对方一把掐住了他的脖子。<br/>　　<br/>　　    “你还敢来上我的身不成？”王惊蛰鄙夷的说道。<br/>　　<br/>　　    菜刀文叹了口气，说道：“这玩意，有点低能儿了啊，鬼王和夜游都不敢拿他的三魂七魄怎么样，你这不是找死呢么？”<br/>　　<br/>　　    鬼上身，无非就是附在人的魂上，这鬼面这么一干，无疑是相当于一脚踢在了铁板上，踢的嗷嗷疼。
