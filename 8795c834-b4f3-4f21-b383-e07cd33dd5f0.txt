　　一个字！
<br/>　　“空”！
<br/>　　起初，元始也不认识这字的意思，这是一个他完全不认识的字，但看到它后，元始心中便自然而然地知道了它的意思
<br/>　　他只觉得这字在闪闪发光，上面充满玄之又玄的道韵气息，一切都说不清，道不明，见到此字后，就仿佛是见到了大道的本源，来到大道最深处，只是看了一眼，他便有一种顿悟的感觉。
<br/>　　这就可笑了，以他如今的修为，竟然还有顿悟一说？但事实的确如此，只看了一眼，他对道的理解便在飞速提高着，特别是对空间、虚空大道的理解，更是达到不可思议的地步。
<br/>　　这是一种大道本源所书的道文，一个字体，便代表着一种大道本源。
<br/>　　这种道文是要比之前他看过的“太玄”两字还要精妙无数倍，两者简直不可同日而语，拿“太玄”与之相比，就好比拿砂砾与黄金相比，两者根本不在一个等级上。
<br/>　　元始心中顿时知道，自己的机缘来了，而且还是逆天机缘。
<br/>　　仅仅看上一眼，他就觉得大道之门在向他敞开，如果能够将之完全参悟或者炼化，那么他的实力将发生翻天覆地的变化。
<br/>　　嗡——
<br/>　　突然，元始印的本源空间之中，一道无比明亮的道痕贯穿整片本源世界，这道道痕元始熟悉，正是空间道纹，亦是本源空间中空间大道的代表，此刻这道纹亮起，近乎遮蔽了整片空间，代表着这本源空间大道彻底弥补了。
<br/>　　自今以后，元始印的本源空间更加坚固，元始印威力也更加强悍，若是借助此印施展空间大道，将会得到无比恐怖的增幅。
<br/>　　元始心中一喜，元始印是他的法宝，此刻空间本源大道的完善，威力增强，带给他的好处也是无与伦比的，助他领悟本源大道不说，就其法宝本身的威力，就恐怖无比了。
<br/>　　元始压下心中激动，开始炼化“空”字，可是，不过片刻时间元始便停下来了，他用尽全力，竟然连接触“空”字的核心之处都接触不了，谈何炼化？
<br/>　　元始摇了摇头，此字不是他如今的修为能够炼化的，至少需要等到他突破天道境后再说。不过，虽然不能炼化，但却可以参悟一二。
<br/>　　元始将那初始宇宙所化塔身炼入体内，太初的气息加入到他的宇宙之中，加速了宇宙的进化，大道法则的完善也使得他体内众多宇宙出现许多修炼文明。从某方面来说，宇宙的进化能够反哺给自身，带动自身修为的进步。
<br/>　　随后，元始继续参悟“空”字的玄妙。
<br/>　　洪荒之中，过了数百年，周王朝的分封制度终于出现了巨大的缺陷，而且这种缺陷也是极为致命的。诸侯太多，一直分封下去，不是每一代的诸侯都会对王朝忠心无比。这就导致了一个结果，诸侯纷乱，群雄并起，列国林立。
<br/>　　西牛贺洲，灵山，西方教。
<br/>　　经过数百年的发展，西方教总算恢复了一点生气，但距离恢复到最初，没有数千年的时间，明显是不可能的。
<br/>　　“师兄，我等如今该如何是好？”
<br/>　　准提道人如今已经恢复了一些伤势，也重新斩出善尸与恶尸，不过，实力却下降了许多，他的脸色并不好看，一半是被气到的，还有一半是对西方教的未来充满担忧。
<br/>　　封神劫中，他们明显可以感受到西方大兴的征兆，但如今那种兆头却早已消失不见了，这让他焦急不已。
<br/>　　“莫急，师弟，人族如今乱象已生，每次乱世同样也代表着一个大世的来临，为兄在此次大世之中感受到思想的潮流，我西方大兴的最后一次机会便在此次乱世之中。”接引缓缓道。
<br/>　　“嗯，若真是如此，那我等该如何？”
<br/>　　准提并没有怀疑自己师兄的话，他的修为并没有自己师兄高，他不知道一些天机也很正常，他问的是如何行动。
<br/>　　“此次机会在人族之中，我会让我的执尸转生到人族，至于以后的事，便等天机明了之时再谋划。”接引道。
<br/>　　“善哉！”准提自然没意见，他道：“我的善尸也该出去一走，为我西方教未来做准备。”
<br/>　　说着，准提身上走出一人，那人一身白袍，头发苍白如雪，手持一白色拂尘，与准提有六分相似，正是准提道人的善尸。
<br/>　　“善哉，贫道自今以后为菩提老祖！”白袍道人开口道。
<br/>　　“善哉，东方之地，该道友走一遭。”准提道。
<br/>　　菩提老祖点了点头，便向着东方飞去。
<br/>　　“我也该安排我的执尸转世了。”接引道人凝重的点头道。
<br/>　　……
<br/>　　洪荒一片平静，相安无事。
<br/>　　“轰——”
<br/>　　这天，人间界突然掀起一股莫名的劫气，只是这劫气并不算强烈，还达不到量劫的地步，但也算得上是人族的一个劫难了。但随着时间的流逝，这劫却有愈演愈烈的趋势。
<br/>　　“嗡——”
<br/>　　这天，一声翁响，瞬间惊动洪荒诸多大能。霎时间，诸圣与众多大神通者纷纷感应，将大量神识降临人间界，观察起因。
<br/>　　而有的大能，则开始演算天机，推演未来。
<br/>　　“什么？人间界浩劫将启，封神量劫才过去，这怎么可能？”
<br/>　　“大世之争，百家争鸣，春秋时代即将来临！这是劫难，也是机缘。”这是一些大能从截取的天机中得知的消息。
<br/>　　“人间界大劫，希望别牵扯到其他地方。”一些深受量劫祸害的修士开始为未来担忧起来。
<br/>　　“此次浩劫并不算太大，我并无殒命之危机，我自当前去走上一遭，若是得到机缘，我便能更进一步，说不定能够突破，需要仔细谋划一番。”有的人却是打起其他主意。
<br/>　　“……”
<br/>　　感应到人间界即将发生剧变，无数大神通者们开始议论纷纷，一个个目中精光闪烁，开始为自己在劫中谋划利益。
<br/>　　……
<br/>　　数年后，南赡部洲一个名为迦毗罗卫国的国家，由一个名为释迦族的人族统治，族长净饭王有一子，名为乔达摩?悉达多。
<br/>　　就这样，一位未来能够成长为释尊佛陀的人物出生了。
<br/>　　同时期，在人族九州之地，洪荒七圣之一的太上老子似有所悟，派自己的执尸老子前往轮回，出生在周王朝统治下的一个名为陈国的诸侯国境内，生而白发，鹤发童颜，取名为李耳。
<br/>　　随后，阐教众仙也接受到了元始的命令，可以派遣尸身通过转世或者直接前往人族历练。众弟子如孔宣遣一尊善尸，前往轮回，出生于鲁国陬邑境内，有名为孔丘。
<br/>　　弟子如慈航道人、文殊广法天尊，自封修为，下山历练，各有所遇。
<br/>　　截教之中，通天教主亦派出多宝道人、无当圣母等，或转世修行，或入世感悟。
<br/>　　春秋时代，大梦春秋，思想碰撞，百家争鸣，一个全新的时代即将来临。
<br/>　　……
