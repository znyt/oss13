　　“所以，你那天听我说她跟我是同年同月同日生，你就怀疑了？”叶少阳问道风。
<br/>　　“救世四子，也是灭世四子，四人之间，因为当年魂力纠缠，生生世世都会有某种联系，你与冷玉情劫纠葛，自不必说，这姑娘与你同年同月同日生，又身负这么多秘密，自然引起我怀疑。”
<br/>　　林三生问瓜瓜：“你还没说完，你的任务，还有什么？”
<br/>　　瓜瓜冲叶少阳抿了抿嘴，道：“我的任务，是先在末法时代找到你，帮助你觉醒……这些我之前说了，还有各一个任务，就是分别找到救世四子，与他们逐个相处，希望从中分辨出真正的天选之子，然后扶持他……鬼王知道我的任务，因此才会一直派人追杀我。”
<br/>　　叶少阳震惊，“那你……为什么没去履行你的任务？”
<br/>　　瓜瓜微微低头，又抬起来，咬着嘴唇说道：“跟你相处久了，就离不开了，再说一旦找到真正的天选之子，你的地位就不保了，我不想看到这种事发生，因此我才一直隐藏秘密，就是为了让你相信自己就是天选之子，老大，我相信天选之子能做到的事，你也一定可以做到，只因为你是叶少阳，在这人间法术界，只有你是独一无二的。”
<br/>　　叶少阳无比动容，想了一想，瓜瓜还真是用心良苦，为自己付出了不少。
<br/>　　恍然，叶少阳明白了什么，转头看着他们一个个，吸着冷气说道：“你们的计划，是杀了另外三个少年天才？”
<br/>　　林三生往前走了一步，盯着他道：“你父亲当日听到圣使与转轮王谈话，说的便是有关救世四子的真相，因此他才会让你一旦遇到跟你差不多的少年天才，就杀了他们。我们跟他老人家想法一致——”
<br/>　　林三生用手中的折扇指着叶少阳，一字一顿说道：“杀了他们三个，你就是真的天选之子！”
<br/>　　叶少阳深深吸气，偷瞄了陈悦一眼，陈宇坐在椅子上，面无表情，不知道在想什么。
<br/>　　“可是，我毕竟是个假的。”
<br/>　　“假作真时真亦假，没了真的，假的也是真的，况且你也是四子之一、是叶法善血脉传人，总比别的人都有资格，再说，现在人间法术界仍然是认可你的，只要你赶在真正的天选之子觉醒死之前，杀了他……”
<br/>　　叶少阳喃喃道：“一共四个人，我知道哪个是？”
<br/>　　“宁杀错，不放过。”道风淡淡吐出这四个字，脸上不带一丝表情。
<br/>　　叶少阳暗暗吸气，“连冷玉也杀？”
<br/>　　林三生道：“其一，你俩的关系，谁是天选之子，都没问题，其二，三界皆知她是转世鬼童，纵然她真的是天选之子，说出来也没人信。而且，为了你，她一定会隐瞒身份。”
<br/>　　军师的分析能力，不是盖的。
<br/>　　“所以，另外两个，一定要杀了吗？你们想过没有，所谓天选之子，一定是有原因的，你们这么做是好心我知道，但这么做，很可能影响到天劫之乱的结果，或许，会多死很多人。”
<br/>　　林三生冷笑，“所以，你让我们大义灭亲，尊别人为天选之子？”
<br/>　　不等叶少阳开口，他往下说道：“按照酆都大帝的谋划，真正的天选之子还没觉醒，是不必暴露你的身份的，但你知道，他为什么要对那几个大佬说出真话，地藏菩萨为什么要派人来捉拿你？因为，你让他感觉到了威胁。”
<br/>　　“我？别闹了好吗，我这点实力，放人间还能装一把比，在大帝和地藏菩萨面前算个啥啊，他们能感到什么威胁？”
<br/>　　林三生哼了一声道：“个人实力，算不上什么，我说的是你的势力。”
<br/>　　“捉鬼联盟吗？在大帝面前也不够看吧。”
<br/>　　“捉鬼联盟，只是表面而已，”林三生跟他起来，“先说橙子，她是鲛人公主，鲛人族虽然世居海上，不怎么登陆，但至少是一个种族，数量也是不少，橙子听你的，鲛人部落也就听你的，小九更不要说，万妖之王，青丘山一脉也是空界举足轻重的势力。
<br/>　　还有我……而今也是空界联军元帅，手下有兵。还有你自己的茅山。哦对，还有那个覃小慧，她是大巫仙家族的巫灵信女，以她的地位，差不多能号召所有南方巫师家族帮你……还有，你之前在幻境世界中拯救的那近万生灵，一大部分散落在人间，对你也是马首是瞻……
<br/>　　这些势力加在一起，可能尸族不太在乎，轩辕山不太在乎，因为他们都是规则破坏者，而阴司，是规则制定者，只要你登高一呼，人间，空界，立刻乱套。酆都大帝怎能不怕？”
<br/>　　小九道：“鬼域也一样，有道风和风之谷在，什么都不要做，只要宣布与太阴山结盟……鬼域也就乱套了。”
<br/>　　叶少阳听他们说完，无奈道：“被你们这样一算，我觉得我还是挺牛逼的嘛。不过，这跟我们之前讨论的有什么关系？”
<br/>　　林三生瞪眼说道：“说了这么久，你还是不明白，这么多的势力都绝对支持你，如果现在突然有人跳出来，说他才是天选之子，就算你不在乎，那些支持你的人，难道会服从于他？远的不说，我们这些人，到时候怎么办，去无条件支持一个新的天选之子，听他号令？这就像是临阵换帅，兵法之大忌！到那时候，天下只会更乱！”
<br/>　　叶少阳这才算彻底明白了，默然道：“这意思就是天选之子，非我不可了？”
<br/>　　林三生又露出阴谋家的表情，道：“宋太祖当初如果拒绝黄袍加身，会有什么后果？天下更乱，他也不得善终。少阳，事已至此，没有退路了。”
<br/>　　没有退路了……
<br/>　　是啊，怪自己从前太高调，在三界之内搞出了这么多事，而今不光是自己，整个捉鬼联盟都被卷到了风口浪尖上，如果能应付天劫之乱，所有人都将分的功劳，如果……现在自己就退缩——哪怕自己发誓没心思去争那个什么天选之子，酆都大帝会相信吗？别人会相信吗？
