　　对于云霆的问题，慕清霄陷入沉思。
<br/>　　“保护雷州是修仙门派的事情，不过，正所谓拿人钱财替人消灾，我取走你体内的东西后，会在雷州布下剑阵，可保雷州百年无忧。”
<br/>　　“百年无忧……”
<br/>　　比起源源不断的引来妖魔鬼怪，云霆更希望雷州百姓安居乐业，当下就答应了下来。
<br/>　　“我答应你，希望你别忘记对雷州的承诺。”
<br/>　　微微一笑，慕清霄有自己的坚持，一言九鼎，既然承诺让雷州百年无忧，等取出雷灵珠，他自然会做到。
<br/>　　如果云霆是无恶不作的纨绔子弟，他可能会强取雷灵珠。
<br/>　　但现在看来，后者就是个老好人，倒也令慕清霄没有出手的理由，以至于磨磨唧唧到现在。
<br/>　　“公子，与云儿一起去云府后院吧，那里比较方便。”
<br/>　　几番交谈下来，云垂野自然不信慕清霄是一位大夫，语气也变得恭敬许多。
<br/>　　“走吧。”
<br/>　　慕清霄已经有些不耐烦，与云霆跟在云垂野身后，慢悠悠的向后院走去，不时聊上几句。
<br/>　　“你刚才说我体内的东西，不知我体内究竟有何东西，竟能吸引妖魔大肆来犯。”
<br/>　　体内的雷霆力量，困扰了云霆足足二十年，令他憎恨与好奇的同时，又不得不去使用它来保护雷州。
<br/>　　“你体内的东西叫做雷灵珠。”
<br/>　　慕清霄也不打算隐瞒，因为没有隐瞒的必要，云霆不过只是凡人，怎会知道雷灵珠是何物。
<br/>　　“雷灵珠？好像在某个地方见过关于这一类信息。”
<br/>　　话语刚落，走在最前面的云垂野沉声道：“云府祖先留下的羊皮卷中记载过，雷灵珠是传说中的神物，但怎会出现云儿身上？”
<br/>　　被云垂野一提醒，云霆就想起在那里见过关于雷灵珠的信息，家族藏书阁中的典籍里面就有记载。
<br/>　　慕清霄一愣，也没想到云垂野居然知道雷灵珠。
<br/>　　不过想来也是，云府的先祖曾经可能是修仙门派中人，留下的古籍中有灵珠的信息，倒也不足为奇。
<br/>　　沉默许久，云垂野语气凝重道：
<br/>　　“云儿，雷灵珠不是我们云府能拥有的东西，只会给云府，乃至整个雷州带来灾难，便交给这位公子吧。”
<br/>　　“父亲，孩儿有分寸！”
<br/>　　云霆重重的点头，雷灵珠这等神物，的确不是他能够掌控的东西。
<br/>　　同时，他也明白慕清霄先前话语中的意思，不是讥讽，反倒是实话实说。
<br/>　　平日里，普通妖魔都让他有些费劲，若是引来一些强大的妖魔，雷州恐怕真要面临偌大的灾难中。
<br/>　　跟在身后，慕清霄微微点头，看来这父子两也不是蠢货，知道雷州现在面临的处境。
<br/>　　不多时，三人就来到云府后院，映入眼帘的是一座小湖泊，一座凉亭，颇有些优雅的意境。
<br/>　　来到凉亭中，慕清霄示意了一声，道：“环境倒是不错，开始吧。”
<br/>　　微微点头，云垂野就退出凉亭，本想站在近处，却被慕清霄一句警告的话语吓退数十米。
<br/>　　取出雷灵珠，可能会引来雷击，云霆是雷灵珠现在的宿主，自然不可能朝他劈，至于慕清霄倒是无所谓。
<br/>　　雷霆对他来说，倒不算太大的威胁，只要不一击将他劈成渣，就相当于是在给他锻体。
<br/>　　至于云垂野，虽有金丹的修为，可一旦被雷击波及到，绝对是灰飞烟灭，落得个悲剧的下场。
<br/>　　待云垂野退后数十米，一直退到院落门口，云霆看了一眼若无其事的慕清霄，当即盘坐在凉亭中。
<br/>　　“开始吧。”
<br/>　　慕清霄不语，在云霆对面盘坐而下，双手合十，空间轻微波动一下，一张无形的屏障就向四周蔓延而去。
<br/>　　为了保险起见，慕清霄布下防御结界，哪怕引来雷击，也不会波及云府。
<br/>　　“开始了。”
<br/>　　不等云霆回答，慕清霄神识释放而出，将云霆整个人都笼罩其中，食指与中指合拢，轻轻点在他的眉宇间。
<br/>　　旋即，一股能量波动以两人为中心向四周扩散而出，小湖的水面也渐渐荡起一阵阵涟漪……
<br/>　　此时，慕清霄的灵魂力量正在云霆体内搜索起来。
<br/>　　不多时，便在后者的腹部位置，寻找到一颗透着蓝色光晕，半个拳头大小的圆润珠子，正是雷灵珠！
<br/>　　雷灵珠，乃是蕴含着无穷法力的雷系灵珠。
<br/>　　慕清霄只是用灵魂力量查探，便能感受出，雷灵珠中蕴含的力量并不比土灵珠庞大，性质却极其的狂暴。
<br/>　　想到炼化土灵珠的土属性能量，慕清霄瞥了眼雷灵珠，知道接下来又得遭罪了。
<br/>　　回过神来，慕清霄不得不说，云霆这小子运气倒是不错，好在他的资质不高，否则，可不仅仅是取出灵珠这么简单了。
<br/>　　如果云霆资质不错，修成金丹，雷灵珠在他的腹部筑巢，他的一身修为也会转移到灵珠中。
<br/>　　聚时，云霆定然能够多发挥一丝雷灵珠的力量，但取出灵珠，就相当于废除他的一身修为。
<br/>　　取出雷灵珠后，云霆就会沦为一介废人，根基损坏，修为溃散，简直比杀了他还难受。
<br/>　　或许普通人会觉得没什么，可对于修士而言，废除修为无疑于杀父之仇，夺妻之恨。
<br/>　　虽然，后者只是个半吊子修士。
<br/>　　不过，现在为时不晚，以云霆的资质，想要修成金丹，不知道要等到猴年马月呢。
<br/>　　毕竟，修道一途可不光光是努力就够了，当然也不缺乏毒奶粉，但资质与资源却占据了绝大部分因素。
<br/>　　而云垂野不过是个金丹修士，由此可见，云府在修炼上的底蕴并不深厚。
<br/>　　没有资质与资源，想要掉渣天？

<br/>　　微微摇头，将复杂的情绪抛之脑后，慕清霄目光变得严肃起来，接下来就是最后一步了。
