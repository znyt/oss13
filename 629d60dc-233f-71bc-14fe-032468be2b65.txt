　　元始一朝悟道，一刀三连斩，三尸尽落。
<br/>　　轰隆隆！
<br/>　　这一刻，元始周身的神窍洞天在迅速扩展，之前已经达到洞天极致的空间扩大到了小世界的程度。
<br/>　　一万里，两万里，三万里……
<br/>　　一直快要扩展到五万里，这才慢慢停了下来。三百六十五个小世界，在疯狂的吸收着天地之间的一切力量。
<br/>　　先天灵气，星辰之力，太阳太阴之力，大地之力，四海之力，虚空之力，混沌之力……各种各样的力量，他完全不怕力量斑驳，造成反噬，他来之不拒。
<br/>　　他的力量，道行，在成倍的增加。达到了准圣的极致，只差一步，便可证道成圣。
<br/>　　但在这一刻，元始压住了，没有第一时间用鸿蒙紫色成圣。
<br/>　　因为，他的道，还没有圆满。
<br/>　　轰隆隆！
<br/>　　一股强大的气势，以昆仑山为中心，扩散而出，一直扩散到整片洪荒天地，凡金仙以下者，在此刻，全部跪伏在地。只因为这气势，实在是太强了。
<br/>　　九秒之后，元始才收起全身的气势，这一刻，他感受到了强大无比的力量。能够在准圣圆满境界中，更进一步，没有什么比这更好的了。
<br/>　　虽然他还没有突破混元，但他知道，那距离自己已经不远了。
<br/>　　这时，三道身影从元始身体之中走出，渐渐地，从虚幻凝聚成真实。一道身穿黑袍，头顶黑莲，与元始没有一丝相像之处，他的眼中，满是淡漠，他的全身，散发着无尽的魔性。
<br/>　　他就是元始的恶尸。
<br/>　　另一道身影，全身穿着白色道袍，头顶白莲，面目与元始有七八分相似，他面上带着无尽善意，他仿佛就是光的化身，带给人间温暖。
<br/>　　他，是元始的善尸。
<br/>　　最后一道，身穿红色长袍，头顶混沌钟，不苟言笑，与元始有七八分相似，他的仿佛道的化身，他的一举一动，都充满了道韵。
<br/>　　他，是元始的执尸。
<br/>　　“见过道友。”这一刻，三尸都朝着元始本尊行了一礼。
<br/>　　“你我本一体，何须多礼。”元始淡淡道。
<br/>　　“斩三尸之法，的确是非常可怕的道法，但，这是鸿钧的道，而不是我的道。”元始喃喃道。
<br/>　　“善恶执三念，本为人体的一部分，斩去了，也就不完整了。我的道，不管他是善，还是恶，他都是我。执念又如何，有了执念，就不可证道了？谁规定的！本座不信。”
<br/>　　这一刻，元始再次将三尸收回体中，仔细体会这一次斩三尸的领悟，他要通过这斩三尸之道，领悟出自己的神通。
<br/>　　不再理会他人，元始继续闭关。
<br/>　　却说洪荒之中，众位大神，在元始突破的那一刻，全都被惊住了。实在是因为元始突破的动静太大了，他们不得不惊。
<br/>　　“什么？！那是……玉清元始？！”
<br/>　　“突破了，怎会有如此大的动静？难道是直接成圣了？”
<br/>　　“玉清元始成圣了？！”
<br/>　　……
<br/>　　众多修士，纷纷猜测，是不是元始成圣了，如若不然，又怎会弄出如此大的动静。
<br/>　　西方灵山，接引与准提正在悟道，忽有所感，抬头看向东方昆仑方向。一股气势惊天而起，轰动洪荒。等这股气势过去了，两人脸色变得凝重起来。
<br/>　　沉默了一会儿，接引说道：“之前是吾等猜错了，那元始并没有突破准圣，他是在积累，一次性斩去三尸。正是难以想象。”
<br/>　　“师兄，他是如何做到的？一次连斩三尸？难道他已经成圣了？”准提问道。
<br/>　　“这倒没有，成圣的话，天上会有无上金光，紫气东来三万里，无上圣威压洪荒。但，如今那些异象都没有出现。他不还没有成圣。”
<br/>　　“这就好，这就好。”准提之前也猜测元始是否成圣，如今得之没有成圣，他松了一口气，缓缓道。
<br/>　　“好？这有什么好的，他就算没有成圣，恐怕也不远了。”接引神色凝重，丝毫没有放松，心情反而更加沉重了。
<br/>　　“哎！为何我西方没有如此俊杰啊。东方大地，人杰地灵，真是令人羡慕。”接引目露悲苦之色，感叹道。
<br/>　　……
<br/>　　太阳星上，帝俊与太一刚收服鲲鹏，回到太阳神宫，就感受到元始的突破的气势，顿时本来极好的心情也变得糟糕透顶。两人脸色阴沉，看向昆仑山，之后，默默叹道：
<br/>　　“哎，看来报仇之事，需要从长计议了。”
<br/>　　“元始，元始……大敌啊。”
<br/>　　不周山，女娲脸色阴沉，在她的旁边，伏羲一脸苦笑。
<br/>　　万寿山，火云洞，幽冥血海，北冥宫……都有各自不同的反应，每人的心情都变得沉重起来，修炼却也更加刻苦了。
<br/>　　显然，他们感受到了元始那股滔天气势，知道了两者间的差距，同为紫霄宫中客，为何会有如此差距？他们心中都是极为骄傲啊，不愿承认自己比别人低。
<br/>　　昆仑山，三清圣地之中，老子与通天一样惊讶，想不到自家大哥不同突破则矣，一旦突破，就是惊天动地。他们想找元始取点真经，但元始并没有出关，他们只好作罢。
<br/>　　巫族盘古殿之内，十二祖巫亦感受到了那股气势，为首一人，空间祖巫帝江面色难看道：“那又是谁在突破？后土妹子，你去过紫霄宫，可知那人是谁？”
<br/>　　在大殿的后面，有一个身穿黄衣裙的漂亮女子，正是土之祖巫后土，她听见帝江的话，道：“大哥，突破的方向在昆仑山，那里住着三清，先前已有两人突破，如今突破的恐怕是那三清之首元始。”
<br/>　　“三清？元始吗？就是他们自称是盘古正宗吗？”帝江道。
<br/>　　“盘古正宗？笑话，还要问我们答不答应！”火之祖巫祝融脾气暴躁，听到此话，冷笑道。
<br/>　　“三清为道祖弟子，不可大意。”时间之祖烛九阴看了众人一样，道：“若是他们与吾等为善还好，若是为恶，那我们也不必在意他们的身份。”
<br/>　　“嗯！”
<br/>　　“大哥，妖族最近闹腾得欢，想必是要搞什么大动作了，我们该如何应对？”水之祖巫共工突然问道。
<br/>　　说道妖族，众人脸色变得难堪，显然，他们在妖族手中吃过亏。帝江看了众人一眼，叫道：“诸位兄弟以为如何？”
<br/>　　“大哥，管他干甚，我等还怕他妖族吗，我们直接点兵，杀了他帝俊、太一小二。”脾气暴躁的祝融大声叫道。
<br/>　　“胡说八道！”共工立马反对，开口说道：“如今妖族势力不明，贸然前去，岂不是中了敌人的奸计。”
<br/>　　水火不融，祝融、共工二人一向不对付，有机会便会大吵一架。
<br/>　　“共工，你……”祝融顿时大怒，想要吵架。
<br/>　　此时，帝江大叫道：
<br/>　　“好了，大吵大闹，成何体统！妖族之事，等查清之后再议。”
<br/>　　共工二人无法，只得坐下。
