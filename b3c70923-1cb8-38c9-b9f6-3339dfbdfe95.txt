　　第111章
<br/>　　罗银山拧起眉头：“擂台校技，无非以武会友，你这莽汉为何出口伤人。”
<br/>　　“不就是打么，废什么话，今日三爷爷就当老师，教教你什么是功夫。”
<br/>　　程大雷一捂脸，若真有本事说这话也就罢了，可张三爷那两下子，上去后不就是挨揍么。
<br/>　　罗银山气得咬牙：“好莽汉，愿你的本事和口气一样大，那边有兵器，你是要比拳脚还是比兵器？”
<br/>　　擂台边摆着一排兵器架，上面刀枪剑戟，斧钺钩叉皆有。张肥走过去，从兵器架上抽出一杆长矛。
<br/>　　罗银山挥挥手，吩咐手下将他的鉄鞭抬过来。
<br/>　　“黑脸汉，刀剑无眼，你给我仔细了！”
<br/>　　“来呀！”
<br/>　　张三爷爆喝一声，众人耳边嗡嗡作响，再回过神来，三爷已抱着铁矛冲了过去。呼嘞嘞宛若一匹黑马，矛尖向着罗银山喉咙扎去。
<br/>　　“捅天！”
<br/>　　罗银山心中一惊，这一矛来得好急，来势汹汹，自己一时竟无招架的办法。
<br/>　　他接不得，只能退，身子一矮，长矛从他头顶扎过去。
<br/>　　罗银山惊出一身冷汗，万万不可小窥这黑脸汉子。
<br/>　　“扎地。”
<br/>　　矛在头顶狠狠砸来，罗银山就势在地上一棍，虽然险险避过，却吓得亡魂皆冒。
<br/>　　“赶猪。”
<br/>　　长矛横甩，逼的罗银山节节败退，罗银山内心有些崩溃，前两招听上去正常又威风，可这第三招听上去就觉得……这是把我当猪赶么。
<br/>　　程大雷此刻来了精神，张三爷了得啊，竟能将罗银山逼到这种田地。刚才他已查看过，这罗银山的等级是顶级。此刻再看张肥的信息，惊奇的发现他的等级已升到优秀。
<br/>　　果不愧是传说潜力的人，自己将『三手矛』给他才多长时间，升级竟然这么快。
<br/>　　擂台上，罗银山已站稳脚跟，他使一根铁鞭，所谓一寸长一寸强，一寸短一寸险，瞅准机会逼到三爷三步之内，令他手中的长矛施展不开。
<br/>　　三招过后，张三爷已露疲相。程大雷叹口气，以优秀对顶级，虽然能占一时便宜，却难以取胜。
<br/>　　罗银山却是越打越猛，手中铁鞭使得威风赫赫。原来这黑脸汉也就是头三招猛，之后一泻千里。哼，无非是一个没有本事的妄人，今日倒要给你点颜色看看，让你知道什么叫天高地厚。
<br/>　　脚下微动，罗银山已用灵活的步法闪到张肥背后，抄起铁鞭向他后脑砸去。
<br/>　　“竖子敢尔！”
<br/>　　一声爆喝，却见一白袍后生窜上擂台，一枪便将罗银山格开。
<br/>　　此人自然就是赵子龙。
<br/>　　他刚亮相就换得台下一片齐声好，有些少妇小姐还不停冲他抛媚眼。
<br/>　　啧！
<br/>　　果然颜值就是正义，赵子龙生得可是真俊，白袍银枪标致得令人发指。如赵子龙这种颜值，是可以让姑娘以身相许的，而像张肥，就只能换得来世当牛做马了。
<br/>　　“三哥且在一旁观战，此人交给我来。”
<br/>　　张三爷惊得一身冷汗，多亏赵子龙及时出现，敢才若是一鞭打实，自己一条命也交待在这里了。
<br/>　　罗银山获胜一场，顿时有些趾高气扬，冲赵子龙道：“哪里来的娃娃，和那黑脸汉子是一样货色。爷爷看你年纪小，不想取你性命，快快滚蛋吧。”
<br/>　　“子龙，莫要饶他，给他点颜色看看。”张肥吃了败仗，在台下大叫。
<br/>　　赵子龙剑眉一挑，长枪探出，他身穿白衣，好若一条出海锦龙。
<br/>　　“好！”
<br/>　　台下齐刷刷的叫好声，大多是大姑娘小媳妇口中喊出来的。罗银山吓了一跳，这是搞什么，他们都疯了。
<br/>　　程大雷也宛若看到前生某些偶像的粉丝，她们……怎么就亢奋了。
<br/>　　在无数啦啦队的助威下，赵子龙如有神助，铁枪在手中使出花来。而罗银山却是士气大丧，一连吃瘪，二十回合之内被赵子龙一招翻背青龙挑翻。
<br/>　　好！
<br/>　　罗银山没想过他会败，而失败又来得如此快，等他倒在地上时，还没明白是怎么会事。等他回过神来，只看一道寒芒向自己扎来。
<br/>　　“不……不要！”
<br/>　　擦！
<br/>　　枪尖捅入罗银山小臂，换来一声凄厉惨叫，他已铁爪功扬名，江湖赫赫扬名铁爪鹰，这次小臂被打断，就算再能接好，功夫也大打折扣。
<br/>　　躺在地上，手捂着胳膊，以恶毒的眼神盯着赵子龙。
<br/>　　好！
<br/>　　台下却如煮沸的白粥般沸腾起来，年当芳龄的女子纷纷解下身上的香囊往台上丢，瞬间赵子龙就沐浴在一片香囊雨中。
<br/>　　传说有美男子潘安，在街上走一趟，姑娘丢过来的鲜花和瓜果都能装满一车。看着这一幕，程大雷暗暗感慨：潘安再世，也就这样啦。
<br/>　　话说自己和赵子龙的颜值也没有什么差距，总的来说不相上下吧，怎么自己就没遇到过这种情形。
<br/>　　啧啧，看来扬州城的审美观也有问题。
<br/>　　赵子龙挑起擂台上的二百两赏金跳下台来，把银子交到程大雷手中。
<br/>　　却听得周围乱糟糟响，那些大姑娘小媳妇都向这边挤来，脸上的表情像是要将赵子龙生吞活剥相仿。
<br/>　　“诸位，冷静冷静，他还是个孩子，你们不要这样……”
<br/>　　赵子龙今年不过才十五岁，按照程大雷的观念就是个孩子。
<br/>　　“诸位，我是他的经纪人，你们有什么要求可以告诉我，他已经累了，需要休息。”
<br/>　　没人搭理程大雷，这些女人就像疯了般往这边闯，这幅景象却是把程大雷几人吓了一跳。
<br/>　　他抓起手中的银子包袱，挥手抖开，纷纷碎银如雨般落下。
<br/>　　“爷今天高兴，赏了。”
<br/>　　哗啦啦人群围过来，有些是抢银子的，有些是奔着赵子龙来的，乱哄哄挤成一片。如此却也把金龙镖局的人隔开。
<br/>　　“大当家，风紧扯呼。”
<br/>　　“扯呼扯呼。”
<br/>　　程大雷趁着这机会，赶紧开溜，不知跑出多远，看身后没人追来，才算是松了口气。
<br/>　　程大雷跑得气喘吁吁，拍了拍赵子龙的肩膀：“没想到啊，你有成为偶像的潜质。”
<br/>　　“什么是偶像？”赵子龙一脸懵懂。
<br/>　　“呃……”程大雷指指自己：“像我这个样的，就是偶像。”
<br/>
