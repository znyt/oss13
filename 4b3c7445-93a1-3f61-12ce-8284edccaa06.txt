　　“不！这不可能！”
<br/>　　无涯老人原本还想，把叶尘当做自己突破的垫脚石，怎么也没有想到，居然会是这个结果！
<br/>　　自己竟然败了！
<br/>　　而且败的如此迅速，如此彻底！
<br/>　　一招！
<br/>　　仅仅只用一招！
<br/>　　“没有什么是不可能的！死吧！”
<br/>　　叶尘才没有功夫和他废话，直接大喝一声，再次一剑挥出。
<br/>　　轰隆隆！
<br/>　　这一剑，比方才那一剑，威力还要强大几分，连空间都被生生撕裂开来，在空中留下一道狭长的虚空裂缝！
<br/>　　“无涯前辈！”
<br/>　　周围的仙界众强者，看到这一幕，顿时纷纷惊呼起来。
<br/>　　眼看这一剑若是劈中，无涯老人就算不死也得脱层皮。
<br/>　　然而在关键时刻，无涯老人身上陡然发出一道妖异的灰色光芒，
<br/>　　“乾坤挪移！”
<br/>　　下一刻，
<br/>　　无涯老人的身形，竟然犹如瞬移般，挪移到数十米外，险而又险的躲过了叶尘这一剑。
<br/>　　“嗯？”
<br/>　　叶尘眉头微微一凝，
<br/>　　“果然是地级功法，而且是空间类功法，最擅长躲闪！”
<br/>　　这道念头在脑海中闪过的同时，
<br/>　　唰唰唰！
<br/>　　叶尘抬手就是三剑，没有丝毫停歇。
<br/>　　然而无涯老人，再次故技重施，竟然全部一一躲过！
<br/>　　“哈哈哈哈！叶狂仙，你想不到吧？老夫这门功法，乃是我逍遥宗无上秘术，纵然你如今实力远超过我又如何？老夫只要和你耗着，等你的秘术时间一到，便是你的死期了！哈哈哈！”
<br/>　　无涯老人见叶尘数次攻击落空，顿时信心大涨，忍不住得意的哈哈大笑起来。
<br/>　　叶尘猛然站定，沉思了片刻之后，忽然嘴角微微一扬，脸上忽然露出一抹戏谑，
<br/>　　“老东西！倘若你参悟了空间法则，我今日还真未必能杀得了你！可惜啊，废物终究就是废物，就算是修炼了地级功法，却也一样是废物！”
<br/>　　说完这话，叶尘脚下猛然一顿，
<br/>　　唰！
<br/>　　下一刻，叶尘已经穿梭空间，来到无涯老人的跟前。
<br/>　　不过后者早就有所防备，在叶尘冲过来的同时，已经再次施展乾坤挪移，逃离了原地。
<br/>　　“桀桀桀！叶狂仙，不用白费力气了！你是打不到我的！”
<br/>　　无涯老人得意的狂笑着，似乎已经立于不败之地。
<br/>　　“是吗？”
<br/>　　叶尘却嘴角微微一扬，依旧一剑斩出，
<br/>　　“十、方、无、敌！”
<br/>　　唰刷刷！
<br/>　　这一剑落下，顿时幻化成成千上万道剑影，以叶尘为中心，向四面八方斩去。
<br/>　　“冥顽不灵！老夫都说了，你是打不中……什么！”
<br/>　　无涯老人充满得意的话语，才刚刚说到一半，顿时仿佛看到了什么不可思议的事情，整个人脸色瞬间煞白，不由得惊呼起来，
<br/>　　叶尘这一招十方无敌使出，以他为中心，方圆数千米之内，每一寸空间竟然都在他的剑气笼罩之下，令无涯老人竟然避无可避，躲无可躲！
<br/>　　“不！这不可能！”
<br/>　　无涯老人惊怒的大吼了一声，只好召出一件灵宝，奋力抵挡了上去。
<br/>　　这是一尊金色小钟，陡然变大数十倍，将无涯老人罩在其中。
<br/>　　然而，
<br/>　　伴随着“轰隆！”一声巨响，那金色大钟瞬间粉碎，
<br/>　　“啊！！”
<br/>　　一声惨叫过后，无涯老人重重坠落在地上，一连狂喷了几口鲜血。
<br/>　　“不！我怎么会输！不可能！！”
<br/>　　或许是因为无法接受失败的事实，无涯老人犹如癫狂般怒吼了一声。
<br/>　　而就在这时，无涯老人原本有些萎靡的气息，竟陡然恢复了几分，紧跟着节节攀升，超过了之前全身之时！
<br/>　　“嗯？”
<br/>　　看到这一幕，叶尘顿时眉头一皱，
<br/>　　“要突破了？”
<br/>　　原来，无涯老人在羞怒欲绝之下，竟然终于踏出了那最后半步，成功迈入了化神境！
<br/>　　“不行！化神境乃是一道分水岭，前后实力差距巨大，一旦让这老头境界稳定下来，而我的秘术效果消失，那我必死无疑！”
<br/>　　一念及此，叶尘当即身体一晃，抡起手中紫琼剑，朝无涯老人杀了过去。
<br/>　　“快！合击！保护无涯前辈！”
<br/>　　与此同时，周围一直在为无涯老人压阵的仙界众强者，也都意识到了叶尘的目的，当即联合在一起出手了。
<br/>　　数十名元婴境强者的联手一击，简直有毁天灭地之功，浩瀚无比，犹如一道天堑，挡在了叶尘面前！
<br/>　　“都给我，滚开！”
<br/>　　叶尘暴喝一声，一剑横扫而出！
<br/>　　嘭嘭嘭！
<br/>　　“啊啊啊！”
<br/>　　瞬间便有数名仙界高手，死于叶尘剑下。
<br/>　　唰唰唰！
<br/>　　叶尘杀人人群之中，紧跟着又是一通乱砍，所过之处，血雾弥漫，不一会的功夫，数十名仙界强者，便已经被他斩杀殆尽。
<br/>　　然而，经过这片刻的耽搁，
<br/>　　那无涯老人已然完成了突破，气息比之前巅峰之时，还要强大数倍！
<br/>　　“叶狂仙！老夫要将你抽筋扒皮，挫骨扬灰！”
<br/>　　无涯老人看到仙界众人，尽数丧命于叶尘之手，顿时气得双眼欲裂，一道青色剑芒冲天而起，朝叶尘狠狠劈落下来！
<br/>　　突破之后的无涯老人，虽然还不算正式踏入化神境，可是实力比之前也强大太多，而且他手中的青色巨剑，俨然是一件极品圣器，竟然和叶尘的紫琼剑不相上下！
<br/>　　轰隆！
<br/>　　一紫一青两道剑芒，在空中相撞在一处，连方圆千米内的空间，都发生了巨大扭曲，下方更是出现了一个巨大的深坑！
<br/>　　原本以险峻著称的天下名山，竟然生生变成了一处深渊。
<br/>　　轰隆隆！
<br/>　　待那扭曲的空间恢复过来，只见天空之上，两道人影相对而立，谁也未曾倒下，竟是旗鼓相当。
<br/>　　“哈哈哈！”
<br/>　　无涯老人哈哈一笑，
<br/>　　“叶狂仙，老夫如今已经踏入化神之境，你，不是老夫的对手！受死吧！”
<br/>　　无涯老人自信满满，再次抡起手中的青色巨剑，朝叶尘扑杀而来！
<br/>　　而这时，叶尘竟然缓缓闭上了眼睛，口中低声喃喃，
<br/>　　“没想到竟然能把我逼到如此地步，看来也只能拿命来拼了！”
<br/>　　“逆生百转，第六转！”
<br/>　　本来施展出第五转，就已经是叶尘此时的极限，只是他怎么也没有想到，这无涯老人竟然临阵突破，无奈之下，只好强行拼上一把！
<br/>　　“噗！”
<br/>　　才运行到一半，叶尘就忍不住狂喷了一口鲜血！
