　　太阳星上，两位女神呆呆地看着眼前的无尽火焰，心中却心思各异。
<br/>　　羲和心中想起平时见帝俊时的画面，那一幕幕谈笑风生的画面在她心头掠过，帝俊那威严的面目渐渐在她心中清晰起来。
<br/>　　“或许，我也该换种方式生活了，那样也不错。”
<br/>　　最终羲和转过身，对着一旁的女娲点了点头，同意了女娲的要求，即同意嫁给帝俊。
<br/>　　见到羲和同意后，女娲心中也松了一口气，毕竟之前她是答应过他哥哥的，一定会将此事办成，若是完成不了，虽然不会怎样，但她心里却会过意不去。女娲朝着羲和微微一笑，顿时万火失色，她笑道：
<br/>　　“那就先恭喜妹妹喜结良缘了！”
<br/>　　羲和淡淡点了点头，面色平淡，看不出丝毫心中所想，答应此桩婚事，她也只是想换种方式而活罢了，没有什么值得惊喜的。
<br/>　　……
<br/>　　太阴星，还是一如既往的冷清，入目所见，寸草不生，一片荒凉孤寂，冷冷清清，这就是太阴星的主题。
<br/>　　这天，寂静的太阴星上迎来了一位客人，她正是女娲。
<br/>　　“望舒道友，还请出来一见。”
<br/>　　女娲来到太阴星的月桂秘境之前，出声喊道。
<br/>　　“哗啦！”
<br/>　　不久，却见两位绝世亮丽的女神从秘境中走了出来，一个身穿白衣，一个一身银装，正是太阴之主望舒与她的徒弟嫦曦。
<br/>　　望舒见来人是女娲，不由得眼神一缩，好似想到了什么，不过，她脸色并没有改变，望着女娲，淡淡出声问道：
<br/>　　“见过女娲圣人，不知道友来我太阴星有何贵干？”
<br/>　　“道友，我们自上一次昆仑山论道一别，便再没有见过，我对道友可是想念的很啊，此次却是要来恭喜道友了！”
<br/>　　女娲毫不在意望舒的态度，她对着望舒拱手道。
<br/>　　“哦？不知喜从何来？”望舒略微诧异道。
<br/>　　“首先，要恭喜道友道行大进，一身修士已经准圣圆满了，真是真人不露相，洪荒之中，比道友强的，恐怕没有几个了。其次，我要恭喜道友这徒儿。”
<br/>　　女娲也有些惊异，她想不到这位太阴之主修为如此强横，竟然不知不觉地到了准圣圆满，恐怕就是帝俊都没有她强，不愧是太古时候的强者啊。若是自己没有成圣，洪荒女性之中，恐怕就是她最强了。
<br/>　　“哦？这关我徒儿何事？”
<br/>　　望舒想起上次元始的提醒，要小心妖族，不由得眼神一缩，难道是他们想对自己出手了？
<br/>　　“天庭的帝俊看上道友这徒儿，想要娶她为妻，特意请我前来做媒，不知道友，意下如何？”
<br/>　　“这……这恐怕不行，我这徒儿，在太阴星待惯了，去别处她待不习惯。”望舒毫不犹豫地便拒绝了。她越发的觉得，这是妖族的阴谋。
<br/>　　“这……道友你不多考虑一下？”
<br/>　　“不行就是不行。”望舒坚决不答应。
<br/>　　“哦？帝俊可有何不对之处，道友为何不答应？”女娲皱起眉头，她也没想到会是这种情况，顿时觉得麻烦至极，但碍于圣人的身份，她只好开口问道。
<br/>　　“他没有什么不好的，就是太霸道了，我这徒儿性子温和，与他不合适。更何况，妖族与巫族正在对抗，我这徒儿还是不要参与进去为好。”望舒淡淡道，毫不在意女娲的圣人身份。
<br/>　　“这……道友，你不问问你徒儿的意见？”女娲有些头疼，却也不好以自己圣人的身份去逼迫她，只好退而求次，让她徒儿做决定。
<br/>　　“老师的决定，便是我的决定。”这时，一旁的嫦羲开口道。
<br/>　　“这……你们不再好好考虑一番？”女娲不由得脸色变得阴沉下来。
<br/>　　“不用再考虑了，我徒儿是不会嫁给帝俊的，慢走，不送！”望舒有些烦了，直接开口赶人。
<br/>　　“哼！既然如此，那便告辞！”
<br/>　　女娲也不想在这继续热脸贴别人的冷屁股，冷哼一声，便直接离开了。
<br/>　　望着女娲离开的背影，望舒眼神不由得变得凝重起来了，她知道自己此次拒绝了帝俊，那么，下次就要兵戎相见了。
<br/>　　一时之间，两人有些沉默了。良久，嫦羲开口说道：
<br/>　　“老师，我想去洪荒走走。”
<br/>　　望舒转过身，静静地看着嫦羲，她知道，若是嫦羲走出太阴星，那么自己就保护不了她了。良久，她问道：
<br/>　　“你是认真的？”
<br/>　　“嗯，老师，我一直都在太阴星上，不曾到洪荒之中走过，如今想去走走。”嫦羲认真的点了点头。
<br/>　　“哎，罢了，既然你想出去，那便出去吧，若是什么时候想家了，那再回来。”
<br/>　　望舒看着嫦羲认真的表情，她知道一旦嫦羲做出决定，便不会改变，不由得有些头疼。不过，最终她还是同意了。现如今这种情况，她出去走走也好，正好可以躲避妖族。
<br/>　　“嗯。”嫦羲应了一声，便向着太阴星外飞去。
<br/>　　望着嫦羲，望舒一阵沉默，眼神深邃，不知道想些什么。
<br/>　　“哎——”
<br/>　　虚空中传来一阵幽深的叹息，久久不散。
<br/>　　……
<br/>　　天庭之中。
<br/>　　“什么？！那嫦羲没有答应？大哥，我这就带人前去太阴星一趟。”
<br/>　　太一听说此事不成，顿时便怒了，竟然看不起我大哥，这还如何得了？不行，必须给她点颜色看看。
<br/>　　“慢！二弟，嫦羲虽然没有答应，但羲和却答应了，如今这种情况，我们天庭不宜生事端，这件事，等天婚过去，在做商量。”帝俊叫住了大怒的太一，说道。
<br/>　　“可是，大哥——”
<br/>　　“没什么可是的，听我的。”帝俊大声道。
<br/>　　“好吧，大哥，那就等大哥你结婚后，再做此事。我想，太阴星是时候归入我妖族了。”太一良久才作罢。
<br/>　　“话说，二弟，你有没有喜欢的女子，若是有，那大哥帮你说亲去。”帝俊换了个话题，对着太一道。
<br/>　　“大哥，你又不是不知道我的性子，我一直在努力修炼，哪有什么喜欢的女子？”太一说道。
<br/>　　“那么，二弟你也应该去找一个啊。”帝俊调笑道。
<br/>　　“大哥，我对着些男男女女之事不感兴趣，此时以后莫要再提了。”太一大声道。
<br/>　　“好吧，那我们还是来商量一下，此次大婚，准备邀请哪些人来参加观礼吧。”帝俊不再笑话他，脸色不由得变得严肃起来，说道。
<br/>　　“嗯。”
