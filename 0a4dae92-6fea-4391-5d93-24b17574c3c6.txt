
<br/>　　<br/>　　    这位常公子也没想到，在潘家园这条街上，前后不过半个小时左右，他一共就去了两家店，贺三刀的宝瑞福和沈掌柜的隆庆祥，全都碰上了王惊蛰，这两家可都是京城老字号了，而且这两店背后的沈掌柜和贺三刀居然还全都同他认识，这就挺耐人寻味的了，常公子先是冲着王惊蛰点了下头，然后要有兴致的打量着对方。<br/>　　<br/>　　    相貌普通，身材中等，木有气质，总结一下，就是没啥出奇的地方。<br/>　　<br/>　　    王惊蛰见对方点头示意，就笑着回应了一下，也没太主动，虽然他从这青年的脸上也看出来对方是非富即贵的那种人。<br/>　　<br/>　　    沈长河看到王惊蛰，短暂的顿了顿，随即点头说道：“都说隔代像，你和王仙芝没什么相似的地方，跟风饶倒是挺像的”<br/>　　<br/>　　    “见过沈大掌柜的，我姐倒是跟我爸挺像的，我可能是长跑偏了……”王惊蛰以晚辈见礼的方式，弯下了腰，然后抬头说道：“我来您这，是想再做两身褂子，之前的那一身已经被我糟蹋坏了”<br/>　　<br/>　　    沈长河摆手说道：“那都不是问题，我先给你量量身”<br/>　　<br/>　　    “好叻！”<br/>　　<br/>　　    隆庆祥的衣服都是手工订制款的，需要裁缝亲自量身才行，这沈掌柜的为人量身就挺有意思了，他不用尺子只用眼睛，手里拿着根笔和本子，大概扫了几眼王惊蛰的袖长和腰围，然后就在本上“唰，唰”的写了几笔，就把王惊蛰的参数给一点不差的看完了。<br/>　　<br/>　　    量完身后，王惊蛰说了几句客套话，然后就准备告辞了，没想到沈掌柜的叫住了他，说道：“正好赶巧了，我们东家前段日子还问过我，说你们王家人有没有来定褂子，我说那都是好多年前的事了，东家叹了口气说想见见你们，可惜了你们王家人都行踪飘忽，找也找不到，这不今天还真是老天长了眼，东家才说完没多久，您就过来了”<br/>　　<br/>　　    王惊蛰笑了笑，摇头说道：“沈大掌柜的想必您也了解，我家可不做上门的生意”<br/>　　<br/>　　    常公子在两人身后，轻“咦”了一声，愈加对王惊蛰有点好奇了，让隆庆祥这四百多年老字号的幕后东家主动开口求人的事可不多，毕竟人家做衣服的关系户，在京城可都是盘根错杂的，更何况你来上门做衣服，是大掌柜亲自出的手，怎么着，找你点事都没有这面子？<br/>　　<br/>　　    沈掌柜的说道：“你家的规矩我们自然懂，但东家可没说找你是做生意啊，吃个便饭总可以吧？要知道，风饶和我还有东家可都是老相识了，你怎么说也是个晚辈吧？”<br/>　　<br/>　　    王惊蛰寻思了下，无奈的摊着手说道：“您把话都说到这份上了，我哪里还能说不行？”<br/>　　<br/>　　    “好叻，您稍等，我这就给东家回个话去……”<br/>　　<br/>　　    沈掌柜的回后面去打电话，王惊蛰背着手闲逛，姓常的青年就走了过来，有话没话的起了个头：“朋友，刚才在贺三刀的茶楼那里也见过，挺巧”<br/>　　<br/>　　    “啊”王惊蛰头也没回的应了一声。<br/>　　<br/>　　    这青年明显是个情商挺高，也很懂语言艺术的人，他跟王惊蛰聊了几句，都是从贺三刀还有沈掌柜的身上入手，对于他本身则是根本啥都没有打听，几句话之后，王惊蛰已经开始不是一个字一个字的往出蹦着回话了，这个开头就表明，他俩已经过了点头之交的层次了，慢慢的熟络了起来。<br/>　　<br/>　　    片刻后，沈掌柜的回来了，跟王惊蛰说道：“东家出门了，要过段日子才会回京，你这段时间……”<br/>　　<br/>　　    王惊蛰说道：“暂时还不走，过几天我再过来一趟就是了，正好我还有点事情在京城要处理”<br/>　　<br/>　　    “哎，那敢情是好了”沈长河笑了笑，把王惊蛰送了出去。<br/>　　<br/>　　    他走了后，那姓常的青年，就兴趣挺大的打听道：“沈掌柜的，这人是什么来头，听着不像是京城口音呢”<br/>　　<br/>　　    “他家几代都是隆庆祥的旧识，祖上人穿的褂子都是我们做的，关系么像是客户，但也相交莫逆，因为隆庆祥最早先的东家有两次都被他祖上的人救过命，所以从那以后这小哥家中人再来隆庆祥做衣裳，就也分文不收了……”沈掌柜的解释了一句，就没再往下说了。<br/>　　<br/>　　    常公子笑道：“怎么，我打听的多了，您这就收口了，对我也瞒着啊？”<br/>　　<br/>　　    沈掌柜的叹了口气，摇头说说道：“小常啊，他这人的身份也不是什么秘密，没啥说不得的，但我却知道你要是明白过来他是谁的话，搞不好就要麻烦上了，所以呢为了避嫌，别显得咱嘴没有把门的，我真就不太想告诉你了”<br/>　　<br/>　　    常公子摇头笑道：“您真是想多了，有个事吧你可能还不知掉，之前我在贺三刀的宝瑞福就见过他了，从您这打听不到的话，我还能去贺三刀那里问问，你知道的，他欠过我不小的人情，要不是我在中间过话，他这些年因为倒卖的那些古董，不知道多少次会被人给抓进去了，你说我若要挑明了问他，他会不言语？”<br/>　　<br/>　　    沈长河顿时哦了一声，寻思了下说道：“既然有这么一茬儿，那我跟你说说也没什么了，他姓王，祖籍朝歌王村，他家祖祖辈辈就是街边巷尾下传说的赊刀人，他是这一代的……”<br/>　　<br/>　　    常公子听闻顿时一愣，半天没回过神来，良久之后才揉了揉脑袋，惊诧的说道：“那我知道了，呵呵，有意思！”<br/>　　<br/>　　    王惊蛰从隆庆祥出来后，在外面吃了饭菜就回到了贺三刀借他的那栋房子里准备休息了，他还等要等几天，等着贺三刀的儿子出状况。<br/>　　<br/>　　    贺三刀有个儿子，今年二十六岁，叫贺然，大学毕业以后呢没接家里的生意，反倒是进了个国企，在中铁某局任工程师，这个贺然为人胸无大志也很老实，从小就对工程一类的事感兴趣，反倒是家里的那些古董他连看都不会看一眼。<br/>　　<br/>　　    这个时候的贺然，在大别山一带干活，中铁某局在那里有一段路正要修。
