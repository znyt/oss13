        “如果，我不同意，你会怎样？”傅夜沉反问。
<br/>
<br/>        童筱颖垂着眸子，面无表情地回答道：“离开你。”
<br/>
<br/>        当她亲眼看到何明旭和林若琴的死，就明白，傅夜沉所要做的事，是在拿命在做。
<br/>
<br/>        八爷和十爷都不是善类，他们对持枪杀人这种事情习以为常。
<br/>
<br/>        说不定哪天，他们就会拿枪指着他傅夜沉。
<br/>
<br/>        得知自己爱的人死了，那种痛是什么滋味，她已经尝过了，所以不想再尝第二次。
<br/>
<br/>        她的心很小，也很简单。
<br/>
<br/>        她装不下那些事关他人性命之忧的大事，只想着自己的小窝是否能让她过得舒坦。
<br/>
<br/>        或许，这就是她的自私。
<br/>
<br/>        “如果不放了季雅珊，你就要跟我离婚。如果我不答应你离开我的组织，你就要离开我。”傅夜沉苦笑，“筱颖，你什么时候学会威胁我了？”
<br/>
<br/>        “你觉得我在威胁你，那就当是我在威胁你好了。”童筱颖抬眸看着傅夜沉。
<br/>
<br/>        “如果三年前，你嫁的人是我，或许，我会毫不犹豫地离开组织。但是，三年前，你没有选择我。这三年来，我一直在做我想要做的事情，我不能半途而废。你理解也好，不理解也好，于我来说，已经不重要了。季雅珊，她身上有上头想知道的秘密，所以，我不会帮你去救她。毕竟，我曾经不顾自己的性命已经救过她一命，我不欠她什么。她落得如今这个下场，完全是她咎由自取。其次，童筱颖，忘了告诉你。我是一个占有欲很强的男人，你是我的第一个女人，这一辈子，就永远都是我的女人。离开我的事，你想都别想。除非我死！另外，对于十爷和八爷来说，你和季雅珊都被我们这边的人抓了。所以，你现在，除了待在我的私宅里，哪里都不能去，同时，也别再想着要离开我。不可能！”傅夜沉疾言厉色地把话说完。
<br/>
<br/>        童筱颖看着傅夜沉的这张俊脸，突然间觉得他变得好陌生。
<br/>
<br/>        他的气场很强大，压迫得她有些喘不过气来。
<br/>
<br/>        童筱颖怔愣了半晌，本能地掀开被子，想要从傅夜沉的面前逃离。
<br/>
<br/>        或许是她的这个动作，让他感到心慌，他的大手，猝不及防地握住她的手腕，将她拉回床上。
<br/>
<br/>        “你想去哪？”
<br/>
<br/>        “我……”童筱颖有些不知所措地凝视着傅夜沉的眼睛，顿了顿后，才有些木讷地回答，“我今晚想睡隔壁房间……想一个人静一静。”
<br/>
<br/>        “你在逃避我？”傅夜沉敏感地皱了皱眉。
<br/>
<br/>        童筱颖矢口否认：“没有，我只是需要冷静一下。”
<br/>
<br/>        “今晚陪我。”他大手一览，扣住她的腰肢，将她软软的身子，贴入了自己怀中。
<br/>
<br/>        童筱颖空余的那只手，下意识抵在他的胸膛上，试图跟他保持着一段距离。
<br/>
<br/>        但傅夜沉身上那炙热的气息，却将她全身笼罩。
<br/>
<br/>        “我好久没这样抱过你了。”他的声音透着磁性，褪去了刚刚那份严肃，很温柔。
<br/>
<br/>        然而，他的这份温柔，她感受不到。
<br/>
<br/>        童筱颖不再抵抗，任由傅夜沉抱在怀中，两眼黯淡无光，脸上也没有任何表情地说：“如果，你想解决生理需求，今晚，你可以去找落以柔，我想，她一定十分愿意。”
<br/>
<br/>        她的这话一出，傅夜沉倏然瞪大了黑眸，心脏就像一根被拧紧的绳索，一边痛着，一边像是要被崩裂。
<br/>
<br/>        他推开了她，怒斥：“童筱颖，你到底有没有把我当你的丈夫？”
<br/>
<br/>        “你连一个平静的家，都给不了我，又何谈是我丈夫？”童筱颖将目光挪向一旁，不敢直视傅夜沉的眼睛。
<br/>
<br/>        她是故意的，故意说这些难听的话气他，只希望他能妥协，去把季雅珊放了。
<br/>
<br/>        “好，很好！既然你不把我当丈夫，今晚我也没必要把你当妻子。”傅夜沉突然冷冷地呵斥。
<br/>
<br/>        童筱颖心悸地回眸之际，只觉自己的双臂上传来一阵吃痛，思绪上还未有任何反应，就被一道力度推倒在大床上。
<br/>
<br/>        “童筱颖，我告诉你！我不是傅昼景！我没法做到心里爱着一个女人，却去上另一个女人的身体！”
<br/>
<br/>        他铿锵有力的声音里满是怒火，撑在她肩膀旁的手背，也因为他一身的邪火而青筋凸起。
<br/>
<br/>        童筱颖有些胆战心惊地看着覆在自己身上的傅夜沉，他英俊的脸上有细密的汗珠渗出，黑白分明的眼睛里更是布满了血丝。
<br/>
<br/>        他在压抑着自己的愤怒，以及体内沸腾的热血。
<br/>
<br/>        所以，迟迟没有进一步的行为。
<br/>
<br/>        上一次，他强行上了她，他知道，给她留下了心理阴影，所以，这一次，他不会再乱来。
