        “我觉得，你是真的疯了！”连华生恼火地踢翻了一旁的椅子。
<br/>
<br/>        “不要告诉筱颖，关于我的事情。如果，我回不来了。你就跟筱颖说，说我移情别恋爱上了落以柔，想和落以柔在一起，不想再见她。我想，筱颖会相信这种谎言。”
<br/>
<br/>        “好！你非得要去救季雅珊，那你最好平安归来。不然，我会让童筱颖去给你陪葬！不信的话，你就试试。”连华生握紧了双拳。
<br/>
<br/>        傅夜沉黯然神伤地抱了抱连华生：“如果，我真的回不来。替我孝顺我爷爷，还有我母亲。至于筱颖，你帮我把她偷偷地送回澳洲去，不要让上头的人知道她的去向。她在澳洲生活了三年她想要过的日子，以后可以继续在那里过她想要过的平静生活。”
<br/>
<br/>        “你不是一直想跟童筱颖在一起吗？你这么做，就是把自己和童筱颖彻底推向两条路！你救了季雅珊，你会踏上一条逃避追捕之路。那么她童筱颖呢？就会被组织的人一直监视着。你们俩今后，想再重新在一起，很难！”连华生义愤填膺地怒吼，他仍旧试图劝阻傅夜沉。
<br/>
<br/>        可是，傅夜沉现在比谁都看得清楚。
<br/>
<br/>        他不去救季雅珊，季雅珊会死。
<br/>
<br/>        筱颖和十爷有联系，就算筱颖不去找十爷，十爷也一定会想方设法地找到筱颖。季雅珊若是死了，筱颖也迟早会知道真相，到时候和他的关系一定会陷入更僵硬的状态。
<br/>
<br/>        如果真到了那个时候，他每天要怎么去面对筱颖？
<br/>
<br/>        筱颖又会怎么看待他？
<br/>
<br/>        她到时候一定会怪他，说她当时求过他去救她的朋友，但是他见死不救，导致她朋友的死亡。
<br/>
<br/>        到那个时候，他和筱颖在一起，只会相互折磨。
<br/>
<br/>        人死了，心结永远没法解。
<br/>
<br/>        只要人还活着，就什么事情都好说。
<br/>
<br/>        “至少，到时候，我还有可以去思念她的理由。倘若，我不去做，我跟她相见不如不见。季雅珊若是死了，筱颖会怨恨我一辈子。”傅夜沉无奈地笑了笑。
<br/>
<br/>        “既然如此，我会在暗中帮你。”
<br/>
<br/>        连华生握紧的双拳终于放松了。
<br/>
<br/>        果然，他劝不动他。
<br/>
<br/>        “我去找落以柔，先送她离开私宅，筱颖就交给你了。”
<br/>
<br/>        傅夜沉放开了连华生，安慰性地拍了拍连华生的肩头，然后外面套了件风衣转身离开了密室。
<br/>
<br/>        他刚从书房里出来，抬眸间，便看到正向她迎面走过来的童筱颖。
<br/>
<br/>        她披散着长发，穿着单薄的睡裙，双手无力地垂在腿边，赤脚走在地上，脸色苍白。
<br/>
<br/>        傅夜沉剑眉微蹙，大步上前，将童筱颖的身子揽腰横抱入怀中。
<br/>
<br/>        童筱颖本能地抓住他的衣襟，垂着眸子，弱弱地说道：“大哥，我求求你，好不好？只要你让你上头那边的人放了雅珊！我就当你笼中的金丝雀！以后，大哥要我做什么，我就做什么。大哥，我求求你，好不好？”
<br/>
<br/>        傅夜沉没有吭声，将她抱回了房间，轻轻地放在了床上。
<br/>
<br/>        “大哥，我求你……”童筱颖不依不饶，声音有些哽咽起来，“我不想雅珊死……我不希望她死……”
<br/>
<br/>        “如果，我告诉你，我去救她，我会死。你还要我去吗？”傅夜沉剑眉微蹙，冷冷地质问。
<br/>
<br/>        童筱颖身心一滞，讷讷地开口回答：“不会的！你和他们是一边的，你们不是那种杀人不眨眼的集团。”
<br/>
<br/>        “你果然什么都不懂，什么都不了解。当然，我也不需要你去了解，也不需要你去懂。”傅夜沉深沉道，顿了顿后，他问了她一个可笑的问题，“筱颖，在你心里，你爱我多一点，还是爱小卫多一点？”
<br/>
<br/>        对，他问的是“小卫”，不是傅昼景。
<br/>
<br/>        如果，他没意会错的话，在筱颖的心里，龙之卫和傅昼景，已经被她完全区分成了两个人。
<br/>
<br/>        那个背叛了她，出轨季思妍的男人是傅昼景。
<br/>
<br/>        那个跟她有过美好的童年，叫着同一个女人为外婆，又在她最需要帮助的时候出现，陪她度过三年平静而温馨的美好时光，给过她让她憧憬的家的男人是龙之卫。
<br/>
<br/>        筱颖想要的，所向往的，只有那个叫“龙之卫”的男人给过她。
<br/>
<br/>        而傅夜沉的这个问题，让童筱颖一时间没有反应过来，甚至更没有答上他的话。
<br/>
<br/>        每个人心里都有一个被自己隐瞒了却没被自己发掘的秘密。
<br/>
<br/>        其实，那个时候，他早就意识到她心里上的这个问题了。
<br/>
<br/>        只是，他不想去面对。
<br/>
<br/>        “算了，当我什么也没问。”傅夜沉苦笑着站直了身子，并向她背过身去，不愿让她看到他眼里的寞落。
