　　说完跪在地上磕头起来。
<br/>　　三足金蟾不以为意，冷哼一声，慢吞吞地说道：“能被本王看中，也是你家的荣幸，本王还能亏待你家不成？再说这里的女子，谁不是别人家的女儿？”
<br/>　　邱文书知道说不通，趴在地上哭起来。
<br/>　　“就这么定了，沐浴斋戒，三日之后，我来接她。”五通神说完，袖子一扇，化作一道黑烟，从门缝中钻了出去。
<br/>　　那被他选中的姑娘，扑在父亲怀里，哇的一声哭起来，父女二人抱头痛哭。边上那几个负责这桩事情的村民面面相觑，上前将他们拉开，好言相劝，之后架住邱文书，将那姑娘送进卧房关了起来……
<br/>　　叶少阳看到这一幕，不忍心看下去，转身离去。
<br/>　　之前他要屠杀五通神，是为了之前那些被他糟蹋的女子，现在他更加坚定了这份信心，为的是不再有姑娘遭受他的折磨。
<br/>　　五通神，非杀不可！
<br/>　　叶少阳元神回到肉身里，跟美华说了这件事情。美华让他去自己的洞府，慢慢商量这件事。叶少阳想来在镇上也没什么事，就跟她走了。
<br/>　　两人一路进山，来到山间的一处水潭附近，美华表示自己的洞府就在下面。
<br/>　　美华从兜里摸出一枚珠子，给叶少阳。“这是避水珠，你含住它，在水下可以坚持一会，进了洞府就可以了。”
<br/>　　叶少阳身上原本也有避水珠，但是没有带过来，于是接过她给的珠子含在口中，美华看着他笑道：“随便给你东西就敢吃，你不怕我害你啊。”
<br/>　　叶少阳愣了一下，瞪眼说道：“什么鬼，这天上谁都可能害我，只有你们不会。”叶少阳说的是捉鬼联盟的成员。
<br/>　　“可是我并不是你那个时代的我。”
<br/>　　“那也一样，都是你。”叶少阳说完跳进水中，有避水珠的帮助，身边水边自动往四周划开，形成了一方空间，以叶少阳为核心，他走到哪里身边都是有空气的，这样身上也不会弄湿。
<br/>　　两人一起往下沉。
<br/>　　从上面看，这个水潭并不大，方圆只有几米，不过进入水中之后，叶少阳才知道这里有多广阔：这个水潭连接的是一条地下河，下面有急流，叶少阳应付不来，美华拉着他的手，一路游过去，潜入河底，立刻有两个看上去好像两个破衣服的家伙游过来，看到美华，立刻退到一边去。
<br/>　　美华带着叶少阳钻进了一个绿色的拳头大小的螺里。
<br/>　　一花一世界，从微观上来说，每一处都是一个空间，以美华的修为，不可能开辟出像山河社稷图那么大的空间，就算是阴阳镜那样的也做不到，不过在一个螺里开辟一个洞府还是可以做到的，很多水鬼水妖也都是这么做的。倒不是在水里不能生存，这么做其实是为了阻挡外敌。
<br/>　　洞府之中，都设有禁制，外人是进不去的，可以让主人在里面修炼的时候，一定程度上保证安全。
<br/>　　美华的洞府，完全是古典的风格：到处都是古色古香的木质家具，床是雕花的架子床，四面都盖着红色的帷幔，还有个红木的梳妆台，上面也摆着一些姑娘用的小玩意，看上去完全是个大家闺秀的闺房。
<br/>　　“怎么样，看上去还可以吧？”美华问。
<br/>　　“见过一次。”叶少阳环顾左右，说道，“我去过你洞府一次，跟这风格差不多。”
<br/>　　美华听了这话，反倒有点诧异，说道：“百年之后，我洞府难道没有变化？”
<br/>　　“基本上没有，连你穿衣服风格都没变。”叶少阳笑道，“你到哪里都是一身旗袍。”
<br/>　　美华有点不好意思，让叶少阳随便做。
<br/>　　过了一会，有个穿古装的丫鬟过来上茶，叶少阳看这妹子，长的挺清秀的，浑身都是妖气，倒不是说修为怎么高深，反而是修为太弱，刚成人形，不懂得隐藏自身的妖气。
<br/>　　真正成形的大妖，都可以尽量将自身的妖气隐藏起来，这样在人间行走的时候，不容易被法师看出来。
<br/>　　“一只蚌精，在这里修行，刚成精不久，被我收了的。”等少女走后，美华对叶少阳说道。
<br/>　　美华把茶端给叶少阳，“这是人间的茶，因为我喜欢，所以采集了一些，你尝尝。”
<br/>　　等叶少阳接过茶，美华眯着眼笑道：“挺漂亮的吧？”
<br/>　　“什么，啊，你说那姑娘啊，是啊还可以，妖精不都是挺漂亮的么。”反正妖精都是自己变化形象，当然不可能把自己变丑。不过这种模仿人类的相貌的，通常都缺少灵气，只有天生为妖的那些，容貌也是天生的，有时候很的是倾城倾国的容貌，例如小九……
<br/>　　想到小九，叶少阳心中又是一阵绞痛。
<br/>　　美华让他在床上坐下，笑着说道：“你想不想让她服侍？”
<br/>　　“什么？”叶少阳正在想有关小九的事，突然听见这么一句，当场就愣住了，问道：“你说什么？”
<br/>　　“我说，如果你喜欢，我让那个蚌精服侍你。”
<br/>　　叶少阳吓的差点跳起来，“你怎么会有这种想法？”
<br/>　　“我刚才看你一直看她，可不是喜欢她吗？”
<br/>　　“我晕啊，我只是感受到她身上妖气太重，忍不住看了几眼，你怎么……”叶少阳相当无语，问道，“话说你总是用丫鬟招待别人？”
<br/>　　美华瞪了他一眼说道：“怎么会！我这里可是只有你一个人类来过，看你是我主人，我才想到这个的，别人做梦去吧。”
<br/>　　叶少阳笑起来，“那什么，你误会了，等你一百年后认识我你就知道了，我不是你想的那种人。”
<br/>　　冲她晃了晃茶杯，“我喝茶就好了。”不过脑海里却也是忍不住想了一下刚那个蚌精的身材，的确挺不错……
<br/>　　美华坐在他身边，有些羞涩地问道：“有一个问题我一直在想，有些难以启齿……我是你的鬼仆，要不要服侍你？”
<br/>　　叶少阳正在喝茶，一口就喷了出来，表情很像微信里的“笑哭”，“我说大姐，你这是瞎想什么呢，你是我鬼仆，跟我姐妹一样，我怎么能对你下手？”
