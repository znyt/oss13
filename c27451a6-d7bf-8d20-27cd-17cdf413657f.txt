　　第331章
<br/>　　一道凉气从严笛天灵盖汇入，直抵脚心，严笛浑身上下瞬间凉透。
<br/>　　就看到程大雷坐在茶桌旁，慢条斯理的喝茶，在他身边还立着一人，严笛却不认得对方是谁。
<br/>　　“你……你怎么来了！”
<br/>　　严笛的声音因为紧张而有些结巴，琴川关城门一天十二时辰都有人把守，自己院落周围有三班卫兵巡逻。可程大雷仍旧出现在这里，某一刻，严笛怀疑自己见到的是不是鬼影。
<br/>　　“想来就来了，倒也算不上困难。”程大雷耸耸肩：“对我来说，很容易。”
<br/>　　严笛下意识往后退了一步，手搁在腰间，身后就是房门。
<br/>　　他是个心很细的人，知道现在卫兵还未经过自己院门，目前要做的，是如何稳住程大雷。
<br/>　　“这里是琴川关，只要我大喊一声，就有成千上万人要杀你，你……不怕么？”
<br/>　　程大雷撇撇嘴，道：“不要说废话了，这个距离，我动手比你说话快。你不信的话，可以试一试。但请注意，机会只有一次喔？”
<br/>　　严笛默然，他不敢冒这个险，于是压低声音。
<br/>　　“你来做什么？”
<br/>　　“看，这样聊天就顺畅很多么，交流才是解除误会的唯一办法。”程大雷道：“我要一些莫鸣米的资料，包括他每天做什么事，什么时候吃饭，吃什么饭，几点睡觉，几时起床……当然越详细越好。”
<br/>　　“你要这些做什么？”严笛有些困惑，他看不出来这流水账般的东西对程大雷来说有什么用。
<br/>　　“我自然有我的用处，至于用来做什么你就不用关心了。”程大雷道：“城门外有棵大杨树，你每天黄昏把资料放入树洞中，我会让人来取，有时候是我过来，有时候也可能是别人。”
<br/>　　“我是不可能背叛将军的。”严笛。
<br/>　　“好啦，好啦，你又不是没有做过。你难道忘了自己在蛤蟆寨交待过什么，这个时间不知道莫鸣米睡着没有，要不要我去找他聊聊。”
<br/>　　莫鸣米皱紧眉头，早就该想到，有自己画押的资料有朝一日将成为要挟自己的工具。现在莫鸣米正一腔怒火无处发泄，如果他看到这些，定然会把这次战败的原因怪在自己身上。
<br/>　　想到这点，莫鸣米下意识觉得不寒而栗。
<br/>　　好在，程大雷要的这些资料并不重要，大概类似流水账一样的东西。难道莫鸣米什么时候吃饭，什么时候上茅房会决定战争的胜败。
<br/>　　这时候外间更梆响起，巡逻的卫兵此刻从门前经过。严笛一直在等这个机会。此刻眉间一厉，正欲转身，开口大喊让卫兵进来。
<br/>　　程大雷猛地出手，手中茶杯砸在严笛嘴上，直接将他砸倒在地上，严笛满口鲜血，吐出一颗门牙。
<br/>　　严笛抬起头，眼里布满惊恐，在他的视线中，程大雷长身而起，身形显得无比高大。
<br/>　　程大雷理了理身上袍子，道：“我说过，有些事情不要轻易尝试，你没有太多机会的。我再给你一次机会……”程大雷伸出大手，五指张开：“乖乖合作，我盯着你呢。”
<br/>　　严笛恍然失神。
<br/>　　“子龙，我们走。”
<br/>　　程大雷和赵子龙在严笛的视线中消失，直到他们的脚步声也在耳中消失不见，严笛才发现自己后背已经被冷汗湿透。
<br/>　　至于程大雷和赵子龙如何避过卫兵神不知鬼不觉的离开，严笛一无所知，正如他不知道二人是如何来的。严笛只知道他们来了，然后他们又走了，如同鬼魅一般。
<br/>　　第二天，严笛便对外宣称昨天起夜时不小心跌倒摔掉了门牙，在家里养伤。然后，他便开始着手准备程大雷所需要的资料。
<br/>　　莫鸣米几时起床，几时睡觉，每天吃什么，又与什么人吃……严笛想破脑袋也想不明白，这些东西有什么意义。似乎自己胡编乱造，程大雷也无办法确认真伪。
<br/>　　如果程大雷真的需要琴川关的情报，也该是兵力部署，仓库位置，或者让自己给莫鸣米饭菜里下毒之类的。
<br/>　　可他并没有这么做，这让莫鸣米觉得稀里糊涂起来。
<br/>　　黄昏时，严笛一个人出城，按照事先约定，把资料放在城外一棵大树的树洞中。
<br/>　　一路小心谨慎，生怕被任何人发现。以严笛在琴川关的身份，倒也不会有人怀疑他。他怕的是程大雷会突然跳出来，给自己一剑。
<br/>　　不得不说，如今严笛已经被程大雷搞出了心理阴影。
<br/>　　如此过去三日，严笛每天藏在树洞中的情报都会被取走，倒也没有什么事情发生，严笛已快习惯了。
<br/>　　当他转身正准备离开的时候，猛地发现程大雷出现在身后。
<br/>　　严笛吓得一哆嗦，结结巴巴道：“你……你……”
<br/>　　“好啦，好啦。”程大雷挥挥手：“我来一趟很不容易的，你想想我什么身份，每次亲自和你接头，你这个间谍的身份不低呐。”
<br/>　　“什么时候，什么时候我成了你的间谍？”
<br/>　　“好了，严笛同志有新任务了，组织上要交给你一个新任务。”
<br/>　　是不是哪里搞错了什么，我什么时候成了有组织的人。
<br/>　　“我需要知道莫鸣米的家庭情况，他有几个妻子，几个孩子，几个小妾……”
<br/>　　“这些很重要？”
<br/>　　“对我来说，很重要。”
<br/>　　“我不会告诉你的，我只给你事先约好的，其他我什么都不会告诉你。”
<br/>　　“好啦，不要激动，作为一个优秀的间谍要学会控制自己的情绪。”程大雷从树洞中掏出那份资料：“辰时起床，细粥、肉糜，饮酒三盅……啧啧，你说你连莫鸣米每天吃什么都告诉我了，其他事情自然会说得一清二楚。莫鸣米要是看到这份东西，他会怎么想？”
<br/>　　严笛一激灵，恍然意识到自己落入一个圈套中。就像温水煮青蛙，水温一点点上升，但已经足够热时，青蛙已没有跳出来的力量。
<br/>　　“好啦，严笛同志，不要有思想包袱，认真完成组织交待给你的任务。”程大雷压低声音，用真诚的语气道：“正义必定会战胜邪恶，同志，光明不远了。”
<br/>　　
<br/>(本章完)
