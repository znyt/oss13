　　兑换《玄阴素女经》后，慕清霄总觉得精神上轻松许多。
<br/>　　上篇能让他的女人修炼成仙，他暂时不需要在为修炼心法的事情担忧。
<br/>　　等王语嫣她们都成仙，他肯定也不在缺五百万掠夺积分，到时候在兑换中篇，下篇也不迟。
<br/>　　此时，望着怀中身穿蕾丝睡衣的佳人，慕清霄嘴角牵起一丝邪异的笑容。
<br/>　　轻轻掀开蕾丝睡衣，一只手慢慢探了进去，掠过雪白嫩滑的肌肤，最后将一只圣女峰握在手中。
<br/>　　“唔唔...!”
<br/>　　睡梦中的欧阳紫嫣似乎有所察觉，粉唇中吐出几声娇吟，美丽的睫毛微微一颤，缓缓睁开朦朦胧胧的双眼。
<br/>　　察觉到在她圣女峰上作怪的大手，绝美的脸颊上，顿时绯红如霞。
<br/>　　“唔，流氓，我还要去集团呢！”
<br/>　　见她嘴上说着，身体上却享受着，慕清霄一个翻身，直接压了上去。
<br/>　　伴随着十多分钟的前事，慕清霄长驱直入，卧室中就响起一阵高低起伏的婉转呻吟。
<br/>　　半个时辰后，欧阳紫嫣香汗淋漓的趴在他的怀中，美眸痴痴的望着他。
<br/>　　“我待会还要去集团，还要去让人收集资料，你去找语嫣她们吧。”
<br/>　　见慕清霄似笑非笑的望着她，欧阳紫嫣恨不得钻进被窝里。
<br/>　　慕清霄也不再调戏她，大手在她的圣女峰上捏了一把，道：“以后别在修炼《太极玄清道》，改换更好的心法。”
<br/>　　忍不住白了他一眼，欧阳紫嫣乖巧的点了点头。
<br/>　　见此，慕清霄食指与中指温柔的指在她的玉额上，旋即将《玄阴素女经》上篇的心法路线，传给了她。
<br/>　　“《玄阴素女经》好玄妙的感觉，这是什么级别的心法？”
<br/>　　记录着脑海中的一部分心法，欧阳紫嫣心中满是喜悦。
<br/>　　“不是普通心法能比拟的，《太极玄清道》与《玄阴素女经》根本没有可比性，这可是真正的证道心法。”
<br/>　　见她露出好奇的目光，慕清霄微笑道：“距离证道太远了，虽只有上篇，但让你们修炼成仙却不是问题，现在唯一缺的就是资源。”
<br/>　　压下兴奋的情绪，欧阳紫嫣忍不住在他脸上亲了一口，她虽然不懂证道是什么意思，但成仙却略懂一些。
<br/>　　空闲的时候，慕清霄也给她们讲过一些修炼的体系。
<br/>　　修炼体系：练体，炼气，筑基，结丹，元婴，化神，炼虚，合体，大乘，渡劫。
<br/>　　大乘过后，便是修士第一劫，渡过九重劫难，便是地仙，也就是所谓的仙人。
<br/>　　地仙寿命长达万年，对于任何人来说，悠长的寿命，容颜永驻都是绝对无法拒绝的诱惑，特别是女人。
<br/>　　最重要的是，她以后都能与慕清霄在一起，甚至永远在一起，这才是令她最高兴的事情。
<br/>　　看到她浮想翩翩的模样，慕清霄捏了捏她小巧的鼻子，道：“不要好高骛远，努力修炼才是正道，资源不必担心，我会想办法的。”
<br/>　　“嗯。”
<br/>　　在床上又是一番耳鬓厮磨后，慕清霄才恋恋不舍的起床。
<br/>　　洗漱过后，欧阳紫嫣与雅儿她们都前往集团，王语嫣和小依也前往学校上课，阿碧和雏田则早早的就起床，开始准备今日的伙食。
<br/>　　陆雪琪穿着宽松的睡衣，盘坐在沙发上，默默的看着电视。
<br/>　　将她揽入怀中，见她脸红羞涩的模样，慕清霄就忍不住调戏她几句，惹的陆雪琪在他腰间软肉上一阵乱掐。
<br/>　　当他露出一副肉疼的模样，这妮子却忍不住帮他轻轻的揉了揉，一副心疼模样。
<br/>　　如此悠闲的日子，慕清霄知道持续不了多久，自然而然的就全身心享受起来，将她抱在怀中，嗅着她的幽幽体香。
<br/>　　“雪琪，《太极玄清道》几乎已经到极限了，需要换别的心法了，你平时在努力，修炼的速度也会变慢。”
<br/>　　沉思片刻，陆雪琪微微点头，修炼《太极玄清道》的她最清楚不过，这门心法确实已经修炼到极限了。
<br/>　　哪怕修为依旧在提升，但提升的速度却非常缓慢。
<br/>　　随后，慕清霄就用同样的方式，将《玄阴素女经》上篇的心法路线，传递到她的脑海中。
<br/>　　将心法路线完全记住后，陆雪琪软软的趴在他的怀中，语气温柔，道：“清霄，我想回去看看师傅。”
<br/>　　闻言，慕清霄微微一叹，将她紧紧抱在怀中，道：“在过几年，到时候我陪你一起回去。”
<br/>　　“嗯，听你的。”
<br/>　　安抚过陆雪琪，慕清霄来到游泳池旁，仰躺在太阳椅上，微闭双眼，思考着昨晚的事情。
<br/>　　李春的出现，无疑给慕清霄带来了很大的影响。
<br/>　　不过，根据系统的回复，地球上确实不存在任何修士，华夏国术这些都是地球上古老的传统，存在并不令人意外。
<br/>　　但李春却是练气期的修士，除了李春，会不会有其他修士存在，比如一年前，全国各地与李春一起消失的那群人。
<br/>　　当然，慕清霄并非惧怕，区区练气期修士，在他眼中不过是挥手可灭的蝼蚁。
<br/>　　而是感到好奇，他们是如何获得修为的？
<br/>　　想了半天，他都想不出个所以然，于是便将此事抛之脑后，等时机到了，相信一切都会水落石出。

<br/>　　《五行幻阵》在天墉城所有阵法中，都是属于最顶级的阵法。
