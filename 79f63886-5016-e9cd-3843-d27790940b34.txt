　　“借助剑源界的力量，纵然是荒天王，少帝，我都足以斩杀，可惜只怕没有这个机会，”
<br/>　　楚风眠感觉到，他如果借助剑源界的力量，面对真正的半步天人境巅峰都足以斩杀。
<br/>　　不过这种机会，却是几乎不会有。
<br/>　　像是荒天王，已经是吃过一次亏了，绝不会再来剑源界。
<br/>　　这五个人也是因为他们来自九华天，身份尊贵，心中狂傲无比，没有将剑源界剑道门，就连楚风眠都没有被他们放在眼中。
<br/>　　才会给了楚风眠这样的机会，可以借助剑源界的力量，将他们五个人全部斩杀。
<br/>　　如果这五个人谨慎一点，在外先攻打剑源界，也不会落得这样的下场了。
<br/>　　不过现在也没有给他们重选的机会了。
<br/>　　这五个人，也为他们的狂傲，付出了代价。
<br/>　　楚风眠心神一动，催动起了吞天秘术。
<br/>　　“好雄厚的精血，这些人的精血，要比起寻常半步天人境的精血，纯粹的多，这些人不知道都是炼化过多少的奇物，才将精血凝炼到了如此地步！”
<br/>　　这五个人的精血，都被楚风眠吸入到了吞天秘术之中，稍微炼化，楚风眠就感受到了这五人精血之中的灵力纯粹。
<br/>　　怪不得这些人，看起来也只是刚刚踏入了半步天人境，一身的灵力就要比起五行刀圣这样的人物还要雄厚。
<br/>　　他们每个人的精血之中，都蕴含着大量残留的灵力，是炼化了奇物，都没有全部炼化，留下的灵力。
<br/>　　这五个人身躯之中，其中的种种奇物，可是在九域之中都消失了数万年的，一般只有在一些古老的遗迹之中，才有存在之物。
<br/>　　现在却都被他们大量的炼化，来增添他们的实力。
<br/>　　“九华天，不愧的圣域，如果可以给我这些奇物，我都有机会冲击半步天人境了。”
<br/>　　楚风眠不由的想到。
<br/>　　他想要冲击半步天人境，寻常的圣药，灵丹等等都已经效果不大了，只有诸天奇物，甚至是仙物，才对楚风眠冲击境界，有些效果。
<br/>　　这五个人身上的精血虽然蕴含着极大的灵力，但是对现在的楚风眠来说，却是没有多大的用处。
<br/>　　楚风眠现在已经不需要在积攒力量了，他所需要的，只是寻找诸天奇物，寻找突破境界，踏入半步天人境的机缘。
<br/>　　虽然说现在楚风眠也可以将五个人的精血，融入到剑源界之中，化为剑源界的力量，可这未免太过浪费了。
<br/>　　毕竟这样五名妖孽天才级别的半步天人境武者，楚风眠想要寻找到，可不容易，除非楚风眠去九华天才可能碰得到。
<br/>　　不过现在楚风眠杀了九华天的这五名半步天人境，现在楚风眠去九华天，简直是老寿星上吊自己找死。
<br/>　　在楚风眠杀了这五个人的时候，楚风眠都感觉到了，他被一股力量所窥探了。
<br/>　　这一股力量已经是将楚风眠的气息，给记住了。
<br/>　　这一股力量的主人，绝对是踏入天人境的人物，楚风眠都没有办法蒙蔽这一切。
<br/>　　只要是楚风眠现在去九华天，只怕很快就被这几人背后的家族所杀。
<br/>　　所以这五人的精血，可不能浪费。
<br/>　　“凝炼成血丹，给寒月漓，洛汐，青梦师姐，燕皇等人来提升实力！”
<br/>　　楚风眠目光闪烁，想了一下，做了决定，他要将这五人的精血，凝炼成血丹。
<br/>　　这五人的精血之中，可都有着许许多多没有炼化的药力，诸天奇物的力量，凝炼成丹，比起古圣丹蕴含的药力还要精粹。
<br/>　　楚风眠运转起吞天秘术来，顿时这五人的精血，便是分成了数百份，开始凝聚成丹。
<br/>　　这五人的精血蕴含的灵力太过雄厚，如果凝炼成一枚丹药，只怕是半步天人境的武者吞噬炼化，都要被活活撑死。
<br/>　　更何况寒月漓，洛汐的境界还只是九劫古帝。
<br/>　　楚风眠即将也要去沧海星域一趟，路途遥远，现在有了这些血丹，也足以增添他们的实力，甚至突破半步天人境。
<br/>　　楚风眠想要突破九劫古帝，踏入半步天人境，可是要比起寻常九劫古帝要难上千倍万倍，所以倒不如先让寒月漓，洛汐借助血丹突破。
<br/>　　凝！
<br/>　　楚风眠手指一点，一枚枚的血丹，便是都被凝聚完成，说是血丹，但是其中已经是没有一点点的血腥味了，有的只有药香。
<br/>　　这一枚枚血丹蕴含的药力，比起中古级别的圣丹还要精粹的多。
<br/>　　楚风眠手一挥，这些血丹便是都被分了下去。
<br/>　　现在剑道门中，有着小九，心魔之王坐镇，已经是没有之前的那般弱小了。
<br/>　　除非像是碰到圣域中人，其他的突发情况，都足以应对了。
<br/>　　只要是洛汐，寒月漓，青梦等人之中，谁真正的踏入了半步天人境，剑道门的实力还可以再进一步，接近到当初圣龙秘境的底蕴了。
<br/>　　毕竟曾经的圣龙秘境之中，一共也就只有五位半步天人境强者。
<br/>　　可惜的就是，楚风眠自己踏入半步天人境，依然是遥遥无期。

<br/>　　楚风眠突然想到，手心一动，五枚空戒出现在楚风眠的手心之中，在杀了这九华天五人的时候，他们的空戒也都被楚风眠顺手收取了。

<br/>　　大量的灵药，诸天奇物，古圣丹，灵石，等等，全部都落入到了楚风眠的眼中。
<br/>　　这一枚枚空戒，都如同一座座宝库一样。
<br/>　　琳琅满目的无数宝物，看到楚风眠都有点眼花。
<br/>　　死在楚风眠手中的半步天人境强者，已经是有着数十个了，可是还从来没有谁的空戒，可以跟眼前这五枚空戒相提并论的。
<br/>　　“这五个人，不愧是来自于圣域九华天的，身上的财富比起齐林山的宝库还要多得多。”
<br/>　　楚风眠观察了一眼，心中便是惊叹不已，心神一动，便是从那火天的空戒之中，取出来了一枚通红的果子。
