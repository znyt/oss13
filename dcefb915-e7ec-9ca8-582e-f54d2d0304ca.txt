　　原来这白发老者，正是公孙林的哥哥公孙茂，他听闻自己的弟弟，被叶尘所害，自然便对他恨之入骨。
<br/>　　“宗主大人，倘若那小子不敢来呢？又或者他根本不知道如何进入蓬莱仙界，那我们的计划岂不是就要落空了？”
<br/>　　药王宗宗主云天青，顿时冷冷一笑道：
<br/>　　“他若是不敢来，那我们便杀入世俗界！今日我邀请其他四大宗门的宗主前来，便是为了商议此事！”
<br/>　　公孙茂听到这话，先是一喜，继而又露出一抹担忧之色，
<br/>　　“可是我们仙界的先祖，早有严令，蓬莱仙界之人，不得随意插手世俗界之事，而且我听闻那家伙，背后依仗世俗界的国家力量，也极为不好惹，我就怕其他四大宗门，不肯轻易出手啊！”
<br/>　　云天青淡淡一笑，
<br/>　　“那小子在世俗界搞出这么大动静，还公然建立修真门派，广招门徒，我就不信其他四大宗门会坐得住？”
<br/>　　“以我对四大宗门的了解，只要咱们药王宗打头阵，他们肯定也乐见其成，根本没有拒绝的理由！”
<br/>　　公孙茂一想也对，这才露出一抹喜色。
<br/>　　云天青接着又道：
<br/>　　“至于那个叫苏蔓的女子，你派人去严加审讯，看看能不能从她的口中，问出叶狂仙的师门传承！”
<br/>　　“倘若三天之后，那叶狂仙还没有出现在蓬莱仙界，你就砍了她的人头，派人给那小子送过去，这就叫杀人诛心！桀桀！”
<br/>　　云天青说到最后，得意的狂笑的起来。
<br/>　　公孙茂当即站起身来，同样露出一抹阴毒的笑容，
<br/>　　“宗主大人高明！我就按照您的吩咐去办！”
<br/>　　……
<br/>　　另外一边，叶尘带着祝小白，用了大约十多分钟的时间，就顺利穿过了血雾毒瘴。
<br/>　　而前方，却空无一物，不但没有任何建筑，连人影都没有。
<br/>　　“怎么会这样？”
<br/>　　祝小白顿时一脸失望。
<br/>　　叶尘笑道：
<br/>　　“不过是一道幻术禁制罢了！”
<br/>　　说着，叶尘直接朝虚空中随手点了继续，前方空间微微一晃，不远处顿时便出现了一排排建筑，俨然如同一座山中小城。
<br/>　　“走吧！”
<br/>　　叶尘当先一步迈入其中，祝小白反应过来之后，也连忙跟上。
<br/>　　等两人都踏入其中，那道幻术禁制便重新复原。
<br/>　　“我们先去救人！”
<br/>　　叶尘说了一句，然后便闭上双眼，将神念全力释放出来，开始搜索苏蔓的行踪。
<br/>　　整个药王谷，足足占地上百亩，好在以叶尘如今的神识，已经可以笼罩方圆五百里，只要苏蔓人不在结界禁制之中，便逃不过叶尘的神魂搜索。
<br/>　　“找到了！”
<br/>　　只用了两三分钟，叶尘便已经搜寻到了苏蔓的气息，
<br/>　　很显然，药王谷的人，怎么也没有想到，叶尘竟然会这么快就找上门来。
<br/>　　不过很快，叶尘眉头一皱，脸上浮现出一抹浓浓的煞气，
<br/>　　“找死！”
<br/>　　……
<br/>　　与此同时，药王谷的大牢之中，
<br/>　　苏蔓被绑在一根木桩之上，身上还有一道道用鞭子抽打的血痕，看上去显然已经受了不少苦。
<br/>　　“小姑娘，没想到你年纪轻轻，骨头倒是挺硬的嘛！”
<br/>　　“本长老不过就是想知道那小子的师门传承，又不是什么大不了的事情，你又何必这么倔强呢？”
<br/>　　公孙茂一脸奸笑的凑到苏蔓的跟前，缓缓开口道。
<br/>　　“我呸！”
<br/>　　苏蔓直接一口血水朝公孙茂吐了过去，可惜那公孙茂，好歹也是金丹境巅峰的高手，被他轻松避开。
<br/>　　“我已经说了，我不知道！而且就算我知道，我也绝对不会告诉你们的！你们要杀便杀，本姑娘绝不会皱一下眉头！”
<br/>　　苏蔓掷地有声，怡然不惧。
<br/>　　公孙茂听到这话，眼眸之中顿时闪过一抹浓浓的杀意。
<br/>　　如果不是宗主云天青有令在先，他才不会和一个世俗界的凡人废话，早就一掌结果了这个倔强的丫头了。
<br/>　　不过很快，公孙茂的两眼之中，又闪过一抹淫秽之色，
<br/>　　“嘿嘿，想死？哪有那么容易！你弟弟叶狂仙杀了我的弟弟，这笔账我就先从你的身上讨还点利息回来！”
<br/>　　说到这里，公孙茂忽然手掌一翻，从乾坤镯里面，取出一枚粉色药丸，然后屈指一弹，便飞入了苏蔓的口中。
<br/>　　苏蔓反应过来之后，当即便想要将药丸吐掉，可惜公孙茂又岂会让她如愿，直接又朝屈指一弹，瞬间便击中了她的檀中穴。
<br/>　　“啊！”
<br/>　　苏蔓穴道中招，顿时惊呼了一声，那颗丹药便一口咽了下去。
<br/>　　“你、你刚刚给我吃了什么东西？”
<br/>　　公孙茂嘿嘿一笑，满脸淫色，道：
<br/>　　“我们药王宗，不仅擅长炼丹，更擅长制毒！”
<br/>　　“刚刚本长老给你吃的，名为七情六欲丹，就算是再绝情的贞洁烈女，吃了这个丹药之后，不出一时半刻，也会变成为荡妇**，比人尽可夫的妓女，还要淫荡百倍！桀桀桀！”
<br/>　　说到这里，公孙茂冲着身后四名药王宗弟子一招手，
<br/>　　“你们四个过来，待会好好伺候伺候这位苏蔓姑娘，也让她见识一下，咱们药王宗的待客之道！桀桀！”
<br/>　　那四名弟子闻言，顿时个个狂喜，
<br/>　　他们早就对眼前这个大美人垂涎三尺，只不过没有上头的命令，所以才不敢越雷池一步，如今得到公孙茂的命令，自然正中下怀，
<br/>　　“多谢公孙长老！”
<br/>　　“公孙长老放心！我们兄弟几个，保证把她伺候的舒舒服服的！嘿嘿！”
<br/>　　“小美人，我们来了！”
<br/>　　……
<br/>　　眼看着那四人，已经开始缓缓朝苏蔓逼近，再加上公孙茂之前的那番话，就算是傻子也明白，接下来要发生什么了。
<br/>　　苏蔓顿时吓得脸色惨白，拼命的想往后退，可是整个人还被绑在刑架之上，根本无法动弹。
<br/>　　“你们不要过来！不要过来！你们这帮禽兽！无耻之徒！快点放了我！否则等我弟弟来了，他一定会将你们碎尸万段！！”
<br/>　　公孙茂顿时得意的哈哈大笑，
<br/>　　“小丫头，我劝你还是不要白费力气了，你弟弟他就是一个胆小鬼，根本就不敢来我药王宗！”
<br/>　　苏蔓顿时更加绝望，就在她心头一横，准备咬舌自尽的时候，忽然一道熟悉的声音，陡然响起，
<br/>　　“是谁说我不敢来的？”
