　　明尼城南面是一片茫茫戈壁，东面则是里海，‘后撤’的大唐军团根本逃不过追兵的眼睛，隔着十余里便可以看到他们正在仓惶逃窜。
<br/>　　伊力克这次也是拼了，在看到自己的敌人逃遁的背影之后，不顾一切的发出了追击的命令，一时间两万拜占庭骑兵蜂拥而出。
<br/>　　龙义的九十二师一直在等待着拜占庭人的进攻，毕竟演戏这种事情实在是太累人了，明明有实力将敌人彻底消息，但却不得不逃跑，而且还要逃出气势，这对于一群骄兵悍将来说实在太困难了。
<br/>　　“准备战斗，拜占庭人上来了。”逃着逃着，身后的马蹄声越来越近，负责断后的一个旅，近五千人以散兵线依次停了下来。
<br/>　　“*&amp;%￥……”拜占庭人乱七八糟的喊声越来越近，他们还是第一次见到有人会用散兵线来面对骑兵，对胜利的渴望让他们再一次加快了速度。
<br/>　　“世人都说唐军彪悍，我看不过尔尔，如果主力在两天之内还不上来的话，这一次凭我们应该就可以把这二十万人搞定。”伊力克远远的看着战场，不无得意的说道。
<br/>　　在他看来，唐军的表现就是大唐指挥官想要断尾求生，用几千人的代价换取数十万人的顺利脱身。而这样的情况必然会使得士气大降，如果进攻得当完全可以一个冲锋直接将断后的五千人彻底拿下。
<br/>　　伊力克身边的随从虽然看不懂战场上的形势，但拍马屁的功夫还是有的，闻言之后立刻马屁如潮，听上让人觉得伊力克就是拜占庭军神，二十万大唐军队完全可以凭借两万人一鼓而下。
<br/>　　但事实真的是这样么？九十二师早就已经做好了战斗准备，等的就是拜占庭人的到来，五千来人，数千支新式步枪，在拜占庭人距离他们还有三百余步的进候便开始了射击。
<br/>　　一时间，荒野戈壁上响起了连排的枪声，只一轮便有千余拜占庭战人倒了下去。
<br/>　　他们的队形实在太密集了，留下阻击他们的那个旅的士兵几乎不用瞄准，只把枪对准那密密麻麻挤在一起的骑兵队伍，齐射之下便不会落空。
<br/>　　看着部下造成的战果，九十二师第一旅的旅长大人几乎崩溃，连蹦带跳的喊道：“枪口抬高，枪口抬高，你们都是猪么，告诉过你们不要打的太狠，听到没有！”
<br/>　　第一旅的士兵当然听到了，不过谁又能看着敌人扑上来，放着战功不要而放空枪呢，所以在拜占庭人犹豫的那一瞬间，第二轮枪声又再次响起，转眼又是千余人倒下。
<br/>　　几乎是瞬眼之间，进攻的人数就少了十分之一，这样的变化让伊力克瞬间就傻了，几乎不敢相信自己的眼睛。
<br/>　　而且唐军到底是怎么做到的？明明还隔着那么远，却可以一下子弄死自己十分之一的进攻部队，这简直太不可思议了，说是在打仗还不如说是在施妖法。
<br/>　　“呯呯……”就在伊力克胆战心惊的犹豫要不要撤退的时候，又是一轮枪响不过这一次倒下的只有三百人左右。
<br/>　　三百人，这样的损伤还在合理的范围之内，伊力克松了口中气，暗道刚刚应该是唐军提前在战场上布下了什么埋伏，现在埋伏的东西用光了，他们造成的伤害也就小了。
<br/>　　这一切说来话长，其实只发生在短短的一瞬间，两万左右的拜占庭人原本兴冲冲等着收割敌人的头颅，从大唐买来的弩已经被拿到手中，只等接近到了一定的位置就可以发起进攻。
<br/>　　可是，突如其来的三轮攻击将所有进攻的拜占庭骑兵一下子打的懵掉了，倒下的战马和士兵成了天然的绊脚石，将后面的同伴拦了下来。
<br/>　　但就算这样，还是有数百人因为躲闪不及而落马，摔得骨断筋折。
<br/>　　这一下子，骑兵高速突进的优势变的荡然无存，再想进攻一是战马没有足够的距离加速，二是伤兵满地拦住去路。
<br/>　　“撤，让他们撤回来！”伊力克眼见攻不上去，只能下达撤退的命令，将正在彷徨的士兵从战场上拉了下来。
<br/>　　而这个时候，唐军的阻击并没有停止，枪声还在不断的持续，每一个呼吸都会有拜占庭骑兵落马，成为战场上的牺牲品。
<br/>　　龙义远远的用望远镜看着战场上的情况，当看到一轮齐射干掉拜占庭上千人的时候，他兴奋的挥了挥拳头，但很快又懊恼的锤了锤脑袋。
<br/>　　该死的，为什么不能全力进攻呢，拜占庭的主力军团到底什么时候才能上来，这特仗打的真是太窝囊了，明明有实力一次全歼来敌，但却不得不放空枪让他们跑掉。
<br/>　　而伊力克在这个时候却陷入了一个进退两难的境地，刚刚一次进攻连唐军的防线都没有摸到，自己这边已经损失了十之一、二的人手，再继续进攻的话会不会重蹈覆辙？
<br/>　　可是如果退兵，似乎也不好向上头交待，毕竟只打过一仗，连唐军的底都没有摸到就退兵回去怎么也要吃些排头。
<br/>　　……
<br/>　　就这样，双方人马在原地对峙了起来，唐军因为害怕将拜占庭人吓跑不敢进攻，拜占庭人因为唐军的反击过于坚决，所以决定等持主力部队。
<br/>　　时间就这样一点一点过去，一直僵持到了天色渐渐暗淡，双方才彻底收起对攻的姿态，各自扎下营地休息。
<br/>　　不过伊力克不知道的，在他扎营的时候，已经有一支三万人的军队从他的侧方绕了过去，向着西面拜占庭与大食的交界处扑了过去。
<br/>　　而此时在明尼城的城外，拜占庭的主力部队也终于赶到，正沿着他们追击的路线前进着。
<br/>　　前两天的战报他们已经收到，知道了唐军粮草被毁的消息这后，拜占庭的主力部队几乎放弃了一切的辎重部队，紧赶慢赶的冲了上来，打算借机会痛打‘落水狗’。
<br/>　　毕竟战场之上，一方在没有了粮草之后，几乎与待宰羔羊没有什么区别，将他们‘吃掉’之后，拜占庭恢复往日的荣光指日可待。
