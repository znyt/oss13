　　这时候，那些浮在水面上的泡沫里的生灵们，身上的泡沫也在罡风之中被挤压成不同形状，随风飘着，里面的生灵惊恐万状。
<br/>　　这些泡沫在罡风的裹挟之下，一个个都升到了半空中，紧紧贴着结界的顶端，就在众人惶恐不知所措的时候，结界的最上面、电弧缓缓打开，出现了一个缺口。
<br/>　　叶少阳，你看看这些人，不久前还对你忠心耿耿，如今呢，一旦遇到危险，谁还认识你？什么天选之子、人间道神，生死之际，又有谁在乎你？”
<br/>　　鲲鹏面带微笑，慢吞吞地说道。
<br/>　　叶少阳知道他说这些是为了打击自己的道心，但也不得不承认，他说得也是有道理的。
<br/>　　那些被泡沫包裹着的生灵，也都听到了鲲鹏的话，一个个脸上都露出了愧疚之色，但是出口就在前面，倒也没人敢反驳，甚至没有人敢回头看叶少阳一眼。
<br/>　　突然，海水之中腾起了一道巨浪，朝着结界上方的缺口射去，叶少阳三人正被罡风吹得痛苦不堪，突然从水里伸出了一个好像触角一样的东西，拦腰将三人裹住，借着波浪腾空而起。
<br/>　　叶少阳转头一看，捆住自己的是原来是花茎，便知道是碧清出手了，想趁着法阵打开的时候逃出去。
<br/>　　影魅等人自然早有准备，加速作法，更多的电弧从上面坠落下来，砸在翻腾的浪花上，浪花带着他们一起坠落在了水中。
<br/>　　这次尝试失败了。
<br/>　　玉玑子看到叶少阳几人狼狈的模样，哈哈大笑，望了一眼那些被泡沫裹住的生灵，突然想到一个阴损的报复手段，冲他们喊道：“你们前面就是出口了，可自行飞出来，但你们想要归顺我法术公会，还需要表下忠心……”
<br/>　　大伙面面相觑，不知道这个可恶的老道士要他们做什么。
<br/>　　“给我大声骂他，骂叶少阳是王八蛋，骂他是畜生！只有骂了他，才让你们出来！”玉玑子说着，念起剑诀，剑阵的出口被两道电弧的光暂时封堵住了。
<br/>　　叶少阳坠落在水中后，又被碧清托起来，在罡风中咬牙说道：“玉玑子，你得是多会儿恨我！”
<br/>　　“恨你？哈哈，你也知道啊，我昆仑一脉，从我手中断了人间的香火，这全是被你害的！老夫我忍辱负重到今天，就是为了报复……你们倒是骂啊，骂他啊！”
<br/>　　几个盘古僧皱起了眉头，其中一个还哼了一声，鲲鹏听到，也觉得这样做有点过，影响了计划，刚要开口阻止，影魅在神识中对他说道：“不要阻拦，这混人虽是为了报复，倒也合了我的意思。”
<br/>　　“师伯，他只是为了自己报仇痛快而已，对我们能有什么好处？”
<br/>　　影魅只回了两个字：“且看！”
<br/>　　时间一点点过去，这些生灵互相望着，意思很清楚，都在等着别人先开口，如果大家都那么做了，那自己也就只能跟着，说白了就是给自己找个安慰。
<br/>　　“快啊！”
<br/>　　玉玑子听不到自己想要的，加速念咒，结界的开口一点点闭合。
<br/>　　“一分钟，只有一分钟时间，到时候结界关闭，你们就要跟叶少阳一起死了！”玉玑子催促着，眉飞色舞地看着叶少阳，静静地等着。
<br/>　　“叶少阳……”一个邪物总算开口，偷偷看了叶少阳一眼，道：“王八蛋！”
<br/>　　“连起来说！大声点！”
<br/>　　“叶少阳王八蛋！”那邪物吞了一下口水，说完这句话，脑袋立刻低下去，不敢看任何人。
<br/>　　玉玑子大笑，“好好好！说的好，说得对！下一个！”
<br/>　　作法将这邪物的泡沫托了出去。
<br/>　　从法阵出去，这泡沫立刻就炸开了，邪物跌落到水里，又游到岸边，自觉地朝玉玑子他们身后走去，没一个人搭理他，他自觉地站到了他们身后去，低着头，他知道，从今以后，自己只能给法术公会当狗了。
<br/>　　“叶少阳王八蛋……”
<br/>　　“叶少阳小畜生……”
<br/>　　有一个人干了头，后面的心理压力就小很多，一个个都跟着骂叶少阳，逃离了剑阵。
<br/>　　“哈哈哈哈……”
<br/>　　玉玑子兴奋地狂笑不止。
<br/>　　“这人疯了！”小马看着玉玑子，摇了摇头，对叶少阳道：“他们是被逼无奈，你可别介意。”
<br/>　　叶少阳苦笑。
<br/>　　半分中过去，已经有近百号人被送出去，后面那些……却一时间卡了壳，没人愿意再说。
<br/>　　玉玑子大声催促他们，警告他们法阵就快要关闭了。
<br/>　　“叶少阳……”
<br/>　　一个蜘蛛精转头看了叶少阳一眼，朗声说道，“叶少阳是顶天立地的大人物！是我们的救命恩人！”
<br/>　　“什么！”玉玑子吃惊地叫起来。
<br/>　　那邪物恨恨说道：“之前让我们念咒，我们也都忍了，就算给法术公会当狗，为了能活下去，也就认了，但是让我辱骂叶少阳……我做不到啊！人间有句话，君子有所为，有所不为，我不是君子，但让我一再违背自己的良心，恩将仇报，我做不到啊！”
<br/>　　“好，成全你！”
<br/>　　玉玑子气急败坏，双手作法，堪破了这蜘蛛精身上的泡沫，一股阴阳罡风飞来，直接将他肉身和魂魄撕碎，化作一滩血雾，落在了水中。
<br/>　　但这血腥的一幕，却没有吓到其他人，反而激起了他们心中悲愤的情绪，不知道谁带头喊起来：“叶少阳是大英雄，你才是畜生，是王八蛋！”
<br/>　　这一声骂，立刻点燃了大伙的情绪，不少人跟着一起骂起来。
<br/>　　多少人一起骂，声音穿透了法阵，传到了在场每个人的心里。
<br/>　　骂声中带着愤怒，带着不屈，带着一直以来被压抑的良心上的负担，在骂声中全都宣泄了出来。
<br/>　　叶少阳昂着头，呆呆地看着这突然发生的一幕，之前被人骂的时候，他是保持了平常心的，但不免还是有些心寒，而眼下发生的这一幕，却远远超出了他的预料。
<br/>　　有情有义的人，还是有的啊。
<br/>　　只要有一个这样的人，自己之前所做的那一切，就不是没有意义，也不会心灰意冷。
