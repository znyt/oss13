
郭德江这时候心里无疑是很崩溃的，当年被起名先生忽悠着用了这个曌字，他还觉得意义不错呢，拿自己女儿的命去和武则天比，现在想想这个想法挺无语的，一个好吃懒做只知道吃喝玩乐的富家女，拿什么跟一代女王相提并论，她确实担不起日月凌空的这个命。<br/>　　<br/>　　郭德江头疼的问道：“王先生，换个名就行了？”<br/>　　<br/>　　“名字肯定得换，但还不算完”王惊蛰点了点头，话留半截就没在往下说了。<br/>　　<br/>　　郭德江叹了口气，却没先表态，而是来到楼上的收藏室，从中取出当年林汶骐卖给他的三件古董。<br/>　　<br/>　　张兵小声跟王惊蛰说道：“哥们，没看出来，你还会两下啊，道上的？”<br/>　　<br/>　　“略懂一二”王惊蛰矜持的说道。<br/>　　<br/>　　张兵顿时欢脱了，他还没等开口，霸王龙一下子就从沙发上蹦了起来，雀跃的说道：“师傅，你给我看看姻缘呗……”<br/>　　<br/>　　王惊蛰嘴角抽搐着说道：“不是，你就这么渴嘛？”<br/>　　<br/>　　霸王龙忽闪着大眼睛一脸娇羞的说道：“空虚了二十几年，每到夜晚总是有些不自觉的躁动起来，小女子对于那男女之事也是心生向往的”<br/>　　<br/>　　这话吧，你要是换一个娇滴滴的姑娘来说，可能别有一番情趣，但换成一头霸王龙跟你说，就会有一种屎到门口，然后出不来的感觉了。<br/>　　<br/>　　“唉，哥们我跟你讲，你要真是会两手的话就给我看看？”张兵诚挚的跟王惊蛰小声说道：“不瞒你说，我爸自从退下来后，这都多少年了，我始终过得不是那么好，日子肯定比小康要强一些，但是吧我这人以前享受惯了，总觉得忽然之间从云顶上掉下来后就有点空落落的，你看有没有啥办法，让我腾飞一下呢？”<br/>　　<br/>　　王惊蛰笑道：“兵哥，人的命天注定，你父亲在位的时候你过得好了，那是享受父辈余荫，他下来了你过得不太好了，那就说明家中的福运也少了一部分，这些都是因果关系不能强求对不对？”<br/>　　<br/>　　张兵脸色不太好看，尴尬的叹着气：“我懂，一朝天子一朝臣么，他不在其位不谋其政，自然就没人在乎他了”<br/>　　<br/>　　王惊蛰见他情绪有点小低落，就拍了下他的肩膀，在耳边说道：“兵哥，你人不错，我来到双城后你怎么做的我都看到了，这也算个人情，我肯定不会忘的，等忙完郭德江的事，我给你指条路试试”<br/>　　<br/>　　张兵眼睛顿时一亮，连忙点头说道：“好叻，谢谢兄弟了”<br/>　　<br/>　　郭德江从楼上拿下三个盒子下来，放到了桌子上，王惊蛰打开后扫了几眼，见是林汶骐说的那三样东西，就给收了起来。<br/>　　<br/>　　“名字不对，肯定得要改，她命格本弱根本就驾驭不住这个曌字，八字又软还偏水……”王惊蛰略微掐指算了算，跟郭德江说道：“把曌字去掉，改为焱吧”<br/>　　<br/>　　霸王龙撅着嘴说道：“听起来怪怪的，好像是个男孩子的名字，我不要”<br/>　　<br/>　　王惊蛰斜了着眼睛说道：“要不换成盖字也行，照样跟你匹配”<br/>　　<br/>　　霸王龙瞬间无声，抿着两片厚厚的嘴唇不敢接话了。<br/>　　<br/>　　郭德江随即问道：“名字能改，那接下来？”<br/>　　<br/>　　“给我拿纸笔过来”王惊蛰说道：“你女儿这个名字有点犯忌讳，触怒上天，所以名改了还不够，你得谢罪才行”<br/>　　<br/>　　纸笔拿过来后放到桌上，王惊蛰提笔就画了一起来，边画边说道：“在你家门前的空地上，修建一座喷泉池，这个形状以香炉为造型，池子中间竖起的三个喷口成品字形，三道水柱喷出来后就是三根朝天香，向上天祈拜的意思，喷池两边修出两条过道，台阶由下而上，这是登天阶，每当你家中人回来，或者有人来你家的时候，势必都会要经过这两条过道然后绕到喷池后面才能进入主屋，这就相当于每次有人来，都拜了天……”<br/>　　<br/>　　“啪”王惊蛰说完把笔扔在桌子上，指着霸王龙说道：“这也不算完，你这女儿前二十四年享受惯了荣华富贵，这一身贵气她本来是受不起的，所以得把这气给卸了才行，往后三年让她出点苦力吧，最好是把她放到山沟里去，当个支教的老师，吃三年苦后再回来，这命格自然就能变了”<br/>　　<br/>　　郭德江收起桌子上的那张纸，点头说道：“明白了王先生，你说的这些我马上就去办，全办好了自然就不会再有问题了吧？”<br/>　　<br/>　　王惊蛰说道：“郭老板，事呢都是事在人为呢，你做好了还哪里来的问题，照着我说的去做就行”<br/>　　<br/>　　王惊蛰说完就把桌子上的东西给收了起来，抬腿就要走，郭德江寻思了下，上前伸手拦着王惊蛰说道：“这么麻烦您了，我还没表示下谢意，王先生晚上一起吃个饭吧？”<br/>　　<br/>　　王惊蛰伸手挡了他胳膊一下，说道：“郭老板我和你之间仅限于此了”<br/>　　<br/>　　“啪”王惊蛰挡开对方胳膊，几步就走了出去，张兵连忙尾随过去，郭德江望着他的背影，无奈的摇了摇头，这人有没有本事，有多少道行，其实你很容易品出来。<br/>　　<br/>　　以郭德江的见地和经验，他对王惊蛰此时是挺服气的，为啥？<br/>　　<br/>　　你若换个懂点三脚猫本事的人，这时候肯定巴不得往他这大老板身上靠呢，然后故作高深的接着在往下扯几句云里雾里的行话，挑逗下他。<br/>　　<br/>　　但王惊蛰呢，办完事拿了东西就走，多余的话一个字都不想说，这是啥，这既是对方不屑与他为伍，人家这段位根本犯不上和他在接触下去了。<br/>　　<br/>　　郭德江挺懊恼的，昨天自己要是表现的礼贤下士一点，那可能双方还有交集，可惜机会跟他擦身而过了。
