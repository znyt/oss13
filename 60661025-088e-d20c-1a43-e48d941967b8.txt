　　曦月听到叶尘这话，先是一喜，继而又摇了摇头道：
<br/>　　“以我目前的神魂力量，还不足以脱离这幅躯壳，必须要先达到出元神的境界！”
<br/>　　“以我如今的状态，现在开始修炼神魂秘术，估计最少也要半年时间！”
<br/>　　叶尘点了点头，
<br/>　　“好！那就等你的元神能够脱离肉身，咱们再返回修真界！”
<br/>　　两人又聊了一会，曦月的精神状态，忽然开始变得萎靡起来，
<br/>　　“叶尘，这幅躯体只是我的临时住所，不能占用太长时间，否则我和宿主的魂魄，都会受到反噬！”
<br/>　　“以后我每天，应该只能占用两个小时左右……”
<br/>　　叶尘自然也明白这个道理，
<br/>　　其实最好的解决办法，就是把萧若曦的躯体夺舍。
<br/>　　但是曦月肯定不会这么做，所以叶尘也无可奈何，
<br/>　　“曦月，你放心，从今开始，我会一直守候你身边！”
<br/>　　‘萧若曦’微笑着点了点头，
<br/>　　“这个叫萧若曦的女孩，帮了我不少，你要帮我好好谢谢她！”
<br/>　　说完这话，她便缓缓闭上了双眼。
<br/>　　过了片刻之后，
<br/>　　萧若曦再次睁开双眼，但是身上的气质，已经发生了大大改变，而且一脸的迷茫之色，看到眼前的叶尘，顿时又露出一抹喜色，
<br/>　　“咦？叶尘，你怎么来了？我们这是在哪儿？”
<br/>　　很显然，此时占据身体控制权的，已经转换成了萧若曦。
<br/>　　叶尘望着萧若曦脸上，那真诚的笑容，不由得想起之前，自己还想让曦月夺舍她的身体，心中顿时大为愧疚。
<br/>　　当下便把情况，大致向萧若曦，全部讲了出来。
<br/>　　萧若曦听完这话，顿时大喜过望，
<br/>　　“我脑袋里面的那位姐姐，真的已经醒了吗？真是太好了！那我以后是不是就不用再吃那些药了？”
<br/>　　听到萧若曦这么说，叶尘顿时越发的惭愧。
<br/>　　他为了让曦月的残魂早日苏醒，除了让萧若曦按时服用养魂丹之外，还经常会让她吃不少补药，虽说对萧若曦的身体也大有裨益，但终究并非萧若曦自愿。
<br/>　　萧若曦帮了自己这么多，自己竟然还想夺舍她的躯体，实在有些太禽兽了。
<br/>　　“你以后如果不想吃的话，就就不要吃了，不过你体内的那位姐姐，每天要占用你两个小时的身体，你愿意吗？”
<br/>　　萧若曦微微一愣，旋即笑道：
<br/>　　“反正我现在每天在学校，也无聊的很，两个小时够不够？多占用几个小时也没关系的！”
<br/>　　叶尘顿时也一愣，没有想到萧若曦，竟然如此大方，换做寻常人，身体被人占用，恐怕早就吓的不行了。
<br/>　　“萧若曦，谢谢你！”
<br/>　　叶尘郑重道了一声谢。
<br/>　　萧若曦摆了摆手，
<br/>　　“我们是朋友嘛，你不用这么客气啦！”
<br/>　　说到这里，忽然摸了摸自己的嘴唇，自言自语道：
<br/>　　“奇怪，我的嘴怎么肿了？难道是上火了？”
<br/>　　“额……”
<br/>　　听到这话，饶是叶尘八百年老怪物的心志，也不由得老脸一红。
<br/>　　原来，方才他和曦月亲热的时候，因为太用力了，所以就把她的嘴唇给生生亲肿了……
<br/>　　这种事情，叶尘肯定是不能说出来的，连忙顾左右而言他，转移话题道：
<br/>　　“那个，你昏迷这么长时间，也该饿了吧？走！我请你吃饭！”
<br/>　　……
<br/>　　两人办了出院手续之后，便离开了医院。
<br/>　　天海第一中心医院，离天海大学并不算远，只有两站公交车的路程。
<br/>　　两人出了医院之后，走着走着便已经回到学校附近。
<br/>　　“前面有一家烧烤店，手艺很不错，我们去尝尝？”
<br/>　　萧若曦忽然提议道。
<br/>　　叶尘自然没有什么意见，说起来，他都已经很久没有吃地球上的食物了。
<br/>　　又向前走了数百米，两人便已经来到萧若曦所说的那家烧烤店。
<br/>　　看的出来，这家烧烤店的生意很好，而且靠近学校，很多在这里吃烧烤的，都是学生模样，应该也都是天海大学的学生。
<br/>　　两人在角落里，找到一处空着的座位坐下，烧烤店的小哥走了过来，递给了两人一个菜单，
<br/>　　“两位想吃点什么？还有各种饮品，随便点！”
<br/>　　萧若曦显然是这样的常客，当即拿起笔，在那张菜单上一阵勾画，然后交给了那小哥。
<br/>　　大约等了二三十分钟，萧若曦点的烧烤，终于做好端了上来。
<br/>　　“对了，若曦，大一新生是不是也该入学了？”
<br/>　　两人随口闲聊着，叶尘忽然开口问道。
<br/>　　萧若曦虽然奇怪叶尘为什么会问这个问题，不过还是老老实实回答道：
<br/>　　“嗯，大一新生入学，一般都是在每年的九月一号，今天是八月二十七，没几天了！你问这个做什么？”
<br/>　　叶尘淡淡一笑，
<br/>　　“我想上大学！”
<br/>　　“噗！”
<br/>　　萧若曦刚刚喝了一口可乐，直接狂喷了出来！
<br/>　　多亏叶尘反应快，身体微微一侧，便从容避开。
<br/>　　这也难怪萧若曦反应会这么大！
<br/>　　叶尘是何等身份？
<br/>　　华夏和平时代的首位元帅，凭一己之力，碾压了岛国、东南亚六国，还有当世第一强国的米国！
<br/>　　这样的人物，居然要上大学？
<br/>　　恐怕任谁听了，都会觉得不可思议。
<br/>　　“我没有听错吧？你你……你要念大学？”
<br/>　　叶尘咧嘴一笑，
<br/>　　“怎么？我就不能上大学了吗？”
<br/>　　“可是，你好像都没有参加高考吧？没有分数，你怎么上大学？”
<br/>　　萧若曦说完这话之后，就意识到自己错了。
<br/>　　以叶尘今时今日的地位，莫说是进入天海大学读书，便是做天海大学的校长，那还不是一句话的事情吗？
<br/>　　果不其然，
<br/>　　叶尘左手一翻，已经从乾坤镯里，取出了一只手机，然后拨通了易山河的电话，
<br/>　　“叶帅，您有何吩咐？”
<br/>　　易山河的语气，比之前恭谨了不少。
<br/>　　之前，叶尘的军衔虽然比他高，但他的资历要深的多，所以两人在军中的地位，相差并不算太多。
<br/>　　可是如今，叶尘从上将，一举提升到了元帅，比君老的军衔都要高出一截，是绝对的华夏第一人，他自然不敢再像之前那么随便了。
<br/>　　叶尘也没有废话，直接道：
<br/>　　“老易，你帮我安排一下，我进入天海大学读书，把我的高考分数设为581分，报考的专业是生物工程……”
