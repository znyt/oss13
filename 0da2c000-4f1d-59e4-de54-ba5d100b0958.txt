<br/>				<br/>　　郭信宁来到凤九的身边，微侧头过看着这个比他矮半个头的少年。<br/>　　<br/>　　“我在谢府中只是下人，自然不能乱了规距。”<br/>　　<br/>　　“可我看你，一点也不像下人，反倒是比玉堂更像主子。”他盯着他看着，仿佛想从他的脸上看出什么来。<br/>　　<br/>　　凤九轻笑着：“那是因为他随意，没有主子的架子。”<br/>　　<br/>　　“是吗？”明显的，他不太相信。<br/>　　<br/>　　两人没再说话，只是慢慢的在周围走着，观察着周围的动静。良久，郭信宁又打破了沉默：“为何你们面对群狼会没受伤？”这个问题在看到他们半点伤口也没有时，便想问了，忍到现在，终于还是问出声来。<br/>　　<br/>　　听着这话，凤九微侧头看了他一眼，脸上带着惊讶：“郭公子很希望我们受伤？”<br/>　　<br/>　　没想到会听到他这么问的郭信宁微愕，他深深的看了凤九一眼，摇了摇头：“不是，只是有些疑惑。”<br/>　　<br/>　　“这有什么好疑惑的？正是因为有你们保护着我们，我们才没受伤啊！”她说得理所当然，仿佛事实就是如此一般。<br/>　　<br/>　　若不是郭信宁有看到当时的场面，知道郭家人不可能在那样的情况下还能护得住他们，真的就要相信了他的话。<br/>　　<br/>　　问不出什么，他也没再问，只是静静的巡着夜，只是，当夜色渐深，周围越发寂静时，他却不由的停下脚步来，看向森林的暗处。<br/>　　<br/>　　“怎么了？”凤九也停下脚步来，询问着。<br/>　　<br/>　　“很奇怪。”郭信宁说着。<br/>　　<br/>　　凤九眸光微动，问：“什么奇怪？”<br/>　　<br/>　　“狼群的血，郭家子弟受伤的血，弥漫在这里久久不散，血腥味如此重，这周围却没有凶兽再出现，有些不合常理。”郭信宁沉思着，想不出是什么原因让这深夜变得平静。<br/>　　<br/>　　但凡有一点在外历练经验的人都会知道，在危险重重凶兽出没的林中，尤其是夜色之下的血腥味更会引起凶兽的躁动，可偏偏，除了那先前与他们一战的狼群之外，这里的血腥味居然没有引来其他的凶兽。<br/>　　<br/>　　这一点，有些不合常理，因为他原以为，今夜必定又会是一场苦战，没想到却是这般的平静。<br/>　　<br/>　　闻言，凤九轻笑着：“你一向都想那么多的吗？没有凶兽再来不是好事？”<br/>　　<br/>　　她的上古威压是往外散发出去的，再加上这郭信宁的实力并不算太强，自然察觉不到有她威压的存在。<br/>　　<br/>　　“对了，从这里面出去，大概得多久的时间？”她还放了丹药在那易宝楼里呢！说了七天后去拿交换物的，若是一直困在这里到时只怕会错过了。<br/>　　<br/>　　“在这地方有我们家族布下的传送阵，只要到了那里就能离开这万毒林了。”<br/>　　<br/>　　郭信宁说着，眉头微拧：“只不过，如今众人都受了伤，而且我看大长老也似乎伤得不轻，这一路只怕还会遇到不少凶兽袭击，想要翻过那座山去到传送阵的话，估计还得花些时间。”<br/>　　<br/>　　听到这话，凤九心头微动。传送阵？看来，阮家在这里面也是布下有传送阵的。<br/>
