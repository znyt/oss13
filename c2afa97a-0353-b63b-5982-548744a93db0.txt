　　叶尘的身影，在海面上陡然消失不见。
<br/>　　下一刻，
<br/>　　他只感觉眼前虚空一阵变幻，已经穿越结界，进入到了另外一方世界之中。
<br/>　　和前两次，进入妖界和魔界的情况不同，这一次，他竟然出现在了一座高台之上！
<br/>　　而且，周围是一排排高大古朴的建筑，这里显然是一座城池。
<br/>　　就在叶尘正暗自纳闷之时，旁边陡然响起一道威严的声音，
<br/>　　“小子！你是哪个门派的弟子？把通行令牌出示一下！”
<br/>　　说话的，是一名三十来岁的魁梧中年男子，满脸的凶煞之色，身后还跟着一群人，个个气势不弱，至少也都是筑基境中期以上的修为。
<br/>　　叶尘顿时一愣，
<br/>　　“什么通行令牌？”
<br/>　　那魁梧中年男子，顿时两眼一瞪，
<br/>　　“但凡我蓬莱仙界弟子，出入世俗界，都需要门派申请通行令牌，难道你连这个基本的常识都不知道吗？”
<br/>　　叶尘顿时恍然，
<br/>　　很显然，蓬莱仙界和妖、魔两界的规矩并不一样，妖、魔两界完全禁止族人进入世俗界，而蓬莱仙界则是只是半封闭状态，只要申请通行令，便可以自由出入。
<br/>　　一念及此，叶尘当即放出神念，在乾坤镯里扫了一遍。
<br/>　　他之前杀了公孙林，随手便将他的乾坤镯给收了，都还没来得及探查。
<br/>　　“小子！本监察官在问你话，你磨磨唧唧什么？还不快点把通行令牌拿出来！”
<br/>　　就在那魁梧中年，眼看已经对叶尘起了疑心，就要准备将他拿下的时候。
<br/>　　叶尘也终于在公孙林的乾坤镯中，找到了一枚黝黑的令牌，于是心念一动，便取了出来，递了过去，
<br/>　　“你说的是个东西吗？”
<br/>　　他初入蓬莱仙界，还没有搞清楚这里的状况，能不招惹麻烦，还是尽量先不要招惹麻烦。
<br/>　　更何况，他现在的当务之急，是要尽快赶到药王宗，将苏蔓从那些人手里救出来！
<br/>　　当然，如果不能蒙混过关，那也只好动用武力硬闯了。
<br/>　　好在他运气不错，那中年魁梧男子接过那黝黑令牌，打量了两眼之后，便又丢给了他，嘴里嘟囔了一句，
<br/>　　“原来是药王宗的弟子，你们药王宗最近出入很频繁啊！”
<br/>　　这魁梧中年男子，似乎对药王宗很是忌惮，只是嘀咕了一句，也不敢继续追问，便直接冲叶尘摆了摆手，
<br/>　　“走吧！”
<br/>　　叶尘下了高台之后，这才来得及仔细观察周围的环境。
<br/>　　只见，这里的建筑，倒是和华夏古代毕竟相似，不过这里的人，穿着打扮，却形形色色，竟然有很多外面的现代服饰，因而叶尘在人群之中，倒也一点也不显得突兀。
<br/>　　“难怪之前那几个人，这么轻易就相信了，我是药王宗弟子，看来这蓬莱仙界倒是与时俱进，和世俗界的联系，远远超过妖界和魔界，只不过世俗界极少有人知道罢了！”
<br/>　　想到这里，叶尘也彻底放下心来，
<br/>　　正打算，随便在大街上拉一个人，问一下药王宗所在。
<br/>　　就在这时，旁边忽然传来一道打骂声，
<br/>　　“妈的！没钱装什么大爷！敢在我们店里吃霸王餐，看老子今天不打断的你腿！”
<br/>　　叶尘循声望去，
<br/>　　只见，一名约莫十三四岁的少年，正在被一群人围殴。
<br/>　　这少年一身锦服，看上去倒也不像是穷人，只是有些狼狈，衣服上还有血迹残存。
<br/>　　而真正让叶尘感到诧异的是，这少年年纪轻轻，居然已经突破到筑基境！
<br/>　　这在修真界万千星域中，或许不算什么，可是在一个小小的地球上，如此年轻就有如此境界，绝对是一个修炼天才，如果能遇到名师，将来成就必定不凡。
<br/>　　一念及此，叶尘顿时起了爱才之心，
<br/>　　眼看那群人就要对他下死手，叶尘当即走上前来，
<br/>　　“住手！”
<br/>　　那群人顿时朝叶尘望了过来，
<br/>　　看到他一身现代服装打扮，众人顿时纷纷露出一抹敬畏之色。
<br/>　　原来，在蓬莱仙界，现代服饰属于稀有商品，价格很高，一般人根本穿不起。
<br/>　　原因很简单，因为普通人是很难出入世俗界的，只有五大宗门和两大商会，才具有自由出入世俗界的权利，而且同样有诸多限制。
<br/>　　因而，现代服饰在蓬莱仙界中，属于稀有商品，一般只有上层社会的人才穿的起。
<br/>　　再加上，叶尘本身就气度不凡，这些人自然不敢轻易得罪。
<br/>　　“阁下是什么人？这小子在我们酒馆里吃霸王餐，难道我们不该教训他一下吗？”
<br/>　　为首的一人，冲着叶尘一拱手，客客气气道。
<br/>　　叶尘眉头微微一皱，
<br/>　　“他欠你们多少钱？我替他付了！”
<br/>　　那人听到这话，脸上顿时浮现出一抹笑容，
<br/>　　“不多不多，也就一两银子而已！”
<br/>　　叶尘顿时一愣，
<br/>　　银子？
<br/>　　他还真没有东西！
<br/>　　不过很快，他从公孙林的乾坤镯中，倒是找到了一些灵石。
<br/>　　‘这里也算是半个修真界，灵石怎么着都应该银子珍贵吧？’
<br/>　　一念及此，叶尘直接取出一块灵石，丢给那人，
<br/>　　“这个够不够？”
<br/>　　那人先是一愣，继而满脸的狂喜之色，双手捧着那块灵石，激动的都有些语无伦次了，
<br/>　　“够了！够了！太多了！公子您稍等，我这就给你找钱！”
<br/>　　叶尘摆了摆手，
<br/>　　“不用找了！我向你打听一件事，只要你能回答我，剩下的钱就是你的！”
<br/>　　那人顿时两眼放光，连忙点头哈腰道：
<br/>　　“公子您请问，只要是我常老三知道的，我一定知无不言，言无不尽！”
<br/>　　叶尘点了点头，淡淡道：
<br/>　　“这个问题其实很简单，你可知道药王宗？”
<br/>　　那人连忙陪着笑道：
<br/>　　“公子说笑了，药王宗乃是我蓬莱仙界五大宗门之一，谁人不知啊？”
<br/>　　叶尘道：
<br/>　　“那药王宗在什么地方，你可知道？”
<br/>　　那人顿时有些懵了，呆呆道：
<br/>　　“自然是在药王谷，公子您不是在说笑吧？这个连三岁小孩都知道啊！”
<br/>　　叶尘也顿时微微有些尴尬，
<br/>　　“这个我自然知道，我的意思是，你们谁要是能带我去药王谷，这个便是他的！”
<br/>　　说完这话，叶尘随手一翻，当即又取出了三块灵石。
<br/>　　众人看到叶尘手中的灵石，顿时眼神全都变得火热起来，不过很快却又纷纷露出一抹为难之色。
