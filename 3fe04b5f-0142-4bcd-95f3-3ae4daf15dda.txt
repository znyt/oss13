　　意大利，罗马，罗马竞技场。
<br/>　　这里曾经是古罗马帝国专供奴隶主、贵族观看斗兽或奴隶角斗的地方。岁月在这座恢弘的古建筑上留下了深深的印迹。
<br/>　　这座竞技场现在成了著名的旅游景点，来罗马的旅行团都会将这里作为必选的参观景点，而这里也会安排斗兽表演，让游客坐在这座恢弘的竞技场中，去感受古罗马帝国奴隶主和贵族观看角斗的体验，只是现在安排的斗兽表演远没有古罗马帝国时的血腥。
<br/>　　即便没有了曾经的血腥和暴力，可是依旧能让观众感到热血沸腾。
<br/>　　韦斯利正带着他的女朋友斯黛拉刚看完一场斗兽表演，在他们旁边的座位上，还坐着一个十八九岁的男孩子，标准了亚洲人外貌，平静的眸子中充满了冷静之色，就算是刚才表演中，斗兽师遭遇到危险，引来周围观众的阵阵惊呼，也没有使得他的眼神发生波动。
<br/>　　年轻的脸上，带着与他年纪不符的成熟，兜帽衫的帽子扣在他的头上，帽子下，他的眼睛不时警觉的向竞技场四周扫去，他的双手戴着一副黑色的皮手套，遮住了他双手上的伤痕。
<br/>　　这个十八九岁的孩子，就是布克尔特派出去的暗杀小组带回去情报中出现的未知人物。
<br/>　　他叫布谷，代号“布谷鸟”，除了是“龙牙”最年轻的佣兵，他还有一个身份，他是夜鹰的一手打造出来的接班人，夜鹰自己的接班人。
<br/>　　布谷是中国人，他还有个妹妹叫布丁，为了给自己患有先天性心脏病的妹妹筹集医药费，只有十岁的布谷去黑拳市场打黑拳被夜鹰发现，夜鹰把布谷带出了黑拳市场，并且把他的妹妹一起接到了“龙牙”。
<br/>　　夜鹰年纪大了，他不能再像以前那样敏捷的做出各种战术动作，完成各种高难度的狙杀，所以他需要找到一个合格的接班人，布谷的反应能力、身体素质还有运动天赋都让夜鹰满意。
<br/>　　云凌风出钱帮助布丁接受了心脏移植手术，布谷也顺利的成为了“龙牙”这个大家庭中的一员，成为了夜鹰的学生。布谷在刚加入“龙牙”的时候，因为营养不良而显得很瘦弱，经常被“龙牙”中的其他队员开玩笑，可是当夜鹰带着布谷离开“龙牙”总部完成全部训练课程再回来的时候，布谷就像是完成了一个蜕变。
<br/>　　看到重新返回“龙牙”的布谷，云凌风第一时间就觉得自己眼前这个孩子变得，他的眼睛变得更加平静，可是在这种平静背后却是有种叫做“锐利”的东西，刺破这种平静透露出来。如果说布谷在刚来到“龙牙”的时候只是一块好钢，那么在他被夜鹰带走并且重新返回的时候，他就已经变成了一把利剑。
<br/>　　韦斯利从座位上站起身来，手很不老实的在斯黛拉的屁股上捏了一下；斯黛拉立刻回给韦斯利一个妩媚的眼神，然后迈步朝着厕所的方向走去，得到斯黛拉给出进攻信号的韦斯利迈步跟了上去。
<br/>　　“喂，你是种马吗？”
<br/>　　韦斯利和斯黛拉之间的眉目传情虽然隐秘，但是却并没有能瞒过布谷的眼睛，冲着韦斯利说道：“昨天晚上折腾了一晚上还不够吗？”
<br/>　　“喂，你明明睡在我们隔壁的房间，为什么会知道这些？”韦斯利眼神不善的看着布谷，说道：“你是不是来偷听了？”
<br/>　　“就你们这点事情还需要偷听吗？是你们叫得太大声了好吗？”布谷撇嘴，瞥了眼韦斯利说道：“就算是我偷听了，你又能怎么样，你打得过我么，老游骑兵？”
<br/>　　韦斯利是“第75游骑兵团”退役再加入的CiA，最后兜兜转转加入了“龙牙”，他也有自己的代号，可是布谷却总是喜欢叫他“老游骑兵”，这让得韦斯利很不爽，但是却又拿布谷无可奈何。布谷刚来“龙牙”的时候，韦斯利还能在几招之内就制服他，可是现在韦斯利是真的打不过布谷，天知道夜鹰是怎么训练布谷的，早就出这么一个变态出来。
<br/>　　韦斯利不爽的哼了一声，转身朝着厕所的方向走去，而布谷则是又重新坐回到了座位上，他可没有跟着韦斯利去厕所，然后听他和斯黛拉做那事的嗜好。
<br/>　　布谷刚坐下，就注意到对面看台上的三个男子朝着厕所的方向走去。这三个男子从布谷进入竞技场的时候，就注意到了他们，他们跟自己这边几乎是前后脚进入到了竞技场，他们有意无意的不断朝着自己这边进行观察，这也是布谷在斗兽表演中始终保持着警惕观察着四周的原因。
<br/>　　其实从进入意大利的时候，布谷就察觉到了有人在跟着自己，只是对方跟踪很专业也很隐秘，甚至瞒过了韦斯利这个老游骑兵。韦斯利本身的警觉性其实并不差，但这是在他身边没有女人的前提，如果他身边有女人，尤其是像斯黛拉这种年轻、貌美、身材又火辣的美女，韦斯利的注意力就全都在女人身上了。
<br/>　　本来布谷还觉得自己跟着韦斯利来意大利简直就是脑残行为，他家伙来这里就是做牲口的，自己来这里就是一个大瓦电灯泡；但是现在布谷却觉得自己来这边来对了。
<br/>　　看到那三个男子走向厕所，布谷也跟着站起身从朝厕所的方向走去。
<br/>　　厕所里此时并没有什么人，韦斯利和斯黛拉也在一间隔间里“开战”了，虽然隔间里的两个人刻意的压低了声音，但是只要走进厕所，还是能立刻明白在隔间里发生了什么事情。
<br/>　　厕所门口三个男子通过隔间里发出的声音，很快就确定了韦斯利的位置，就在三个人相互之间传递了一下眼神，准备动手的时候，布谷从后面挤开了三个人，走到小便池前，假装方便。实际上，布谷的手已经摸向了挂在牛仔裤上的战术折刀，如果这三个男子执意要进入厕所，并且要有所动作的话，布谷是不介意在这里动手的。
