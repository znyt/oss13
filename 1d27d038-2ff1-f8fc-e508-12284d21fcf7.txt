　　“飞坦小心，这个家伙不简单。”侠客少见的收起脸上嬉笑，很是郑重的提醒。
<br/>　　赏金猎人并不全是职业猎人，或者说大多数所谓的赏金猎人，只不过是一些武力团体。很多不是念能力者，多依靠火器。
<br/>　　但他负责幻影旅团的情报，比其他人感触更深刻。知道那一千人，大都是念能力者。不会念的，已经被原始森林给淘汰。当初能够走到苏重面前的，都有一手绝活。
<br/>　　可就是如此，却依旧近乎被全灭。
<br/>　　飞坦眼皮抬都没抬，抽出别在后腰上的细长雨伞。身影一阵模糊，瞬间消失在原地。
<br/>　　下一刻，陡然出现在苏重身前。手中雨伞猛然刺向苏重胸口，锋利尖端闪烁着凛冽寒光。
<br/>　　苏重神色淡然，安静漠然看着前方：“速度不错。”
<br/>　　叮！
<br/>　　一点火星蹦起，不知何时，苏重的右手已然挡在胸前。
<br/>　　雨伞尖端定在苏重掌心，可任凭飞坦如何发力，竟丝毫不能刺破手掌！
<br/>　　他的雨伞材料特殊，足可抵抗数吨重力压迫。即便是窝金，也无法从容抵挡雨伞攻击！
<br/>　　没感觉到念力，这种身体强度，是怪物吗？飞坦心头一沉，不敢大意。身形一闪，就要从另一方向发动攻击。
<br/>　　“想走？可没那么容易！”
<br/>　　右手猛的攥紧雨伞，顺势一掰。
<br/>　　啪！
<br/>　　用独特技术打造，坚硬无比的雨伞，一下就被苏重折成了两段。
<br/>　　不等飞坦做出反应，苏重身影瞬间像出现在飞坦身前。右手握拳炮弹一般打出。
<br/>　　砰！
<br/>　　飞坦整个人弯成一个虾米状，好似腰间帮了一根绳索，被猛地向后拖拽而去。
<br/>　　落在地上，抱着肚子竟然站不起来。
<br/>　　本来还有些看戏心思的旅团成员，再也没有丝毫轻松心态。
<br/>　　飞坦实力强劲，可刚冲上去，就被原样打回来，竟一合都没撑下来！
<br/>　　“可恶！竟然敢打伤我。”飞坦一脸的扭曲，血水口水混合着从嘴角流下，眼中冰冷杀气，几乎让空气凝结。
<br/>　　“喂，飞坦，你冷静一点儿！别乱来！”侠客脸色大变。
<br/>　　飞坦一旦陷入暴怒，立刻就会爆发出强力攻击，可那攻击根本就敌我不分好吧！
<br/>　　“快走！”
<br/>　　几乎下一刻，芬克斯瞬间出现在信长身边，扛起信长转身就跑。
<br/>　　侠客顺势抓住窝金胳膊，也不管对方是不是重伤，拖死狗一样夺路狂奔。
<br/>　　“去死吧！”
<br/>　　几乎在旅团成员刚刚开始动身的瞬间，飞坦一声大喝。
<br/>　　强大的念力好似绝地闸口一般，从那矮小的身体中猛然爆发而出。
<br/>　　几乎是眨眼间，一身黑红古怪衣服已经把他包裹起来。下一刻，头顶好似出现了一个充满吸力的黑洞，飞坦的所有念力全部飞入其中。
<br/>　　嗡！
<br/>　　空气一震嗡鸣，无形中波纹迅速飞散而去。
<br/>　　整个树林好似被惊动，一种大难临头的感觉，瞬间降临无数活物脑海。
<br/>　　短暂寂静之后，整个森林暴动起来。所有活物全都不要命的往外跑。
<br/>　　“开玩笑的吧！这么强大的波动？飞坦这是要疯啊！”芬克斯一声怪叫，好似屁股后面被狼咬了一口似得，撒开丫子跑得更快啦。
<br/>　　“飞坦很生气，后果很严重！”木讷的弗兰克利好似一个坦克一般，一步迈出震的地面直发抖。
<br/>　　“死啦！死啦！要死啦！飞坦越生气，念能力的威力就越大，快跑啊！跑慢了，我们就惨啦！”侠客咋咋呼呼，拖着牵扯伤口，快要被疼晕过去的窝金，呼呼狂窜。
<br/>　　库洛洛飞速奔跑，眼中闪过一道失望。能打赢窝金、信长，实力强大。他本来打算多收一个团员。
<br/>　　可惜，似乎对方对自己等人很了解，却不打算加入。
<br/>　　现在好了，有飞坦动手。对方死得连渣都不会剩。就算他知道旅团的情报，也不足为虑。
<br/>　　轰！
<br/>　　一颗直径足有两米的巨大火球，凭空出现在飞坦头顶，恐怖的热浪翻卷扩散。
<br/>　　几乎瞬间，就把周围的树木烤成了灰烬。
<br/>　　“嘿嘿嘿。去死吧！”
<br/>　　带着极致高温的火球，瞬间就冲道苏重头顶。立刻开始急剧膨胀。
<br/>　　苏重的头发，都被这恐怖高温炙烤的卷曲。苏重不仅没有紧张，反而露出古怪神色。
<br/>　　然后他伸出双手，好似拥抱天空一般，对着好似小太阳一般的火球。
<br/>　　生命虹吸！
<br/>　　呼！
<br/>　　巨大火球下方，突然出现一个凸起，然后越来越长，逐渐向下移动，下一刻竟然没入的苏重双手中。
<br/>　　好似破了洞的皮球，岩浆似的热流从破洞流下，灌入苏重体内。刚开始膨胀的火球，只是颤抖了两下变后继无力，随着岩浆河流逝逐渐变小。
<br/>　　只是几个呼吸的时间，刚才还恐怖异常的火球，竟全被苏重吸收！
<br/>　　嗤！
<br/>　　一股白眼从苏重身上升腾而起，遍布全身的鳞甲纹路变成了黑红之色，好似有岩浆在皮肤下流动一般。
<br/>　　“不可能！”
<br/>　　飞坦目瞪口呆。那可是个小型太阳。温度高达六千多度！钢铁扔进去也会被瞬间焚毁。他看到了什么，竟然有人把一个小型太阳吸进了身体里？！
<br/>　　“纯阳四段提升了生命虹吸，本以为只是加大了范围和力量。没想到，竟然提升了本质。不仅能够吸收植物体内的能连，竟然连火焰的能量也能吸收？真是意外之喜。”
<br/>　　苏重转头看向依然一副惊呆模样的飞坦，咧嘴一笑：“你的攻击让我发现了新能力，我很高兴。作为回报，接我一拳不死，我就不再追究幻影旅团。”
<br/>　　不等飞坦做出回应，苏重右手拳头陡然包裹上了一层白光。
<br/>　　暗杀拳·崩山！
<br/>　　轰！
<br/>　　一道五十米粗细的巨大光柱陡然爆发，极致的震荡毁灭力量疯狂爆发。
<br/>　　飞坦根本来不及反应，就被强大拳劲淹没。瞬间就被冲没了身影。
<br/>　　“不对啊，怎么还没爆炸？”侠客跑了一会儿，有些奇怪。
<br/>　　其他几人身影忍不住一顿。
<br/>　　“不应该啊，飞坦这招的爆发力一向强大，怎么过了这么久没有动静？”芬克斯不解道。
<br/>　　“会不会出事？”富兰克林木讷脸上闪过一道忧色。
<br/>　　“不会。飞坦这个能力攻防一体。他那身怪怪的衣服可结实的很，可不只是能防高温。”侠客轻松道。
<br/>　　飞坦能够局限出一个小型太阳，同时能具现出一身高防护的衣服。太阳爆炸伤人，衣服保护自己，攻防一体。
<br/>　　轰！
<br/>　　一声巨响陡然从身后传来，几人心中一松。这下不用担心了，终于爆炸了。
<br/>　　“不对啊，我怎么感觉声音越来越近？”芬克斯耳朵一动，一脸古怪。
<br/>　　“不好！”库洛洛脸色大变，猛然回头，就看到一道铺天盖地的飓风陡然淹没而来。飓风之中，一道道扭曲力量清晰可见。凡是被扭曲力量碰到的东西，立刻成了粉末。
<br/>　　危险！
<br/>　　几乎是本能的，他瞬间来到几个同伴身前。生拉硬拽的把几人拖到身前，浑身念力陡然爆发。
<br/>　　砰！
<br/>　　一声闷响，几人凭空消失在原地。
<br/>　　几乎在几人离开的瞬间，恐怖风暴席卷而来。所过之处，寸草不留，所有高过地面的东西，全被摧毁殆尽！
