        &lt;h3&gt;第三卷第二百九十九章  顾恩赐&lt;/h3&gt;
<br/>
<br/>        顾恩赐坐在自己豪华的办公室里，志得意满的看着手中的方案，这一次新城的项目已经内定给他了，自从上次一起吃饭之后，呼延临庆就找到他们几家，每个人都分配了一些项目，让大家齐心协力将这些项目都拿下来，肥水不流外人田，只要大家抱团没有做不成的事情。
<br/>
<br/>        他是新城，宋长生是高新，许弋澄是莲余，呼延临庆塔纳，目前要一起动的项目都在他们的手中，如果说他们最大的竞争对手是谁，那就是陈子昂，陈子昂不但财力雄厚而且技术过关，所有的技术都是他们自己研发的，而他们也只是代理。
<br/>
<br/>        如果说他们单打独斗谁也不是陈子昂的对手，但是几家联合起来那就不一样了，他们只要在西城占领一片天空就好了，陈子昂全国都有分公司，你可以去别的地方啊，何必要在西城和他们抢饭吃？
<br/>
<br/>        顾恩赐之前和陈子昂一直也有业务上的来往，而且两家公司离得不远，前后楼，陈子昂公司有什么风吹草动他基本上都能知道，而且两家的库房都是在地下车库旁边的负一层。
<br/>
<br/>        他在it一条街上好多年了，这里的一草一木他都清楚，公司的规模不大也不小，养了十几口人，代理了一些产品，日子过得也是有滋有味。
<br/>
<br/>        他的老婆是某系统的小老总，他们夫妇又用妻弟的名义开了一家大型的烟酒超市，代理了好几个品牌的高档烟酒，地址在红砖路上，在红砖路上，不，是在西城，他们都是响当当的人物，因为妻子的原因，很多人都得到他们家来批发一些商品，生意做得红红火火。
<br/>
<br/>        看着自己家的标书已经做好了，他想去宋长生和许弋澄哪里走走，看看他们的标书做得怎么样，虽然有把握拿到项目，如果标书做得不好的话，也是白搭，因为也有一些专家是死脑筋，他们是很较劲的，有一点失误都会被他们找到，然后都会给你废了，尤其是易倾秋还放言一定要拿到新城的项目，这就不得不让他更要提高警惕了。
<br/>
<br/>        顾恩赐准备先去宋长生那里，宋长生离他还有一段距离，顾恩赐下楼取车的时候，发现陈子昂的库房门口有人在卸酒，好十几箱，陈子昂拿那么多酒干什么？
<br/>
<br/>        难道陈子昂要结婚了准备办酒宴？这几天陈子昂确实是在西城，听说她爸生病做手术，难道是她爸爸不在了？
<br/>
<br/>        顾恩赐站着看了一会，忍不住走到陈子昂的库房门前，朝里面看了看，库房里面还堆了不少，买这么多酒干什么？
<br/>
<br/>        “哎，你们家有什么喜事吗？用这么多酒？”
<br/>
<br/>        “哇，顾总啊，我们也不知道要这么多酒干什么，陈总让拉回来的。”一个正在清点货物的人看见是顾恩赐笑着说。
<br/>
<br/>        “买这么多酒，肯定是有大事情发生，要不然总会用这么多酒。”
<br/>
<br/>        “也许吧，我们也不知道干什么。”
<br/>
<br/>        顾恩赐带着疑虑走了。
<br/>
<br/>        顾恩赐到了宋长生的公司，宋长生不在，顾恩赐想看方案一看不成，于是就又转到了许弋澄的公司。
<br/>
<br/>        许弋澄最近在延城有项目，延一延二的项目都被他拿下来了，许弋澄觉得他手中的软件部分有些瑕疵，不利于后期的发展，他还想完善一下。
<br/>
<br/>        他最近正在苦恼用那家的做参考，呼延临庆是个半吊子，洛城的项目到现在都搞不定，国外的又不太适合，国内也有几家，但是他又怕后期服务是个问题，他想长久发展，就要做好自己的品牌，软件是他的硬伤。
<br/>
<br/>        顾恩赐到了许弋澄的公司，许弋澄正在开会研究软件的问题，研究来研究去也没有让许弋澄感觉满意，刚好顾恩赐到了，许弋澄也就结束了会议。
<br/>
<br/>        俩个人坐下后，顾恩赐说：“看你愁容满面的，是碰到什么难题了吗？不如说出来我听听，也许能帮到你。”
<br/>
<br/>        “就你眼前的水准你帮不了我的。你是来看新城的方案吧？放心我们的方案不会有什么的，我到时候价格是拦标价。”
<br/>
<br/>        “嗨，你这话说得我就不爱听了，再怎么说我们现在都是相互扶持的人了，有什么说出来，大家想想办法也许能帮忙啊。”
<br/>
<br/>        “延一和延二的项目，我觉得我手中的软件缺点东西，多少还是有点瑕疵的，想找个厂家对比一下，或者糅合一下，看看能不能更好一些。”
<br/>
<br/>        “要是软件部分，说实话我还是佩服陈子昂他们的，你看看他们做的项目，现在哪一个不是走在前端，呼延那个半吊子问题多的很，你千万不要用他们的，后期会坑死你的。”顾恩赐大大咧咧的说。
<br/>
<br/>        “啊？”许弋澄心里一惊，他怎么就没有想到陈子昂？
<br/>
<br/>        “不是，我不是说呼延临庆的坏话，我是就事论事，你想用了也不是不可以，你当我没有说过这样的话。”顾恩赐觉得自己说错了话。
<br/>
<br/>        “没事，顾兄你这话也就只能在我这里说说，别人你就不要这样说了，毕竟我们现在是在抱团取暖，哈哈哈！”许弋澄大笑。
<br/>
<br/>        顾恩赐倒是提醒了他，陈子昂有好几个现成的项目可以借鉴，他为什么不去问问陈子昂呢？
<br/>
<br/>        他在心里一直是排斥陈子昂的，因为陈子昂很骄傲从来都没有将他放在眼里，不过现在想一想人家有骄傲的资本，上学是学霸，现在就更不用说了。
<br/>
<br/>        许弋澄到现在心里还是藏着秘密的，偶尔在睡梦里会出现年少时陈子昂那欢快的笑声，那个时候的他特别喜欢陈子昂，知道夏颜和陈子昂是好朋友，便偷偷的给陈子昂写信，让夏颜交给陈子昂，可是陈子昂每次通过夏颜约他的时候对他冷冷淡淡的，好像根本就不认识他一样，这让他的心里很受挫，加上夏颜热情似火，年少的他和夏颜一来二去就热络起来了。
<br/>
<br/>        他每次和夏颜约会，陈子昂都在旁边远远的看着，他以为陈子昂会伤心会难过，可是陈子昂什么反应都没有，过了不久，夏颜和陈子昂闹掰了，他也就再也不能和陈子昂一起了。
<br/>
<br/>        他开始和夏颜一起逃课，玩乐，原本学习很好的他，成绩一落千丈，最后只落得考一个二本院校，而陈子昂连二本也没有考上，高中之后陈子昂从他的眼前彻底消失了，从上大学之后，他和夏颜的来往也就淡了下来，不过许弋澄也算是个痴情的人，既然和夏颜在一起，他就想要和她过一生，大学毕业就想回西城发展，夏颜却选择了傅海生。
<br/>
<br/>        这么几年，他一直不怎么联系夏颜，也排斥和陈子昂来往，不过顾恩赐倒是提醒了他，陈子昂的经验是值得借鉴的，为了今后的发展，他应该向陈子昂请教。
