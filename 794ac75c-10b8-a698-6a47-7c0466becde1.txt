        不过，对于傅夜沉宠女儿的这件事情。

<br/>        童筱颖忍俊不禁道：“没办法，大哥上辈子，一定是有情债没还，所以老天爷才会派星儿来治他。”
<br/>
<br/>        另一边，市区某个高档的网咖门外的奶茶店前，两个女生正鬼鬼祟祟地往一杯冰激凌奶茶里加料。

<br/>        两个女生彼此交换了一个眼神，故作淡定地将奶茶透明拱形盖重新盖好，然后携手进了网咖的大门。
<br/>
<br/>        她们后脚刚进去，此时，一个白衣黑裤，身段颀长，碎发利索，背影帅气的男生也跟着走了进去。
<br/>
<br/>        “星儿，你的冰激凌奶茶来了！”女生声音娇软，将手里的奶茶朝连星儿递了过去。
<br/>
<br/>        连星儿刚赢了一局游戏，于是从椅子上起身，走出雅座，撑了个懒腰，活动一下双手的筋骨。
<br/>
<br/>        “星儿，你打游戏比我们班那群男生还要厉害！”另一个女生笑里藏刀地夸赞。
<br/>
<br/>        连星儿得意洋洋地笑道：“我可是拜了‘大神’当师父，名师出高徒嘛！我自然是超级厉害！”
<br/>
<br/>        “来来来，你的冰激凌奶茶，再不吃的话，就要化掉了！”女生连忙催促道。
<br/>
<br/>        连星儿这才去接那杯冰激凌奶茶。
<br/>
<br/>        可结果，她的奶茶刚拿到手，还未握稳，后背突然传来一道力度推了她一下。
<br/>
<br/>        “哗”地一声，冰激凌奶茶直接泼到了面前这个递奶茶的女生的胸上。
<br/>
<br/>        女生“啊啊”地尖叫了起来。
<br/>
<br/>        连星儿连忙将空奶茶杯放下，一边抽起桌上的纸巾递给女生，一边四下看看到底是哪个“混蛋”不小心撞了她。
<br/>
<br/>        只见一个看上去很清瘦，身段修长，穿着白色衬衣，衣摆扎入了黑色笔挺的休闲裤里，宽肩窄腰，臀部微翘，背影十分迷人的男生刚好路过。
<br/>
<br/>        “喂，你给我站住！你撞到我了，害我泼了我朋友一身冰激凌奶茶，你居然连句道歉的话也没有！”连星儿顿时恼火地喊了一声。
<br/>
<br/>        男生也因此停下了脚步，淡定地转过身来。
<br/>
<br/>        而他这一转身，不仅让一旁的两个女生看得眼睛发直，就连连星儿也情不自禁地瞪大了眼睛。
<br/>
<br/>        男生五官俊挺有型，绝不输给荧幕上任何一个偶像男明星。
<br/>
<br/>        就连她的爱豆洛羽都没这男生帅！
<br/>
<br/>        尤其是男生那双眼睛，除了是好看的双眼皮类型以为，还特别的亮，就像天上的星星一样。
<br/>
<br/>        怎么能有男生长得这么好看呢？
<br/>
<br/>        两个女生忍不住在心底泛起了花痴。
<br/>
<br/>        只有连星儿第一个回过神来，走上前去找男生说理：“你得给我和我的朋友道歉！”
<br/>
<br/>        男生此刻脸上面无表情，淡漠地瞥了一眼那边的两个女生后，又回眸垂下眼帘睨着站在自己面前的这个女孩。
<br/>
<br/>        “滚。”
<br/>
<br/>        “？！”连星儿一脸懵。
<br/>
<br/>        男生冷酷到几乎不近人情。
<br/>
<br/>        “你撞到我，一句道歉没有也就算了，竟然还叫我‘滚’，你脑子有病吗？”连星儿缓过神来后，气急败坏道。
<br/>
<br/>        男生睇了连星儿一眼，然后抬起手来看了看腕表上的时间，接着二话不说地转身大步流星地离去。
<br/>
<br/>        连星儿连忙跟上去，却只见那男生进了网咖里的一张“闲人免入”的房门，当她走过去时，房门已经自动关上了。
<br/>
<br/>        她连忙去压手柄，门却已经被反锁了。

<br/>
<br/>        连星儿气愤地在门上踹了一脚，这张深青色的防盗门上瞬间就留下了她的脚印。

<br/>        两个女生在旁看着，面面相觑后，一句话也不再多说。

<br/>        连星儿这才想起自己的正事来。

<br/>        她在家里可不敢玩这种5V5的电脑竞技游戏，因为要是被爹地发现了，只怕是上十万的电脑都会被爹地给砸了去，所以，她才不得不跑出来偷偷地玩。
