　　第495章
<br/>　　程大雷这句话说完，张肥和阿喜都耸拉着脑袋抬不起头来。
<br/>　　他们不知道程大雷这句『好厉害』是说那神秘潜入的刺客，还是说那两条被人飞石砸石的恶犬。反正不会是说喜字队或者张字军。
<br/>　　“大当家放心，我定把那刺客揪出来，交给大当家审问。”张肥咬牙切齿。
<br/>　　“全城搜寻，她定然逃不脱，就算掘土三尺，也要把她找出来。”阿喜道。
<br/>　　二人磨拳霍霍，定要发动人手，把全城的地皮洗一遍，也要将那刺客找出来雪耻。
<br/>　　“先等等。”
<br/>　　程大雷倒是没有太着急，该来的终究会来的，总该想个办法应对才是。那聂隐娘不是一般人，用一般手段定然也对付不了她，程大雷估计全城搜寻难免会兴师动众，最后的意义怕也不大。
<br/>　　“大当家莫非已有了主意？”阿喜问。
<br/>　　“还没有，不过现在首先要做的……”程大雷顿了顿，道：“先给这房子加固一下。”
<br/>　　程大雷拿出的办法，实在令阿喜和张肥有些无语。他画出图纸，由工匠坊加工出铁栏，把屋子的四面墙壁都加固。包括窗户，门口，如果拆掉墙，程大雷相当于住在一个铁笼子内。程大雷已经打定主意，自己足不出户，每天的饭菜让人隔着门缝送进来。
<br/>　　在施工的工程中，程大雷坐在房内静坐不动，他身穿照月蛤蟆甲，腰上佩着匹夫剑，鬼面斧放在一手就能抓到的位置。
<br/>　　“大当家，有必要如此么？”
<br/>　　徐神机站在外面，隔着窗户看着里面的程大雷。
<br/>　　“小心谨慎一万年都不嫌长，疏忽大意一次就够了。”程大雷安坐不动，想要平安活下去，就得牺牲些什么。
<br/>　　现在程大雷牺牲的东西就是尊严。
<br/>　　“那大当家打算这样躲多久，总不能一直躲在里面吧？”徐神机。
<br/>　　程大雷叹了口气，至于要躲多久，其实他心里也不清楚。
<br/>　　程大雷遇刺的事情，山寨大部分人都知道了。对于这件事，大家不得不谨慎，程大雷需要的房屋加固，在一天之内便完成了。
<br/>　　以城主府为中心，加强了防守，纵然程大雷所没必要全城搜寻，但这件工作也不可能不做。只不过没有必要搞得那么兴师动众。不过，也的确和程大雷预料的相同，全城搜捕的结果是一无所获。
<br/>　　那刺客就像个鬼魂一般，凭空而来，凭空消失，谁也不知她身在何处。但程大雷知道，她并不曾离开，如今就躲在一个自己想不到的位置，等待合适的机会，给自己致命一剑。
<br/>　　每想起这件事，程大雷就一身冷汗。幸亏自己早有提防，不然那一夜聂隐娘神不知鬼不觉的潜入，也就一剑报销了自己的性命。等第二天徐神机等人知晓的时候，看到的也就是自己人头落地的惨象。
<br/>　　一日一日过去，程大雷没离开屋子一步。这样的日子的确是很无聊的，其他人也见不到面，程大雷每日和坐牢差不多。
<br/>　　不过，程大雷大概知道，自己如此无聊，聂隐娘的日子也不好过，现在比的就是谁更能忍。
<br/>　　屈指算来，这已是程大雷躲起来的第七夜。
<br/>　　这天夜里，城主府大殿的屋顶上悄悄多了一个人。没有人想到，这几日聂隐不仅没有离开蛤蟆城，事实上，她连城主府都没有出，就躲在一间无人居住屋子的房梁上。
<br/>　　张肥全城搜寻，但城主府中多女眷，自然不能让一帮士兵乱闯。不过是草草看过一遍，并没能找到聂隐娘。
<br/>　　她窝在城主府中，听下人闲谈，也对程大雷目前的情况掌握一些。
<br/>　　在知道程大雷躲在一个铁屋子内足不出户后，她心中暗骂了一声狡猾。她一生杀过许多人，有的卑鄙，有的残暴，但如程大雷这般胆小如鼠，当真还是平生第一次遇到。
<br/>　　程大雷在等，她也在等，等一个合适出手的机会。
<br/>　　今夜，她悄悄离开了藏身之处，在屋顶上行走，向程大雷藏身的屋子行去，一边也探察城主府今日的警戒有没有异样。
<br/>　　如果这是一个圈套，她第一时间就可以有所察觉。
<br/>　　程大雷的屋子里还亮着灯，她脚尖一点，已经落在屋顶之上，轻盈得宛若狸猫。
<br/>　　聂隐娘自幼在山中长大，与白猿为友，不知不觉修成一身绝妙的轻功。翻山越岭，蹿房越脊如履平地。
<br/>　　她在房顶掀开一张瓦片，悄悄往里看，见程大雷点着，良久一动不动，连书也不翻一页。
<br/>　　聂隐娘不得不怀疑这是不是什么假人，曾经有目标用草人伪装，想要引她进入圈套，最后自然是失败了。
<br/>　　程大雷忽地打了声哈欠，站起身，从床底下抽出马桶开始小解。
<br/>　　聂隐娘的眼神并没有避开，杀过许多人，目标在她眼中不分男女，无非是待宰的羔羊而已。某种程度上，她心里根本没有男女的概念，从小在山中长大，也没什么朋友，身边不过一个教她杀人的尼姑。在她眼里，人只分两种，可杀的，或不可杀的。
<br/>　　她不仅没有避开视线，还借机打量程大雷的相貌，与脑海中的记忆对比，避免房间中这人是其他人假扮的。
<br/>　　确认无误，房中就是程大雷，他果然想出这样一个法子，把自己关进一个笼子里，以为这样就可以逃脱性命。
<br/>　　不得不说，想法有些过分天真，也忒小看了聂隐娘。
<br/>　　这座铁笼，能让外面的人无法轻易进来，但在他想出去的时候，也不能轻易出去。
<br/>　　他给自己打了一座坟墓，自己只要闯进去，十息之内，必然可以割下他的脑袋。
<br/>　　至于聂隐娘自己的逃生问题，在刺杀目标得手之前，他不会想这个问题。
<br/>　　聂隐娘深吸一口气，一掌敲在屋顶之上，伴随着哗啦啦瓦片坠地的声音，她出现在房间内，手中一柄手掌大小的匕首。
<br/>　　程大雷刚提上裤子，就听见了头顶的巨响，还没回过神，就见一女子立在自己面前。
<br/>　　他终于知道聂隐娘是如何模样，也从未感觉死亡距离自己这么近。
<br/>(本章完)
