　　剑师兄这一句话，令在场，突然变得安静无比。

<br/>　　七大宗门，招惹其一，几乎便是代表着死路一条。

<br/>　　一名武者，看着楚风眠，这就是一尊无法无天的人物。
<br/>　　归墟天开启的无数岁月中，都没有出现过这种事。
<br/>　　“怎么，剑苍穹，你们天剑宗的弟子，也被这小子杀了？”
<br/>　　那盘师兄盘纯，听到剑苍穹的话，也是有些惊讶一声道。
<br/>　　他本以为楚风眠所说的，将七大宗门弟子全部斩杀，只是一句狂言。

<br/>　　剑苍穹冷冷的开口道。
<br/>　　“碎尸万段？好大的口气，不过是楚某的手下败将罢了，什么七大宗门弟子，在我的面前想要杀了你们，跟踩死几个蝼蚁，没有任何区别，今天谁出手，我就先杀了谁！”
<br/>　　楚风眠负手而立，看着那剑苍穹，大笑着开口道。
<br/>　　他的每一句话，简直都是狂妄到了极点。
<br/>　　但是眼前，楚风眠，的确也有着狂妄的资格，面对这七大宗门其二，都毫不退缩。
<br/>　　这份霸气，就是大部分武者，无法想象的。
<br/>　　“放肆！”
<br/>　　剑苍穹爆喝一声，彻底怒喝，楚风眠的一句手下败将，简直是刺到了他心中的逆鳞，令他愤怒到了极点。
<br/>　　之前他动用千里杀剑，打算将楚风眠隔空斩杀，却是居然被楚风眠轻易化解，更是一阵嘲笑。
<br/>　　这一直以来在剑苍穹的心中，简直就是一种无上羞辱。
<br/>　　他剑苍穹，身为中域百绝榜上的第一人，身份更是还要公孙道之上。
<br/>　　在九域之中武道最为繁华的中域，都是真正的无敌人物，超级强者，人人敬仰的存在。
<br/>　　可是他却是在楚风眠的面前，失利了，尤其还是当着其他天剑宗弟子的面。
<br/>　　这一直以来都是被他视为最大的耻辱，这段时间，剑苍穹每时每刻都恨不得要斩杀了楚风眠，洗刷了这耻辱。
<br/>　　现在众目睽睽下，被直接揭露出来，剑苍穹又羞又怒，简直是气愤到了极点。
<br/>　　“敢杀我天剑宗弟子，便是罪大恶极，人人得以株之，今日我就先杀了你，祭奠我师弟的在天之灵！”
<br/>　　说话之间，剑苍穹突然出手，身上的气势，全部展现出来。
<br/>　　这剑苍穹的境界，甚至是还要高过那盘纯，乃是生死境八重的境界。
<br/>　　握在他手中的灵剑，剑光爆发。
<br/>　　这剑苍穹手中的灵剑，都是一把天级灵器，尤其是他本来都是一位极为强大的剑修。
<br/>　　这灵剑，在他的手中，更是足以爆发出最为强大的力量。
<br/>　　剑光闪动，这剑苍穹猛然一剑落下，虽是一剑，却是在斩出一刻，突然万道剑气，同时斩了出来。
<br/>　　斩出一剑如万剑。
<br/>　　“一剑万重！这是剑道极为高深的地步！斩出一剑，可抵万剑，这可是传说中无数剑道宗师，都难以领悟的境地，只有天剑宗的一些老古董才懂得，这剑苍穹的剑道，居然走到了这一步！”
<br/>　　看着剑苍穹的一剑，不少武者都是惊呼一声。
<br/>　　一剑万重，足以称得上是剑道中传说的境界了。
<br/>　　只有一些老古董，才可以领悟到的剑道境地，在场的不少武者中，许多人也都是第一次见到这一重境界。
<br/>　　“剑苍穹，不愧的千年难遇的剑道奇才，单单是这一剑，便是足以证明他的名气！”
<br/>　　无数武者一边惊叹，一边都是看向了楚风眠所在。
<br/>　　想要看看这一剑，楚风眠该是如何应对。
<br/>　　尤其是楚风眠，同为剑修，这是一场真正的剑修之战。
<br/>　　“一剑万重，不错，但是你的一剑万重，还没有达到大成，万剑之中，还有一千为虚。”
<br/>　　楚风眠看着剑苍穹打出的一剑，淡然之间开口道。
<br/>　　“什么？”
<br/>　　听到楚风眠的话，剑苍穹的脸色都是突然之间大吃一惊。

<br/>　　漫天剑气，从空中厮杀，整个天羽城中的武者，都是胆战心惊。
